# 课程总览

这里汇总本仓库已经整理的课程资料，帮助你先建立一张“资料地图”，再进入具体章节复习。

## 项目定位

本项目把课程笔记、复习资料、题目讲解、图片和 PDF 按课程整理为在线阅读站。它更适合作为课后复盘和期末复习入口，而不是替代课堂讲义或教材。

建议使用方式：

- 平时学习：按课程章节阅读，遇到概念不清楚时使用搜索。
- 期末复习：先看课程总览和章节目录，再集中阅读复习资料、题库或习题精讲。
- 查缺补漏：通过左侧目录定位章节，通过页面右侧“本文目录”跳到具体小节。

## 已整理课程

### 大一下

- 离散数学 I：集合、关系、映射、图论、逻辑等基础内容，是后续算法、数据库和理论课程的重要基础。

### 大二下

- 微机系统：围绕处理器、8086/Pentium、汇编语言、存储管理、输入输出和接口展开，适合按章节逐步复习。
- 操作系统：覆盖进程、线程、同步互斥、内存管理、文件系统等核心知识，适合结合题目理解概念边界。
- 概率论：包含随机事件、随机变量、数字特征、大数定律、参数估计和假设检验等内容，公式较多，适合在线阅读时配合搜索。
- 算法分析与设计：用于整理算法思想、复杂度分析和典型题型，是后续刷题与工程实践的基础。
- 计算机系统结构：关注体系结构、流水线、存储层次、并行和性能分析等内容，需要和微机系统、操作系统交叉理解。

## 阅读建议

先从课程 README 或章节复习资料进入，遇到具体概念再用搜索定位。对公式密集的页面，建议使用桌面端阅读；对图片和 PDF 较多的课程，建议直接在在线站中预览并结合原始文件查看。

## 贡献与维护

如果后续补充资料，尽量保持稳定命名：课程目录使用学期前缀，章节资料使用序号开头。新增 Markdown 或 PDF 后运行 `python3 tools/generate_sidebar.py`，侧边栏会自动刷新。
