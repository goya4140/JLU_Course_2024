# 7.5 图着色问题 — 复习资料

> **对应作业：** 平时作业题7（最大团问题）与图着色问题密切相关（均属图论上的回溯法应用）

---

## 一、问题描述

### 1.1 图着色问题（GCP）

给定无向连通图 $G=(V,E)$，用 $m$ 种不同颜色给图中顶点着色，要求**任意两个相邻顶点的颜色不同**。

- **图着色优化问题：** 求所需的**最少颜色数**（称为图的色数 $\chi(G)$）
- **图的 m-着色判定问题：** 给定 $m$，判断是否存在合法的 $m$-着色方案；若存在，给出方案

> 图着色问题是最著名的 **NP-完全问题**之一。

### 1.2 实际应用背景

| 应用场景 | 对应关系 |
|---------|---------|
| 地图着色（相邻区域不同色） | 区域 → 顶点，相邻 → 边 |
| 课程排课（同学不能同时上两门课） | 课程 → 顶点，冲突 → 边 |
| 寄存器分配（编译优化） | 变量 → 顶点，同时活跃 → 边 |
| 频率分配（相邻基站不同频） | 基站 → 顶点，干扰 → 边 |

## 二、问题建模

### 2.1 解向量与约束条件

用 **n-元组** $(x_1, x_2, \ldots, x_n)$ 表示解，$x_i$ 表示顶点 $i$ 的颜色编号：

| 约束类型 | 内容 | 说明 |
|---------|------|------|
| **显式约束** | $x_i \in \{1, 2, \ldots, m\}$，$1 \leq i \leq n$ | 每个顶点的颜色从 $m$ 种中选 |
| **隐式约束** | 若顶点 $i$ 和顶点 $j$ 之间有边，则 $x_i \neq x_j$ | 相邻顶点颜色不同 |

### 2.2 多米诺性质验证

若已着色的顶点 $(x_1, \ldots, x_k)$ 中存在相邻顶点同色，则无论后续如何着色，该冲突始终存在 → 满足多米诺性质，可以剪枝。

### 2.3 解空间树

- n 个顶点、m 种颜色 → 解空间树是**完全 m 叉树**
- 叶结点个数：$m^n$
- 总结点数：$1 + m + m^2 + \cdots + m^n = \dfrac{m^{n+1}-1}{m-1}$

> 解空间树类型：**排列树**（每个顶点从 $m$ 种颜色中选一种，允许重复）

---

## 三、约束函数 OK(k)

### 3.1 约束函数的作用

判断**顶点 $k$ 着色为 $C(k)$ 是否合法**，即是否与已着色的顶点产生冲突。

### 3.2 伪代码

```
procedure OK(k)
// 判断顶点 k 的颜色 C(k) 是否与前 k-1 个顶点的颜色冲突
// C 为全局数组，进入时已置入前 k 个值
// 返回 true 表示合法，false 表示冲突
  i ← 1
  while (i < k) do
    if (顶点 i 和顶点 k 之间有边存在  and  C(i) = C(k))
      then return false       // 相邻且同色，冲突
    endif
    i ← i + 1
  repeat
  return true
end OK
```

### 3.3 约束函数说明

- 只检验顶点 $k$ 与**前 $k-1$ 个已着色顶点**之间的冲突
- 两个条件同时满足才冲突：① 有边相连（相邻）② 颜色相同
- 时间复杂度：$O(k)$，最坏 $O(n)$

## 四、回溯算法 MCOLORING

### 4.1 完整伪代码

```
procedure MCOLORING(V, E, C, n, m)
// 图 G=(V,E)，n 个顶点，m 种颜色
// C(1:n) 记录各顶点颜色，初始全为 0
  C(1:n) ← 0;  k ← 1
  while (k ≥ 1) do
    C(k) ← C(k) + 1                             // 尝试下一种颜色
    while (not OK(k)  and  C(k) ≤ m) do
      C(k) ← C(k) + 1                           // 当前颜色冲突，换下一种
    repeat
    if C(k) ≤ m then
      if k = n
        then print(C);  return true              // 所有顶点着色完毕，输出方案
        else k ← k + 1;  C(k) ← 0              // 处理下一个顶点
      endif
    else
      k ← k - 1                                 // 顶点 k 无合法颜色，回溯
    endif
  repeat
  return false                                   // k=0，整棵树遍历完，无方案
end MCOLORING
```

### 4.2 算法逻辑说明

| 情况 | 处理 |
|------|------|
| $C(k) \leq m$ 且 $k = n$ | 找到完整合法着色方案，输出并返回 true |
| $C(k) \leq m$ 且 $k < n$ | 当前顶点着色合法，继续下一个顶点（$k+1$） |
| $C(k) > m$ | 当前顶点 $m$ 种颜色均不合法，**回溯**（$k-1$） |
| $k = 0$ | 回溯超过根结点，不存在合法 $m$-着色方案，返回 false |

> 算法找到**第一个**合法方案即返回；若要找**所有**方案，将 `return true` 改为继续搜索。

## 五、实例分析

### 5.1 课件实例（n=5，m=3）

图的边集：$A$-$B$，$A$-$C$，$B$-$C$，$B$-$D$，$B$-$E$，$C$-$D$，$D$-$E$

搜索过程（动态树，按 A,B,C,D,E 顺序着色）：

```
结点1：根
  A=1 → 结点2
    B=1：与A相邻且同色 → 跳过
    B=2 → 结点4
      C=1：与A(1)不相邻？A-C有边，C(1)=A(1)=1 → 跳过
      C=2：与B(2)相邻且同色 → 跳过
      C=3 → 结点7
        D=1：与B无冲突，与C(3)不冲突 → 结点8
          E=1：与B(2)相邻，1≠2 ✓；与D(1)相邻，1=1 → 跳过
          E=2：与B(2)相邻，2=2 → 跳过
          E=3：与B(2)不冲突，与D(1)不冲突 ✓ → 结点11（但D-E有边，E不能=D=1）
          （…继续枚举）
        D=2：与B(2)相邻，2=2 → 跳过
        D=3：与C(3)相邻，3=3 → 跳过（D-C有边）
      → D无合法色，回溯到C=3处，C全部颜色用尽，回溯到B=2
    → B已用完3种
  → 继续到B=2,C=3,D=1时找到 E=2，即方案 (1,2,3,1,2)
```

**找到的第一个解：** $(A,B,C,D,E) = (1,2,3,1,2)$，即颜色方案：
```
A→颜色1，B→颜色2，C→颜色3，D→颜色1，E→颜色2
```

验证：所有边 $(A,B),(A,C),(B,C),(B,D),(B,E),(C,D),(D,E)$ 两端颜色均不同 ✓

## 六、求最少色数（优化问题）

### 6.1 从判定到优化

图着色**优化问题**转化为**判定问题**：

1. 令 $m = 1$，调用 MCOLORING，若返回 true → 色数为1
2. 令 $m = 2$，调用 MCOLORING，若返回 true → 色数为2
3. 依次增大 $m$，直到第一次返回 true，此时的 $m$ 即为**最少色数 $\chi(G)$**

### 6.2 时间复杂度

- 单次判定：$O(n \cdot m^n)$（最多 $m^n$ 个叶结点，每个结点 OK 检验 $O(n)$）
- 求最优：最多调用 $n$ 次判定，总复杂度 $O(n^2 \cdot m^n)$

## 七、与最大团问题的联系（对应作业题7）

### 7.1 最大团问题描述

给定无向图 $G=\langle V, E \rangle$，**完全子图** $G'=\langle V', E' \rangle$（$G'$ 中任意两顶点间均有边）称为 $G$ 的一个**团**。**最大团**即含顶点数最多的团。

### 7.2 问题建模（作业题7(1)）

转化为**判定问题**：判断是否存在大小为 $l$ 的团。

**解向量：** 不定长 k-元组 $(x_1, x_2, \ldots, x_k)$，$x_i$ 为顶点编号，$x_i < x_{i+1}$（按升序避免重复）

| 约束类型 | 内容 |
|---------|------|
| **显式约束** | $x_i \in \{1,2,\ldots,n\}$，且 $x_i < x_{i+1}$（升序排列） |
| **隐式约束** | $x_1, x_2, \ldots, x_k$ 中任意两个顶点之间都有边（构成完全子图） |

### 7.3 解空间树（作业题7(2)，n=5 判断是否存在大小为3的团）

n=5 时，判断大小为3的团，解空间为所有从5个顶点中选3个的组合（升序排列），共 $\binom{5}{3}=10$ 个：

```
                        根
         /      /     |     \      \
       x1=1   x1=2  x1=3  x1=4   x1=5
       / | \   /|    /
    x2=2 3 4  x2=3 4  x2=4
    /|  | |   |  |     |
 x3=3 4 5 4 5  4 5    5
```

各叶结点对应的候选团：
$(1,2,3),(1,2,4),(1,2,5),(1,3,4),(1,3,5),(1,4,5),(2,3,4),(2,3,5),(2,4,5),(3,4,5)$

### 7.4 约束函数设计

**可行性约束（必要条件）：** 在选入 $x_k$ 时，$x_k$ 必须与已选的所有顶点 $x_1, \ldots, x_{k-1}$ 都有边相连：

$$\forall i < k: (x_i, x_k) \in E$$

**上界剪枝（充分剪枝）：** 当前已选 $k-1$ 个顶点，剩余可选顶点数为 $n - x_{k-1}$（编号大于 $x_{k-1}$ 的顶点），若即使全选也达不到目标大小 $l$，则剪枝：

$$k - 1 + (n - x_{k-1}) < l \implies \text{剪枝}$$

等价地：$x_{k-1} > n - l + k - 1$ 时剪枝。

### 7.5 最大团算法设计（作业题7(3)）

**策略：** 从 $l = n$ 开始向下枚举，第一次判定成功时的 $l$ 即为最大团大小。

```
procedure MAX_CLIQUE(G, n)
// 寻找图 G 的最大团
  bestSize ← 0
  bestClique ← ∅
  call CLIQUE_BACKTRACK(1, [], 0, n, G, bestSize, bestClique)
  print bestClique
end MAX_CLIQUE

procedure CLIQUE_BACKTRACK(k, current, size, n, G, bestSize, bestClique)
// current: 当前已选顶点集合，size: 当前团大小
  // 上界剪枝：即使后续全选也不超过当前最优
  if size + (n - k + 1) ≤ bestSize then return endif

  for x ← k to n do
    // 可行性约束：x 是否与 current 中所有顶点相邻
    if x 与 current 中所有顶点均有边 then
      current.add(x)
      if size + 1 > bestSize then
        bestSize ← size + 1
        bestClique ← current的副本
      endif
      call CLIQUE_BACKTRACK(x+1, current, size+1, n, G, bestSize, bestClique)
      current.remove(x)
    endif
  repeat
end CLIQUE_BACKTRACK
```

## 八、易错点汇总

| 易错点 | 正确理解 |
|--------|---------|
| **图着色是 NP 完全问题** | 不存在已知的多项式时间算法（除非 P=NP），回溯法是精确求解的实用方法 |
| **OK 函数两个条件缺一不可** | 必须同时满足"有边"且"同色"才冲突；无边的顶点即使同色也合法 |
| **算法找到第一个解即返回** | MCOLORING 找到一个合法方案就 return true；若需所有方案，需去掉 return 继续搜索 |
| **$k=0$ 的含义** | while 循环条件 $k \geq 1$，当 $k=0$ 时退出循环，说明**不存在** $m$-着色方案 |
| **叶结点数 vs 总结点数** | 叶结点数为 $m^n$；总结点数为 $(m^{n+1}-1)/(m-1)$，两者不同 |
| **最大团的升序约束** | 解向量要求 $x_i < x_{i+1}$，防止 $(1,2,3)$ 和 $(3,1,2)$ 重复计数 |
| **最大团的两种剪枝** | ①可行性剪枝（新顶点必须与已选所有顶点相邻）②上界剪枝（剩余顶点全选也不够） |

## 九、考点速查

| 考点 | 核心内容 |
|------|---------|
| **问题类型** | NP-完全问题，图的 m-着色判定 |
| **解向量** | n-元组，$x_i \in \{1,\ldots,m\}$，解空间 $m^n$ |
| **解空间树** | 完全 m 叉树，总结点数 $(m^{n+1}-1)/(m-1)$ |
| **约束函数 OK(k)** | 顶点 $k$ 与前 $k-1$ 个顶点逐一检验：有边且同色则返回 false |
| **回溯条件** | $C(k) > m$（所有 $m$ 种颜色均不合法）→ $k \leftarrow k-1$ |
| **求最少色数** | 从 $m=1$ 依次增大，第一次 MCOLORING 返回 true 时的 $m$ 即为色数 |
| **最大团建模** | 不定长升序 k-元组，隐式约束为任意两顶点间有边 |
| **最大团剪枝** | 可行性剪枝（与已选顶点均相邻）+ 上界剪枝（剩余不够目标大小） |
