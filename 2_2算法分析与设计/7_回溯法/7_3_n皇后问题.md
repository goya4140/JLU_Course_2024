# 7.3 n-皇后问题 — 复习资料

> **对应作业：** 平时作业题2（回溯法效率估计，6-皇后问题）

---

## 一、问题描述与建模

### 1.1 问题描述

在一个 $n \times n$ 的棋盘上放置 $n$ 个皇后，使得任意两个皇后互不攻击，即：

- 不在同一**行**
- 不在同一**列**
- 不在同一条**斜角线**上

### 1.2 解向量与约束条件

用 n-元组 $(x_1, x_2, \ldots, x_n)$ 表示解，$x_i$ 表示第 $i$ 个皇后放在**第 $i$ 行第 $x_i$ 列**。

| 约束类型 | 内容 | 说明 |
|---------|------|------|
| **显式约束** | $x_i \in \{1, 2, \ldots, n\}$，且没有两个 $x_i$ 相同 | 保证不同行、不同列 |
| **隐式约束** | 没有两个皇后在同一条斜角线上 | 需要约束函数检验 |

> 解空间元组个数为 $n!$（每行选不重复的列 → 排列数）

### 1.3 解空间树（排列树）

- n=4 时，解空间为所有满足显式约束的 4-元组
- 叶结点个数 = $4! = 24$
- 总结点数 = $1 + 4 + 4 \times 3 + 4 \times 3 \times 2 + 4! = 1 + 4 + 12 + 24 + 24 = 65$

## 二、约束函数 PLACE(k)

### 2.1 斜角线判断

设两个皇后分别位于 $(i, x_i)$ 和 $(k, x_k)$，它们在同一条斜角线上，当且仅当：

$$i - x_i = k - x_k \quad \text{（左上到右下的斜线）}$$
$$i + x_i = k + x_k \quad \text{（右上到左下的斜线）}$$

综合以上两个条件：

$$\boxed{|x_i - x_k| = |i - k|}$$

### 2.2 约束函数伪代码

```
procedure PLACE(k)
// 判断第 k 行皇后放在 X(k) 列是否与前 k-1 个皇后冲突
// 返回 true 表示可以放置，false 表示冲突
  i ← 1
  while (i < k) do
    if X(i) = X(k)  or  ABS(X(i) - X(k)) = ABS(i - k)
      then return false
    endif
    i ← i + 1
  repeat
  return true
end PLACE
```

**说明：**
- `X(i) = X(k)`：两皇后在同一列（违反显式约束）
- `ABS(X(i)-X(k)) = ABS(i-k)`：两皇后在同一斜角线（违反隐式约束）

## 三、回溯算法 NQUEENS(n)

```
procedure NQUEENS(n)
  X(1) ← 0;  k ← 1
  while (k > 0) do
    X(k) ← X(k) + 1                           // 尝试下一列
    while (X(k) ≤ n  and  not PLACE(k)) do
      X(k) ← X(k) + 1                         // 当前列冲突，继续向右
    repeat
    if X(k) ≤ n then
      if k = n
        then print(X)                          // 找到一个解，输出
        else k ← k + 1;  X(k) ← 0             // 处理下一行
      endif
    else
      k ← k - 1                               // 当前行无合法列，回溯
    endif
  repeat
end NQUEENS
```

### 算法执行逻辑

| 情况 | 处理方式 |
|------|---------|
| 当前列满足约束（`X(k) ≤ n` 且 `PLACE(k)` 为真） | 若已是第 $n$ 行则输出；否则进入第 $k+1$ 行 |
| 当前列不满足约束 | 继续尝试 $x_k+1, x_k+2, \ldots$ |
| 第 $k$ 行所有列均不满足（`X(k) > n`） | 回溯：$k \leftarrow k-1$ |
| $k = 0$ | 搜索结束（所有解已找完） |

## 四、4-皇后问题完整执行过程（动态树）

### 执行过程描述

**初始：** $x_1 = 0$，$k = 1$

**步骤1：** $x_1 \leftarrow 1$，PLACE(1)=true → $k=2$，$x_2 \leftarrow 0$

**步骤2：** $x_2 \leftarrow 1$，与 $x_1=1$ 同列，PLACE(2)=false
**步骤3：** $x_2 \leftarrow 2$，$(1,1)$ 与 $(2,2)$ 在同一斜线，PLACE(2)=false
**步骤4：** $x_2 \leftarrow 3$，PLACE(2)=true → $k=3$，$x_3 \leftarrow 0$

**步骤5~6：** $x_3 = 1$ 或 $x_3 = 4$，均冲突（与 $x_1=1, x_2=3$ 冲突）
**步骤7：** $x_3 > 4$，回溯 $k=2$

**步骤8：** $x_2 \leftarrow 4$，PLACE(2)=true → $k=3$，$x_3 \leftarrow 0$

**步骤9~10：** $x_3=1,2$ 冲突，$x_3=2$ 通过 → $k=4$

（按此步骤继续，最终找到第一个解 $(2,4,1,3)$，共生成16个活结点）

### 第一个解的棋盘（$(x_1,x_2,x_3,x_4) = (2,4,1,3)$）

```
  列  1  2  3  4
行1 [ ][ Q][ ][ ]   x1=2
行2 [ ][ ][ ][Q]    x2=4
行3 [Q][ ][ ][ ]    x3=1
行4 [ ][ ][Q][ ]    x4=3
```

> 4-皇后问题共有 **2 个解**：$(2,4,1,3)$ 和 $(3,1,4,2)$（互为对称）

## 五、效率估计（蒙特卡罗方法）

### 5.1 n-皇后静态树结点数

以 n=6（作业题2(1)）为例：

**条件：** 每行每列各不相同（即满足显式约束：n-皇后的显式约束为每行选不同的列号）

$$\text{静态树结点数} = 1 + 6 + 6 \times 5 + 6 \times 5 \times 4 + 6 \times 5 \times 4 \times 3 + 6 \times 5 \times 4 \times 3 \times 2 + 6!$$

$$= 1 + 6 + 30 + 120 + 360 + 720 + 720 = \mathbf{1957}$$

| 层 | 含义 | 结点数 |
|----|------|--------|
| 0（根） | 空状态 | 1 |
| 1 | 第1行已放置（6种列） | 6 |
| 2 | 第1、2行均已放置（6×5种） | 30 |
| 3 | 前3行（6×5×4种） | 120 |
| 4 | 前4行（6×5×4×3种） | 360 |
| 5 | 前5行（6×5×4×3×2种） | 720 |
| 6（叶） | 所有6行（6!种） | 720 |
| **合计** | | **1957** |

### 5.2 蒙特卡罗方法估计不受限结点数

**算法思路（ESTIMATE）：**

1. 从根结点出发，沿树生成**一条随机路径**
2. 在第 $k$ 层，计算约束函数通过后的合法儿子结点数 $m_k$
3. 从 $m_k$ 个合法儿子中**随机选一个**继续向下
4. 不受限结点的估计数：

$$\boxed{\hat{m} = 1 + m_1 + m_1 m_2 + m_1 m_2 m_3 + \cdots}$$

其中 $m_i$ 表示第 $i$ 层**随机选中的结点**的通过约束函数的儿子结点数。

**示例（题目图1所示随机路径）：**

假设随机路径过程中，各层通过检验的儿子数依次为 $m_1, m_2, m_3, m_4, m_5$，则：

$$\hat{m} = 1 + m_1 + m_1 m_2 + m_1 m_2 m_3 + m_1 m_2 m_3 m_4 + m_1 m_2 m_3 m_4 m_5$$

多次试验取平均值以减小误差。

### 5.3 8-皇后问题效率估计（课件参考数据）

| 指标 | 数值 |
|------|------|
| 静态树结点数 | 109601 |
| 蒙特卡罗估计均值 | ≈ 1625 |
| 比例（不受限/总数） | **≈ 1.48%** |

> 回溯法将 8-皇后问题的实际搜索规模压缩至静态树的约 **1.5%**，效率极高。

## 七、蒙特卡罗方法的特点

### 优点
- 适用于寻找**所有答案结点**的场景
- 约束函数固定不变，计算方便
- 对树中**同一级**的结点均适用

### 缺点
- 只求一个解时，实际生成结点数**远小于** $\hat{m}$，估计值偏大
- 随着搜索深入，约束函数本可以更强（考虑了更多历史信息），但蒙特卡罗使用**固定约束函数**，不能体现这一点

## 八、易错点汇总

| 易错点 | 正确理解 |
|--------|---------|
| **静态树结点数计算** | 第 $k$ 层有 $P(n, k) = n \cdot (n-1) \cdots (n-k+1)$ 个结点（已考虑不重复列的显式约束），注意从层0（根）到层n（叶）共 $n+1$ 层 |
| **斜角线判断公式** | 是 $\|x_i - x_k\| = \|i - k\|$，**不是** $x_i - x_k = i - k$（绝对值！） |
| **PLACE函数的作用** | 只检验第 $k$ 行与**前 $k-1$ 行**的冲突，不检验后续行 |
| **算法终止条件** | 外层 while 循环条件是 $k > 0$，$k=0$ 表示回溯超过根结点，搜索结束 |
| **蒙特卡罗与实际结点数** | $\hat{m}$ 是**估计**的不受限结点数，不等于回溯算法实际生成的结点数；多次试验取平均才接近期望值 |
| **4-皇后的解数** | 4-皇后有 **2个解**：$(2,4,1,3)$ 和 $(3,1,4,2)$；n-皇后找一个解不需要遍历全树 |

## 九、考点速查

| 考点 | 核心内容 |
|------|---------|
| **显式约束** | $x_i \in \{1,\ldots,n\}$，且 $x_i$ 两两不相同（不同列） |
| **隐式约束** | 任意两皇后不在同一斜角线 |
| **解空间大小** | $n!$（叶结点数），静态树总结点数 $= \sum_{k=0}^{n} P(n,k)$ |
| **斜角线冲突条件** | $\|x_i - x_k\| = \|i - k\|$ |
| **约束函数** | PLACE(k)：与前 $k-1$ 行的列号和斜角线逐一检验 |
| **蒙特卡罗公式** | $\hat{m} = 1 + m_1 + m_1 m_2 + m_1 m_2 m_3 + \cdots$ |
| **8-皇后效率** | 不受限结点仅占静态树的约 **1.48%** |
| **解空间树类型** | n-皇后属于**排列树**（解是对列号集合的排列） |
