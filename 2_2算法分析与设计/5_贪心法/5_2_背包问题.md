## 二、背包问题（5.2）

### 2.1 问题描述

已知有 n 种物品和一个可容纳 M 重量的背包，每种物品 i 的重量为 $w_i$，将物品 i 的某一部分 $x_i$ 放入背包就会得到 $p_i x_i$ 的效益（$0 \leq x_i \leq 1$，$p_i > 0$）。

$$\text{极大化} \quad \sum_{1 \leq i \leq n} p_i x_i \qquad 0 \leq x_i \leq 1,\ p_i > 0$$

$$\text{约束条件} \quad \sum_{1 \leq i \leq n} w_i x_i \leq M \qquad w_i > 0,\ 1 \leq i \leq n$$

> 注意：这是**分数背包**，物品可以取一部分（$x_i$ 可以是小数）。与 0/1 背包不同。

---

### 2.2 三种量度标准对比

以 n=3，M=20，$(p_1,p_2,p_3)=(25,24,15)$，$(w_1,w_2,w_3)=(18,15,10)$ 为例：

| 量度标准 | $(x_1,x_2,x_3)$ | $\sum w_i x_i$ | $\sum p_i x_i$ | 问题 |
|---|---|---|---|---|
| 按效益值 $p_i$ 非增次序 | $(1,\ 2/15,\ 0)$ | 20 | **28.2** | 容量消耗过快 |
| 按重量 $w_i$ 非降次序 | $(0,\ 2/3,\ 1)$ | 20 | **31** | 效益值增长缓慢 |
| 按 $p_i/w_i$ 非增次序 | $(0,\ 1,\ 1/2)$ | 20 | **31.5** ✓ | 每消耗单位容量获得当前最大效益 |

**结论**：按单位重量效益值 $p_i/w_i$ 的非增次序排列，是背包问题的最优量度标准。

---

### 2.3 GREEDY-KNAPSACK 算法

```
procedure GREEDY-KNAPSACK(P, W, M, X, n)
  // 先将物品按 pi/wi 比值的非增次序排序（降序）
  // P(1:n)：效益值；W(1:n)：重量；X(1:n)：解向量；M：背包容量
  real P(1:n), W(1:n), X(1:n), M, cu
  integer i, n
  X ← 0
  cu ← M                           // cu 表示背包剩余容量
  for i <- 1 to n do
    if W(i) > cu then exit endif  // 如果当前物品太重，装不下，直接退出循环
    X(i) <- 1                     // 否则，把这个物品全部拿走
    cu <- cu - W(i)               // 背包剩余容量减少
  repeat
  if i ≤ n then X(i) ← cu / W(i)  // 物品 i 放入一部分
end GREEDY-KNAPSACK
```

时间复杂度：排序 $O(n\log n)$，贪心选取 $O(n)$，总体 $O(n\log n)$。

### 2.4 定理5.1：背包问题贪心解最优性证明

**定理5.1**：如果 $p_1/w_1 \geq p_2/w_2 \geq \cdots \geq p_n/w_n$，则算法 GREEDY-KNAPSACK 对于给定的背包问题实例生成一个最优解。

**证明**：

**（1）贪心解的结构**

设 $X=(x_1,\ldots,x_n)$ 是贪心算法生成的解。若所有 $x_i=1$，则显然是最优解。否则，设 $j$ 是使 $x_j \neq 1$ 的最小下标，由算法可知：

$$x_i = \begin{cases} 1 & 1 \leq i < j \\ x_j \in [0,1) & i = j \\ 0 & j < i \leq n \end{cases}$$

即贪心解形如 $(1,\ldots,1,\ x_j,\ 0,\ldots,0)$，且 $\sum w_i x_i = M$（背包装满）。

**（2）假设存在更优解**

若 X 不是最优解，则必存在最优解 $Y=(y_1,\ldots,y_n)$，使得 $\sum p_i y_i > \sum p_i x_i$。不失一般性，假定 $\sum w_i y_i = M$。

设 $k$ 是使 $y_k \neq x_k$ 的最小下标（这样的 $k$ 必定存在）。

**（3）证明 $y_k < x_k$**（三种情况分析）

- 若 $k < j$：则 $x_k = 1$，因 $y_k \neq x_k$，故 $y_k < x_k = 1$；
- 若 $k = j$：若 $x_k < y_k$，则因 $\sum w_i x_i = M$，会推出 $\sum w_i y_i > M$，与假设矛盾；若 $y_k = x_k$，与 $y_k \neq x_k$ 矛盾。故 $y_k < x_k$；
- 若 $k > j$：前 $k-1$ 个分量中，$y_i = x_i$（$i < j$ 时均为1，$i = j$ 时 $y_j = x_j$），再加 $w_k y_k > 0$ 会使总重超过 M，矛盾。故 $k > j$ 的情况不存在。

**（4）构造新解 Z**

>构造 Z的过程就像“移多补少”，然后计算新解Z比旧解Y是否多赚了钱，若多赚了，则假设的最优解Y不成立，意味着我们多拿的“高价值”物品带来的收益，一定大于等于扔掉的“低价值”物品带来的损失。

将 $y_k$ 增加到 $x_k$，同时从第 $k+1$ 到 $n$ 的分量中减去等量重量（保持总重仍为 M）。构造新解 $Z=(z_1,\ldots,z_n)$，其中 $z_i = x_i$（$1 \leq i \leq k$），且：

$$\sum_{k < i \leq n} w_i(y_i - z_i) = w_k(z_k - y_k)$$

**（5）证明 $\sum p_i z_i \geq \sum p_i y_i$**

$$\sum p_i z_i = \sum p_i y_i + (z_k - y_k)p_k - \sum_{k < i \leq n}(y_i - z_i)p_i$$

将 $p_k = w_k \cdot (p_k/w_k)$，$p_i = w_i \cdot (p_i/w_i)$ 代入：

$$= \sum p_i y_i + (z_k - y_k)w_k \cdot \frac{p_k}{w_k} - \sum_{k < i \leq n}(y_i - z_i)w_i \cdot \frac{p_i}{w_i}$$

因物品按 $p_i/w_i$ 非增排列，$p_k/w_k \geq p_i/w_i$（$i > k$），故：

$$\geq \sum p_i y_i + \left[(z_k - y_k)w_k - \sum_{k < i \leq n}(y_i - z_i)w_i\right] \cdot \frac{p_k}{w_k} = \sum p_i y_i$$

即 $\sum p_i z_i \geq \sum p_i y_i$。

**（6）得出结论**

若 $\sum p_i z_i > \sum p_i y_i$，则 Y 不是最优解，矛盾；故 $\sum p_i z_i = \sum p_i y_i$。此时 Z 也是最优解，且 Z 与 X 相比少了一个不同的分量。反复应用上述替换，最终将 Y 转换成 X 且目标函数值不减，从而 **X 也是最优解**。$\square$

### 【作业题1】0/1背包问题

**题目**：将 5.2 节讨论的背包问题改成：

$$\text{极大化} \quad \sum_{i=1}^{n} p_i x_i \qquad x_i \in \{0, 1\},\ 1 \leq i \leq n$$

$$\text{约束条件} \quad \sum_{i=1}^{n} w_i x_i \leq M$$

即物品只能整件装入或不装（不能取一部分）。证明按 $p_i/w_i$ 非增次序的贪心策略**不一定能得到最优解**。

**证明（反例）**：

取 $n=2$，$M=10$，$(p_1,p_2)=(11,6)$，$(w_1,w_2)=(6,5)$，另有物品3 $(p_3=6,w_3=5)$，$n=3$，$M=10$：

| 物品 | $p_i$ | $w_i$ | $p_i/w_i$ |
|---|---|---|---|
| 1 | 11 | 6 | **1.833**（最高） |
| 2 | 6 | 5 | 1.2 |
| 3 | 6 | 5 | 1.2 |

**贪心策略**：
- 选物品1（$w=6\leq10$），装入，剩余容量 $= 4$；
- 物品2（$w=5>4$），跳过；
- 物品3（$w=5>4$），跳过；
- 贪心解：$x=(1,0,0)$，效益 $= \mathbf{11}$。

**最优解**：选物品2和物品3，$w=5+5=10\leq10$，效益 $=6+6=\mathbf{12}$。

**结论**：贪心解效益为11，最优解效益为12，贪心策略**未得到最优解**。

根本原因：0/1背包中物品不能拆分，选了单位效益最高的物品后，可能因重量问题导致剩余容量"浪费"，而若选多件较小但总效益更高的物品反而更优。
