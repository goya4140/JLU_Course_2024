# 第五章 贪心方法 · 5.4 最小生成树问题

---

## 一、问题定义

**最小生成树**：设 $G=(V,E)$ 是一个加权无向连通图，$|V|=n$。$G$ 的一棵**生成树**是一棵无向树 $T=(V,E')$，其中 $E' \subseteq E$，包含所有 $n$ 个顶点，恰好有 $n-1$ 条边且无环。生成树的**权**是 $E'$ 中所有边权之和。$G$ 的**最小生成树（MST）** 是具有最小权值的生成树。

**贪心策略**（量度标准）：每次从剩余边中选取权值最小且**不形成回路**的边加入 $T$。这是直接以目标函数（权值和最小）作为量度标准。

---

## 二、Kruskal 算法

### 2.1 算法伪代码

```
// 输入：加权连通无向图 G=(V,E)
// 输出：G 的一棵最小生成树 T

T ← ∅                              // T 存储已选边的集合
while |T| < n-1 do
    从 E 中选择一条权值最小的边 (v, w)
    从 E 中删除该边
    if 将 (v,w) 加入 T 中不形成回路
    then T ← T ∪ {(v,w)}
    else 放弃 (v,w)
    endif
repeat
return T
```

> **实现说明**：判断是否形成回路使用并查集（集合树）——若边 $(v,w)$ 的两端点属于同一棵集合树（即同一连通分量），则加入后会形成回路，否则不会，并执行 UNION 合并两个连通分量。

### 2.2 时间复杂度

- 对 $e$ 条边排序：$O(e \log e)$
- 每次 UNION/FIND 操作接近 $O(1)$（带路径压缩），共 $e$ 次：$O(e)$
- 总体时间复杂度：$O(e \log e)$

由于连通图的边数 $e \leq \dfrac{n(n-1)}{2}$，故 $O(e\log e) = O(n^2 \log n)$（以顶点数表示）。

## 三、算法实例运行

以下图为例（5个顶点，8条边）：

```
        A ---300--- B
        |  \     / |
       50   80 70  75
        |     X   |
       200  /  \  60
        | /    \ |
        E ---90--- D ---（与C相连）
                C（与B相连70，与D相连60）
```

各边按权值从小到大排序：

| 步骤 | 边 | 权值 | 加入/放弃 | 说明 |
|---|---|---|---|---|
| 1 | (A,E) | 50 | **加入** | A、E 不在同一连通分量 |
| 2 | (C,D) | 60 | **加入** | C、D 不在同一连通分量 |
| 3 | (B,C) | 70 | **加入** | B、C 不在同一连通分量 |
| 4 | (B,D) | 75 | **放弃** | B、D 已连通（B-C-D），加入会形成回路 |
| 5 | (A,D) | 80 | **加入** | {A,E} 与 {B,C,D} 不在同一连通分量 |
| 6 | — | — | 已选 4 条边（n-1=4），结束 | |

**最小生成树**：边集合 $\{(A,E),(C,D),(B,C),(A,D)\}$，总权值 $= 50+60+70+80 = 260$。

## 四、Kruskal 贪心解最优性证明

**定理**：Kruskal 算法对于每一个无向连通图 $G$ 产生一棵最小生成树。

### 4.1 证明思路

设 $T$ 是 Kruskal 算法产生的生成树，$T'$ 是 $G$ 的某棵最小生成树（MST）。若 $T = T'$，结论显然成立。若 $T \neq T'$，则通过**反复替换**将 $T'$ 转换为 $T$，并证明每次替换后总权值不增，从而 $T$ 也是 MST。

### 4.2 完整证明

**（1）选取替换对象**

设 $e$ 是一条满足 $e \in E(T)$ 但 $e \notin E(T')$ 的**最小权值**边（即 Kruskal 最先选入 $T$ 但不在 $T'$ 中的边）。

将 $e$ 加入 $T'$，由于 $T'$ 是生成树（无环），加入 $e$ 后 $T' \cup \{e\}$ 中必然出现**恰好一个环**。设该环为 $e, e_1, e_2, \ldots, e_k$。

**（2）环中必存在不属于 $T$ 的边**

若环中所有边 $e_1, e_2, \ldots, e_k$ 都属于 $E(T)$，则 $T$ 也包含这个环——与 $T$ 是树（无环）矛盾。故环中至少有一条边 $e_j \notin E(T)$。

**（3）确定 $e_j$ 与 $e$ 的权值关系**

由于 $e_j \notin E(T)$：Kruskal 算法在考虑 $e_j$ 时，$e_j$ 被**放弃**了，说明当时加入 $e_j$ 会形成回路。而 $e$ 是在考虑 $e_j$ **之前或之后**被选入 $T$ 的：

- 若 $e_j$ 的权值 $c(e_j) < c(e)$，则 Kruskal 应在 $e$ 之前考虑 $e_j$，并且当时 $e_j$ 若不形成回路就会被加入，但我们假设 $e_j \notin E(T)$，矛盾。
- 故 $c(e_j) \geq c(e)$。

**（4）构造新的最小生成树**

令 $T'' = T' \cup \{e\} \setminus \{e_j\}$（用 $e$ 替换 $T'$ 中的 $e_j$）。

- $T''$ 仍然是生成树（删去环上一条边 $e_j$，消除回路，连通性不变）；
- $T''$ 的权值 $= $ 权值$(T') + c(e) - c(e_j) \leq$ 权值$(T')$（因为 $c(e) \leq c(e_j)$）；
- 由于 $T'$ 是 MST，$T''$ 权值不能小于 $T'$ 权值，故 $c(e) = c(e_j)$，$T''$ 也是 MST。

**（5）反复替换**

$T''$ 与 $T$ 的差异比 $T'$ 与 $T$ 的差异少一条边（$T''$ 包含了 $e$，而 $T'$ 不包含 $e$）。反复对 $T''$ 应用上述步骤，最终将 $T'$ 转换为 $T$，且权值始终不增。故 $T$ 也是最小生成树。$\square$

### 4.3 证明框架总结（替换论证法）

| 步骤 | 内容 |
|---|---|
| 1 | 设贪心解 $T$，假设存在更优解 $T'$ |
| 2 | 找到在 $T$ 中但不在 $T'$ 中的**最小权值边** $e$ |
| 3 | 将 $e$ 加入 $T'$ 产生环，找到环中不属于 $T$ 的边 $e_j$ |
| 4 | 利用贪心选边顺序，证明 $c(e_j) \geq c(e)$ |
| 5 | 用 $e$ 替换 $e_j$，新解权值 $\leq$ 原最优解权值 |
| 6 | 反复替换，将最优解转换为贪心解，故贪心解也是最优解 |

## 五、贪心策略分析

### 5.1 问题要素对应

| 贪心法要素 | 最小生成树中的对应 |
|---|---|
| **约束条件** | $T$ 含 $n$ 个顶点，$n-1$ 条边，$E' \subseteq E$，无环 |
| **目标函数** | 所有边权之和最小 |
| **量度标准** | 每次选当前权值最小且不形成回路的边 |
| **最优量度标准** | ✅ 已由定理证明成立 |

### 5.2 与其他贪心问题量度标准的比较

| 问题 | 量度标准 | 是否最优 |
|---|---|---|
| 背包问题 | $p_i/w_i$ 非增次序 | ✅ 是（分数背包）|
| 作业调度 | $p_i$ 非增次序 | ✅ 是 |
| 最小生成树 | 边权最小且不成环 | ✅ 是 |
| 货郎担（TSP） | 最近邻居 | ❌ 只得近似解 |

## 六、常见考点与易错点

### 6.1 关于"不形成回路"的判断

Kruskal 实际实现中，用**并查集（集合树）**来快速判断是否形成回路：
- FIND(v) 和 FIND(w)：找到 $v$、$w$ 各自所在连通分量的根节点
- 若根节点相同 → 加入 $(v,w)$ 会形成回路 → 放弃
- 若根节点不同 → 不会形成回路 → 加入，并 UNION 两棵树

### 6.2 时间复杂度辨析

- 若以**边数** $e$ 表示：$O(e \log e)$
- 若以**顶点数** $n$ 表示（最密集图 $e = O(n^2)$）：$O(n^2 \log n)$
- 稀疏图（$e = O(n)$）：$O(n \log n)$

### 6.3 易错点

1. **边数与顶点数的关系**：最小生成树恰好有 $n-1$ 条边（$n$ 为顶点数），不是 $n$ 条。

2. **权值相等时的选择**：若有多条边权值相同，任选其一均可，最终结果都是合法的 MST（可能不唯一）。

3. **证明中"最先选入但不在 $T'$ 中的边"**：$e$ 的选取方式（最小权值的 $T \setminus T'$ 中的边）是证明成立的关键，不能随意选任意边。

4. **Kruskal vs Prim**：Kruskal 按边的权值排序，适合稀疏图；Prim 从一个顶点出发逐步扩展，适合稠密图。本课程只要求掌握 Kruskal。

## 七、本节小结

- Kruskal 算法：按**边权从小到大**排序，每次选不形成回路的最小边，直到选满 $n-1$ 条边
- 时间复杂度：$O(e \log e) = O(n^2 \log n)$
- 贪心量度标准（边权最小且不成环）是**最优量度标准**，证明使用替换论证法
- 最小生成树是贪心法能取得**全局最优解**的经典案例
