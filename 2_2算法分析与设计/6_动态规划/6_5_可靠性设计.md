# 6.5 可靠性设计 — 复习资料

---

## 一、问题描述

### 1.1 系统结构

设计一个由 $n$ 种设备以**串联方式**连接组成的系统：

$$D_1 \longrightarrow D_2 \longrightarrow \cdots \longrightarrow D_n$$

- $r_i$：设备 $D_i$ 正常运转的概率（**单台可靠性**）
- 整个系统的可靠性：$\prod_{i=1}^n r_i$（各级设备独立工作）

**串联系统的可靠性问题：** 短板效应显著，任何一级故障都导致系统失效。

**可靠性对比示例：**

| 场景 | 各级可靠性 | 系统可靠性 |
|------|----------|----------|
| 10级均匀 | $r_i=0.99$，$i=1,\ldots,10$ | $0.99^{10} \approx 0.904$ |
| 3级差异大 | $r_1=0.9,\ r_2=0.8,\ r_3=0.5$ | $0.9\times0.8\times0.5=0.36$ |

### 1.2 改进方案：冗余设备

每级设置 **$m_i$ 台相同设备并联**，只要其中至少一台正常即可：

$$\phi_i(m_i) = 1 - (1-r_i)^{m_i}$$

> **示例：** $r_i=0.99$，$m_i=2$，则可靠性 $= 1-(1-0.99)^2 = 1-0.0001 = 0.9999$，大幅提升！

$$D_1 \parallel D_1 \longrightarrow D_2 \parallel D_2 \longrightarrow \cdots \longrightarrow D_n \parallel D_n$$

## 二、可靠性设计最优化问题

### 2.1 问题形式化

**在总成本不超过 $c$ 的约束下，确定各级设备的配置台数 $m_i$，使系统可靠性最大。**

$$\text{极大化} \quad \prod_{1 \leq i \leq n} \phi_i(m_i) = \prod_{1 \leq i \leq n} \left[1-(1-r_i)^{m_i}\right]$$

$$\text{约束条件} \quad \sum_{1 \leq i \leq n} c_i m_i \leq c, \quad 1 \leq m_i \leq u_i$$

### 2.2 各级设备最大台数 $u_i$

每种设备至少装 1 台，其余成本全部分配给设备 $j$：

$$u_j = \left\lfloor \frac{c + c_j - \sum_{k=1}^{n} c_k}{c_j} \right\rfloor$$

> **含义：** 其他 $n-1$ 种设备各装1台后，剩余预算最多还能给设备 $j$ 装多少台。

---

## 三、递推关系式

### 3.1 子问题定义

$f_i(x)$：在成本约束 $x$ 下，对**前 $i$ 种设备**进行可靠性设计所能达到的**最大可靠性**。

$$f_i(x) = \max_{1 \leq m_i \leq u_i} \left\{ \phi_i(m_i) \cdot f_{i-1}(x - c_i m_i) \right\}$$

**初始条件：**

$$f_0(x) = 1 \quad (x \geq 0)$$

**最终目标：** $f_n(c)$，即在总成本 $c$ 下系统的最大可靠性。

> **与0/1背包的类比：**
> - 背包：$f_i(X)=\max\{f_{i-1}(X),\ f_{i-1}(X-w_i)+p_i\}$（加法效益）
> - 可靠性：$f_i(x)=\max\{\phi_i(m_i)\cdot f_{i-1}(x-c_i m_i)\}$（乘法效益，且 $m_i$ 可取多值）

---

## 四、序偶对法求解

### 4.1 序偶对含义

$$S_i = \{(f,\ x)\}$$

- $f$：前 $i$ 种设备在花费 $x$ 时能达到的**最大可靠性**
- $x$：前 $i$ 种设备的**已用成本**

### 4.2 支配规则

对 $(f_1, x_1)$ 和 $(f_2, x_2)$，若：

$$f_1 \geq f_2 \quad \text{且} \quad x_1 \leq x_2$$

则 $(f_2, x_2)$ 被 $(f_1, x_1)$ **支配**，从集合中舍去。

> **直觉：** 花费更少（$x_1 \leq x_2$）但可靠性更高（$f_1 \geq f_2$）的方案永远优于另一个，后者永远不可能是最优解。

> ⚠️ **注意与0/1背包支配规则的方向差异：**
> - 背包：$W$ 更大且 $P$ 更小时被支配（效益越大越好）
> - 可靠性：$x$ 更大且 $f$ 更小时被支配（成本越小越好，可靠性越大越好）

### 4.3 算法设计思想

```
S0 ← {(1, 0)}                         // 初始：无设备，成本0，可靠性1
求解各级 ui，i=1,2,...,n

for i ← 1 to n do
  for j ← 1 to ui do
    计算 φi(j) = 1-(1-ri)^j             // 第 i 级配置 j 台的可靠性
    Sij ← {(φi(j)×f, x+ci×j) | (f,x)∈Si-1，且 x+ci×j+∑ck≤c}
    //         i+1≤k≤n  （保证后续各级至少各配1台）
  repeat
  Si ← 基于支配规则归并 ∪(1≤j≤ui) Sij
repeat

// Sn 中 f 最大的序偶即为最优解
// 回溯：判断最优序偶来自哪个 Snj，确定 mn，类推 mn-1,...,m1
```

**剪枝条件：** 加入 $S_{ij}$ 的序偶须满足 $x + c_i j + \sum_{i+1 \leq k \leq n} c_k \leq c$，即剩余预算足够后续各级各装至少1台。

## 五、完整计算过程（3级系统实例）

### 实例参数

| 设备 | 成本 $c_i$ | 可靠性 $r_i$ |
|------|-----------|------------|
| $D_1$ | 30 | 0.9 |
| $D_2$ | 15 | 0.8 |
| $D_3$ | 20 | 0.5 |

总预算 $c = 105$

### 第一步：计算各级最大台数 $u_i$

$$u_1 = \left\lfloor \frac{105 + 30 - (30+15+20)}{30} \right\rfloor = \left\lfloor \frac{70}{30} \right\rfloor = 2$$

$$u_2 = \left\lfloor \frac{105 + 15 - 65}{15} \right\rfloor = \left\lfloor \frac{55}{15} \right\rfloor = 3$$

$$u_3 = \left\lfloor \frac{105 + 20 - 65}{20} \right\rfloor = \left\lfloor \frac{60}{20} \right\rfloor = 3$$

### 第二步：$i=1$，$c_1=30$，$r_1=0.9$

$$\phi_1(1) = 0.9, \quad \phi_1(2) = 1-(1-0.9)^2 = 0.99$$

$$S_1^1 = \{(1 \times 0.9,\ 0+30)\} = \{(0.9,\ 30)\}$$
$$S_1^2 = \{(1 \times 0.99,\ 0+60)\} = \{(0.99,\ 60)\}$$

归并，无支配（$0.9<0.99$，$30<60$）：

$$\boxed{S_1 = \{(0.9,\ 30),\ (0.99,\ 60)\}}$$

### 第三步：$i=2$，$c_2=15$，$r_2=0.8$

$$\phi_2(1)=0.8,\quad \phi_2(2)=1-(0.2)^2=0.96,\quad \phi_2(3)=1-(0.2)^3=0.992$$

**构造 $S_{21}$（$m_2=1$，花费 $+15$）：**

$$S_2^1 = \{(0.9\times0.8,\ 30+15),\ (0.99\times0.8,\ 60+15)\} = \{(0.72,\ 45),\ (0.792,\ 75)\}$$

**构造 $S_{22}$（$m_2=2$，花费 $+30$）：**

$$S_2^2 = \{(0.9\times0.96,\ 30+30),\ (0.99\times0.96,\ 60+30)\} = \{(0.864,\ 60),\ (0.9504,\ 90)\}$$

注意：$(0.9504, 90)$ 需检查剩余预算：$90 + c_3 = 90+20=110 > 105$，**超出预算，舍弃！**

**构造 $S_{23}$（$m_2=3$，花费 $+45$）：**

$$S_2^3 = \{(0.9\times0.992,\ 30+45),\ (0.99\times0.992,\ 60+45)\} = \{(0.8928,\ 75),\ (0.9821,\ 105)\}$$

$(0.9821, 105)$：$105 + 20 > 105$，**超出预算，舍弃！**

**归并所有候选：**

$$\{(0.72,45),\ (0.792,75),\ (0.864,60),\ (0.8928,75)\}$$

检查支配：$(0.792, 75)$ vs $(0.8928, 75)$：同等成本，$0.792 < 0.8928$，$(0.792,75)$ 被支配，**舍弃！**

$$\boxed{S_2 = \{(0.72,\ 45),\ (0.864,\ 60),\ (0.8928,\ 75)\}}$$

### 第四步：$i=3$，$c_3=20$，$r_3=0.5$

$$\phi_3(1)=0.5,\quad \phi_3(2)=1-(0.5)^2=0.75,\quad \phi_3(3)=1-(0.5)^3=0.875$$

**构造 $S_{31}$（$m_3=1$，花费 $+20$）：**

$$S_3^1 = \{(0.72\times0.5,\ 65),\ (0.864\times0.5,\ 80),\ (0.8928\times0.5,\ 95)\}$$
$$= \{(0.36,\ 65),\ (0.432,\ 80),\ (0.4464,\ 95)\}$$

**构造 $S_{32}$（$m_3=2$，花费 $+40$）：**

$$S_3^2 = \{(0.72\times0.75,\ 85),\ (0.864\times0.75,\ 100),\ (0.8928\times0.75,\ 115)\}$$
$$= \{(0.54,\ 85),\ (0.648,\ 100)\}$$

$(0.8928\times0.75, 115)$：$115>105$，**超出预算，舍弃！**

**构造 $S_{33}$（$m_3=3$，花费 $+60$）：**

$$S_3^3 = \{(0.72\times0.875,\ 105),\ \ldots\}= \{(0.63,\ 105)\}$$

后两项均超出预算，舍弃。

**归并所有候选：**

$$\{(0.36,65),\ (0.432,80),\ (0.4464,95),\ (0.54,85),\ (0.648,100),\ (0.63,105)\}$$

检查支配：$(0.63, 105)$ vs $(0.648, 100)$：$x=105>100$ 且 $f=0.63<0.648$，$(0.63,105)$ 被 $(0.648,100)$ 支配，**舍弃！**

$$\boxed{S_3 = \{(0.36,\ 65),\ (0.432,\ 80),\ (0.4464,\ 95),\ (0.54,\ 85),\ (0.648,\ 100)\}}$$

## 六、最优解与决策序列回溯

$S_3$ 中 $f$ 最大的序偶为 $(0.648,\ 100)$。

**回溯过程：**

| 步骤 | 当前序偶 | 来自哪个 $S_{ij}$？ | 决策 | 追踪到 |
|------|---------|----------------|------|--------|
| $i=3$ | $(0.648, 100)$ | $S_3^2$（$m_3=2$，花费$+40$） | $m_3=2$ | $(0.648/0.75,\ 100-40)=(0.864,\ 60)$ |
| $i=2$ | $(0.864, 60)$ | $S_2^2$（$m_2=2$，花费$+30$） | $m_2=2$ | $(0.864/0.96,\ 60-30)=(0.9,\ 30)$ |
| $i=1$ | $(0.9, 30)$ | $S_1^1$（$m_1=1$，花费$+30$） | $m_1=1$ | $(1, 0)=S_0$ |

$$\boxed{\text{最优决策序列：}(m_1,m_2,m_3)=(1,2,2)，\text{最大系统可靠性} = 0.648}$$

**验证：**
- 总成本：$1\times30 + 2\times15 + 2\times20 = 30+30+40=100 \leq 105$ ✅
- 系统可靠性：$\phi_1(1)\times\phi_2(2)\times\phi_3(2) = 0.9\times0.96\times0.75 = 0.648$ ✅

## 七、与 0/1 背包问题的对比

| 对比维度 | 0/1 背包 | 可靠性设计 |
|---------|---------|----------|
| **决策变量** | $x_i \in \{0, 1\}$ | $m_i \in \{1, 2, \ldots, u_i\}$ |
| **目标函数** | $\sum p_i x_i$（加法） | $\prod \phi_i(m_i)$（乘法） |
| **约束条件** | $\sum w_i x_i \leq M$ | $\sum c_i m_i \leq c$ |
| **初始条件** | $f_0(X)=0$ | $f_0(x)=1$ |
| **递推操作** | 取 $\max$，加法合并效益 | 取 $\max$，乘法合并可靠性 |
| **序偶含义** | $(P_j, W_j)$：效益值与重量 | $(f, x)$：可靠性与已用成本 |
| **支配方向** | $W$ 更大且 $P$ 更小被支配 | $x$ 更大且 $f$ 更小被支配 |
| **$S_i^j$ 生成** | 将 $(p_i,w_i)$ 加到 $S_{i-1}$ | 将 $(\phi_i(j)\cdot,\ c_i j+)$ 作用于 $S_{i-1}$ |

## 八、易错点汇总

| 易错点 | 正确理解 |
|--------|---------|
| **$u_i$ 的计算** | $u_i=\lfloor(c+c_i-\sum_{k=1}^n c_k)/c_i\rfloor$，分子是总预算加上设备 $i$ 成本再减去所有设备各1台的总成本 |
| **支配规则方向** | 可靠性设计中：$f$ 更小且 $x$ 更大的被支配；与背包问题（$P$ 更小且 $W$ 更大被支配）结构相同，但序偶含义不同 |
| **剪枝条件** | 构造 $S_{ij}$ 时，要求 $x+c_ij+\sum_{i+1\leq k\leq n}c_k\leq c$（确保后续各设备至少各1台） |
| **初始条件** | $S_0=\{(1,0)\}$，可靠性初值为1（乘法中性元），不是0 |
| **$\phi_i(m_i)$ 公式** | $\phi_i(m_i)=1-(1-r_i)^{m_i}$，是并联可靠性公式，注意底数是 $(1-r_i)$ 而非 $r_i$ |
| **回溯方法** | 判断最优序偶来自哪个 $S_{nj}$（即配置了 $j$ 台），再从对应 $S_{i-1}$ 中找到前驱序偶，逐级回溯 |

## 九、考点速查

| 考点 | 关键内容 |
|------|---------|
| **第 $i$ 级可靠性公式** | $\phi_i(m_i)=1-(1-r_i)^{m_i}$ |
| **最大台数** | $u_j=\lfloor(c+c_j-\sum c_k)/c_j\rfloor$ |
| **递推关系式** | $f_i(x)=\max_{1\leq m_i\leq u_i}\{\phi_i(m_i)\cdot f_{i-1}(x-c_im_i)\}$，初值 $f_0(x)=1$ |
| **序偶含义** | $(f,x)$：成本 $x$ 下前 $i$ 级的最大可靠性 $f$ |
| **支配规则** | $f_1\geq f_2$ 且 $x_1\leq x_2$ 时，$(f_2,x_2)$ 被支配舍去 |
| **与背包的核心区别** | 目标是**乘法**（可靠性之积），初值为1；背包是**加法**，初值为0 |
| **实例结论** | $(m_1,m_2,m_3)=(1,2,2)$，最大可靠性 $=0.648$，总成本 $=100$ |
