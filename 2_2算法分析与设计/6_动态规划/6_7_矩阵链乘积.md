# 6.7 矩阵链乘积问题 — 复习资料

---

## 一、问题描述

### 1.1 背景

给定 $n$ 个矩阵的序列（矩阵链）$A_1, A_2, \ldots, A_n$，计算它们的连乘积 $A_1 A_2 \cdots A_n$，使得所需的**标量乘法次数最少**。

### 1.2 乘法代价

计算两矩阵 $A_{p \times q}$ 与 $B_{q \times r}$ 的乘积 $T_{p \times r}$：
- $T$ 共有 $p \times r$ 个元素
- 每个元素需要 $q$ 次乘法
- **总乘法次数 = $p \times q \times r$**

### 1.3 括号方式影响代价

矩阵乘法**满足结合律**，不同的加括号方式计算量差异悬殊。

**例：** $(A_1)_{10\times100}$，$(A_2)_{100\times5}$，$(A_3)_{5\times50}$

| 括号方式 | 计算过程 | 总乘法次数 |
|---|---|---|
| $(A_1 A_2) A_3$ | $10\times100\times5 + 10\times5\times50$ | $5000 + 2500 = \mathbf{7500}$ |
| $A_1 (A_2 A_3)$ | $100\times5\times50 + 10\times100\times50$ | $25000 + 50000 = \mathbf{75000}$ |

两种方案相差 **10倍**！穷举所有括号方式的数量是 $\Omega(2^n)$，必须用动态规划。

---

## 二、最优性原理证明

**命题：** 矩阵链乘积问题满足最优性原理。

**证明：**

设 $A_i A_{i+1} \cdots A_j$ 的最优括号方案在 $A_k$ 后分割，即最优分法为 $(A_i \cdots A_k)(A_{k+1} \cdots A_j)$。

则子链 $A_i \cdots A_k$ 和 $A_{k+1} \cdots A_j$ 的括号方案也必须分别最优——若某个子链有更优的括号方案，则用其替换后整体代价更小，与原方案最优矛盾。✓

---

## 三、递推关系式

### 3.1 符号约定

设 $A_t$ 是 $p_{t-1} \times p_t$ 的矩阵（$t = 1, \ldots, n$），即矩阵维度由数组 $p_0, p_1, \ldots, p_n$ 描述：

$$A_1: p_0 \times p_1, \quad A_2: p_1 \times p_2, \quad \ldots, \quad A_n: p_{n-1} \times p_n$$

设 $m(i,j)$ 为计算矩阵链 $A_i A_{i+1} \cdots A_j$ 所需的**最少标量乘法次数**。

### 3.2 递推关系式（核心）

$$\boxed{m(i,j) = \begin{cases} 0 & i = j \\ \displaystyle\min_{i \leq k < j} \left\{ m(i, k) + m(k+1, j) + p_{i-1} \cdot p_k \cdot p_j \right\} & i < j \end{cases}}$$

**含义：**
- 在 $A_k$ 后分割，左半链 $A_i \cdots A_k$ 最优代价 $m(i,k)$，结果矩阵大小 $p_{i-1} \times p_k$
- 右半链 $A_{k+1} \cdots A_j$ 最优代价 $m(k+1,j)$，结果矩阵大小 $p_k \times p_j$
- 合并两个结果矩阵的代价 $p_{i-1} \cdot p_k \cdot p_j$
- 遍历所有 $k$ 取最小值

> ⚠️ **与 OBST 对比：**
> - OBST：$C(i,j) = \min\{C(i,k-1)+C(k,j)\} + W(i,j)$，$k$ 范围 $(i,j]$
> - MCO：$m(i,j) = \min\{m(i,k)+m(k+1,j)+p_{i-1}p_kp_j\}$，$k$ 范围 $[i,j)$

---

## 四、自底向上计算

**计算顺序：** 按子链长度 $w = j - i + 1$ 从小到大：

1. $w=1$（单矩阵）：$m(i,i) = 0$
2. $w=2$（两矩阵相乘）：$n-1$ 个
3. $w=3$：$n-2$ 个
4. $\ldots$
5. $w=n$：1 个，即 $m(1,n)$

**标记函数 $s(i,j)$：** 记录使 $m(i,j)$ 取最小值时的分割点 $k$，用于最后回溯最优括号方案。

---

## 五、算法描述（MCO）

```
procedure MCO(n)
// A1 为 p0×p1, A2 为 p1×p2, ..., An 为 p(n-1)×pn
int m(1:n, 1:n), s(1:n, 1:n)   // s 记录最优分割点

// 初始化 w=1 和 w=2 两种情况
for i ← 1 to n do
  m(i,i) ← 0;  s(i,i) ← 0
repeat
for i ← 1 to n-1 do
  m(i, i+1) ← p(i-1) * p(i) * p(i+1)
  s(i, i+1) ← i
repeat

// 自底向上，按问题规模 w=2,3,...,n-1 逐层计算
for w ← 2 to n-1 do           // w = j - i，链长为 w+1
  for i ← 1 to n-w do
    j ← i + w
    k ← 在 [i, j) 中使 {m(i,l)+m(l+1,j)+p(i-1)*p(l)*p(j)} 取最小值的 l
    m(i,j) ← m(i,k) + m(k+1,j) + p(i-1)*p(k)*p(j)
    s(i,j) ← k
  repeat
repeat
end MCO
```

**复杂度：**
- 时间复杂度：$O(n^3)$
- 空间复杂度：$O(n^2)$

## 七、时间复杂度分析

### 朴素递推：$O(n^3)$

- 外层循环 $w$ 从 $1$ 到 $n-1$，共 $n-1$ 次
- 内层循环 $i$ 最多 $n$ 次
- 最内层遍历 $k$ 最多 $w$ 次
- 总操作数：$\sum_{w=1}^{n-1}(n-w)\cdot w = O(n^3)$

### 穷举括号数：卡特兰数

$n$ 个矩阵的不同括号方案数为第 $n-1$ 个**卡特兰数** $C_{n-1} = \Omega(4^n / n^{3/2})$，指数增长，穷举不可行。

## 八、与 OBST 的横向对比

| 对比维度 | OBST（6.6节） | 矩阵链乘积（6.7节） |
|---|---|---|
| **状态定义** | $C(i,j)$：含 $a_{i+1}\ldots a_j$ 的最优树成本 | $m(i,j)$：$A_i\cdots A_j$ 的最少乘法次数 |
| **分割方式** | 选根 $k$，$i < k \leq j$ | 选分割点 $k$，$i \leq k < j$ |
| **合并代价** | $+W(i,j)$（与 $k$ 无关） | $+p_{i-1}p_kp_j$（与 $k$ 有关） |
| **初始条件** | $C(i,i)=0$（空树） | $m(i,i)=0$（单矩阵） |
| **优化方法** | Knuth 优化降至 $O(n^2)$ | 无类似优化，保持 $O(n^3)$ |
| **时间复杂度** | $O(n^3)$ → $O(n^2)$ | $O(n^3)$ |

## 九、易错点汇总

1. **$p$ 下标**：$A_t$ 是 $p_{t-1} \times p_t$ 的矩阵，合并 $A_i\cdots A_k$ 与 $A_{k+1}\cdots A_j$ 的代价是 $p_{i-1} \cdot p_k \cdot p_j$，注意左侧用 $p_{i-1}$（不是 $p_i$），右侧用 $p_j$。

2. **$k$ 范围**：MCO 中 $k$ 的遍历范围是 $[i, j)$（即 $i \leq k < j$），不包含 $j$；而 OBST 中 $k$ 的范围是 $(i, j]$（$i < k \leq j$）。两者方向相反，不要混淆。

3. **链长 $w$ 的含义**：课件中用 $w = j - i$ 表示"多出来的矩阵数"，$w=1$ 对应两矩阵，$w=n-1$ 对应 $n$ 矩阵的完整链。

4. **回溯括号方案**：用 $s(i,j)$ 递归回溯，左半为 $(i, s(i,j))$，右半为 $(s(i,j)+1, j)$，两部分各自递归直到 $i=j$。

5. **合并代价与 $k$ 有关**：不像 OBST 的 $+W(i,j)$ 是常数，MCO 的合并代价 $p_{i-1}p_kp_j$ 随 $k$ 变化（$p_k$ 不同），不能提前提出，必须在 $\min$ 内部计算。

## 十、考点速查

| 考点 | 核心内容 |
|---|---|
| 乘法代价公式 | $A_{p\times q} \cdot B_{q\times r}$ 需要 $p \cdot q \cdot r$ 次乘法 |
| 矩阵维度表示 | $A_t$ 为 $p_{t-1}\times p_t$，由数组 $p_0,\ldots,p_n$ 描述 |
| 递推关系式 | $m(i,j)=\min_{i\leq k<j}\{m(i,k)+m(k+1,j)+p_{i-1}p_kp_j\}$ |
| 初始条件 | $m(i,i)=0$ |
| 时间复杂度 | $O(n^3)$ |
| 空间复杂度 | $O(n^2)$ |
| 最优括号方案 | 用标记函数 $s(i,j)$ 递归回溯 |
| 穷举方案数 | 第 $n-1$ 个卡特兰数，$\Omega(4^n/n^{3/2})$，不可穷举 |
