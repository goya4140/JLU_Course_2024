## 🗂️ 知识体系（含例题）

### 谓词逻辑的基本概念 `⭐简答` / `🔢计算`

**引入：** 命题逻辑将命题看作孤立的整体，无法描述问题内部的成分和逻辑联系（例如著名的三段论“凡人必死，张三是人，张三必死”在命题逻辑中无法被证明为恒真）。为了刻画命题的内部结构以及个体间的共同特性，我们需要引入“谓词”与“量词”的概念。

**定义：**

> * **个体 (Individual)**：可以独立存在的思维对象（如人、自然数等）。个体构成的非空集合称为个体名称集合 $D$。
> * **谓词 (Predicate)**：定义在 $D$ 的 $n$ 次笛卡尔积上，取值于 $\{0,1\}$ 的 $n$ 元函数。一元谓词描述个体性质，多元谓词描述个体间关系。
> * **量词 (Quantifier)**：用于限定个体范围的语句。分为 **全称量词 (Universal Quantifier)** $\forall x$（表示“对任意 $x$”）和 **存在量词 (Existential Quantifier)** $\exists x$（表示“存在一个 $x$”）。
> 
> 

**关键性质：**

* **语义等价性**：当个体域 $D=\{x_0, x_1, ...\}$ 可数时，全称量词 $\forall x G(x)$ 等价于无穷合取 $G(x_0) \wedge G(x_1) \wedge ...$ ；存在量词 $\exists x G(x)$ 等价于无穷析取 $G(x_0) \vee G(x_1) \vee ...$ 。
* **自由与约束**：被量词作用的变量称为**约束变量 (Bound Variable)**，未被量词作用的变量称为**自由变量 (Free Variable)**。
* **改名规则**：可以在量词作用区域内将约束变量改名，但新变量不能与改名区域中的其他自由变量或约束变量冲突。

**注意：** 谓词本身只是命题函数，不是命题。只有代入具体的个体，或加上量词约束后，它才具有真值，成为一个命题。

#### 📝 例题

**【课件例题】** 设个体域 $D=\{1, 2, 3\}$，谓词 $P(x)$ 表示 $x > 2$。试判断下列公式的真值：
(1) $\exists x P(x) \rightarrow P(2)$
(2) $P(3) \rightarrow \forall x P(x)$

**解：**
(1) $\exists x P(x)$ 等价于 $P(1) \vee P(2) \vee P(3)$，即 $0 \vee 0 \vee 1 = 1$。
而 $P(2)$ 真值为 $0$。
所以原公式真值为 $1 \rightarrow 0 = 0$。
(2) $P(3)$ 真值为 $1$。
$\forall x P(x)$ 等价于 $P(1) \wedge P(2) \wedge P(3)$，即 $0 \wedge 0 \wedge 1 = 0$。
所以原公式真值为 $1 \rightarrow 0 = 0$。

### 谓词公式与解释 `🔢计算`

**引入：** 有了常量、变量、谓词和量词后，我们需要像代数表达式一样，严格规定什么样的符号串是合法的，并明确如何在具体的语境中计算它们的真伪。

**定义：**

> * **项 (Term)**：常量符号、变量符号是项；若 $f$ 是 $n$ 元函数符号，$t_1, ..., t_n$ 是项，则 $f(t_1, ..., t_n)$ 也是项。
> * **原子 (Atom)**：若 $P$ 是 $n$ 元谓词符号，$t_1, ..., t_n$ 是项，则 $P(t_1, ..., t_n)$ 是原子。
> * **公式 (Formula)**：由原子通过逻辑联结词（$\neg, \vee, \wedge, \rightarrow, \leftrightarrow$）和量词（$\forall, \exists$）有限次组合而成的合式符号串。
> * **解释 (Interpretation)**：由非空个体域 $D$ 以及对公式中所有常量、函数、谓词符号的具体映射指定所组成的集合。
> 
> 

**关键性质：**

* **分类**：对于一切解释都为真的公式称为 **恒真 (Tautology)**；都为假的称为 **恒假 (Contradiction)**；至少存在一个解释使其为真的称为 **可满足 (Satisfiable)**。
* **量词顺序**：多重量词时，同类量词可交换，但异类量词不可交换。（如 $\forall x \exists y P(x,y)$ 不等于 $\exists y \forall x P(x,y)$）。

**注意：** 公式 $\forall x(P(x) \rightarrow P(a))$ 与 $\forall x P(x) \rightarrow P(a)$ 含义截然不同！前者要求域内每个元素都满足蕴涵关系，后者是两个独立命题的蕴涵。

#### 📝 例题

**【课件例题】** 给定公式 $G = \exists x(P(f(x)) \wedge Q(x, f(a)))$ 和 $H = \forall x(P(x) \wedge Q(x, a))$。
设解释 $I$ 为：$D=\{2, 3\}$，$a=2$；
函数 $f(2)=3, f(3)=2$；
谓词 $P(2)=0, P(3)=1$；
对于任意 $x, y \in D$，均有 $Q(x,y)=1$。求 $G$ 和 $H$ 在 $I$ 下的真值。

**解：**
对于 $G$：


$$T_I(G) = T_I((P(f(2)) \wedge Q(2, f(2))) \vee (P(f(3)) \wedge Q(3, f(2))))$$

$$= T_I((P(3) \wedge Q(2, 3)) \vee (P(2) \wedge Q(3, 3)))$$

$$= (1 \wedge 1) \vee (0 \wedge 1) = 1$$

对于 $H$：


$$T_I(H) = T_I(P(2) \wedge Q(2, 2) \wedge P(3) \wedge Q(3, 2))$$

$$= 0 \wedge 1 \wedge 1 \wedge 1 = 0$$

### 谓词公式的等价关系与蕴涵关系 `⭐证明`

**引入：** 推理的本质是从已知事实推导新结论。在谓词逻辑中，演绎推理的有效性完全依赖于公式间的等价与蕴涵关系。

**定义：**

> * **等价关系 (Equivalence Relation)**：若在任意解释下，公式 $G$ 和 $H$ 的真值都相同（即 $G \leftrightarrow H$ 恒真），则称 $G$ 与 $H$ 等价，记作 $G = H$。
> * **蕴涵关系 (Implication Relation)**：若在任意满足 $G$ 的解释下，$H$ 也必定被满足（即 $G \rightarrow H$ 恒真），则称 $G$ 蕴涵 $H$，记作 $G \Rightarrow H$。
> 
> 

**关键性质：**

* **重要等价式（量词转换律）**：
$\neg(\forall x G(x)) = \exists x (\neg G(x))$
$\neg(\exists x G(x)) = \forall x (\neg G(x))$
* **量词分配律（极其重要）**：
$\forall x (G(x) \wedge H(x)) = \forall x G(x) \wedge \forall x H(x)$
$\exists x (G(x) \vee H(x)) = \exists x G(x) \vee \exists x H(x)$
* **常用基本蕴涵式**：
(a) $\exists x(A(x) \wedge B(x)) \Rightarrow \exists x A(x) \wedge \exists x B(x)$
(b) $\forall x A(x) \vee \forall x B(x) \Rightarrow \forall x(A(x) \vee B(x))$
(c) $\forall x(A(x) \rightarrow B(x)) \Rightarrow (\forall x A(x) \rightarrow \forall x B(x))$

**注意：** 对于析取 $\vee$ 和全称量词 $\forall$、合取 $\wedge$ 和存在量词 $\exists$，它们之间**只有蕴涵关系，没有等价关系**！例如 $\forall x A(x) \vee \forall x B(x) \neq \forall x(A(x) \vee B(x))$。

#### 📝 例题

**【课件例题】** 举反例证明 $G = \forall x(A(x) \rightarrow B(x))$ 与 $H = \forall x A(x) \rightarrow \forall x B(x)$ 不等价。

**解：**
取个体域 $D=\{2, 3\}$。
定义谓词：$A(2)=0, A(3)=1$；$B(2)=0, B(3)=0$。
对于 $G$：当 $x=3$ 时，$A(3) \rightarrow B(3)$ 为 $1 \rightarrow 0 = 0$，因此 $\forall x(A(x) \rightarrow B(x))$ 真值为 $0$。
对于 $H$：$\forall x A(x)$ 的真值为 $A(2) \wedge A(3) = 0$。根据蕴涵定义，$0 \rightarrow \text{任何值} = 1$，故 $H$ 的真值为 $1$。
两者真值不同，故不等价。

### 范式 `🔢计算`

**引入：** 谓词公式的表示方法多种多样，为了统一形式以便于计算机处理和定理自动证明，我们需要将公式化为标准化的形式：前束范式和 Skolem 范式。

**定义：**

> * **前束范式 (Prenex Normal Form)**：具有 $Q_1 x_1 ... Q_n x_n M$ 形式的公式，其中 $Q_i x_i$ 是全称或存在量词（首标），$M$ 是不含有任何量词的公式（母式）。
> * **Skolem 范式 (Skolem Normal Form)**：在前束范式的基础上，进一步消除了所有“存在量词 $\exists$”的范式。
> 
> 

**关键性质：**

* **化前束范式的步骤**：
1. 消去 $\leftrightarrow$ 和 $\rightarrow$。
2. 使用德摩根律和量词否定等价式，将 $\neg$ 移到原子前。
3. 通过约束变量改名，确保不同量词控制的变量名不同。
4. 将所有量词提取到公式最左侧。


* **化 Skolem 范式的步骤**：
1. 先化为前束范式，且保证母式为合取范式形式。
2. 从左至右消去存在量词 $\exists x$：
* 若 $\exists x$ 左侧没有任何全称量词，用一个新的**常量符号 $c$** 替换母式中所有的 $x$。
* 若 $\exists x$ 左侧有全称量词 $\forall y_1, ..., \forall y_m$，则用一个新的$m$ 元函数符号 $f(y_1, ..., y_m)$ (称为 Skolem 函数) 替换母式中所有的 $x$。


3. 删去首标中的存在量词。



**注意：** 改名和提量词时必须严格检查量词的约束域，避免改变公式的原有逻辑含义。

#### 📝 例题

**【课件例题】** 将公式 $\forall x \forall y(A(x) \rightarrow B(x,y)) \rightarrow (\exists y C(y) \rightarrow \exists z D(z))$ 化为前束范式。

**解：**
(1) 消去 $\rightarrow$ 联结词：


$$\neg \forall x \forall y (\neg A(x) \vee B(x,y)) \vee (\neg \exists y C(y) \vee \exists z D(z))$$


(2) 将 $\neg$ 内移到原子之前：


$$\exists x \exists y (A(x) \wedge \neg B(x,y)) \vee (\forall y \neg C(y) \vee \exists z D(z))$$


(3) 约束变量改名（后半部分的 $y$ 改为 $t$ 以免冲突）：


$$\exists x \exists y (A(x) \wedge \neg B(x,y)) \vee (\forall t \neg C(t) \vee \exists z D(z))$$


(4) 提取量词到全式最前：


$$\exists x \exists y \forall t \exists z ((A(x) \wedge \neg B(x,y)) \vee \neg C(t) \vee D(z))$$


为了整理成合取范式做母式，使用分配律：


$$\exists x \exists y \forall t \exists z ((A(x) \vee \neg C(t) \vee D(z)) \wedge (\neg B(x,y) \vee \neg C(t) \vee D(z)))$$

## 📝 简答题题库

**Q：为什么有了命题逻辑还需要引入谓词逻辑？**
**A：**

* 命题逻辑把命题看作不可分割的整体，无法描述命题的内部结构（主语和谓语）。
* 命题逻辑无法表达事物之间的内在联系和共同性质（如全称和特称关系），因此无法形式化像“三段论”这样基于个体性质的推理过程。
* 谓词逻辑通过引入“个体”、“谓词”和“量词”，有效克服了这些局限，极大地增强了逻辑表达能力。

**Q：简述谓词公式中约束变量和自由变量的区别。**
**A：**

* 出现在某量词作用辖域内，且受该量词直接控制（即与量词指导变元同名）的变量称为约束变量。
* 未受任何量词约束，在公式中具有独立指代意义的变量称为自由变量。
* 同一个变量名在不同辖域内可以既作为自由变量又作为约束变量，通过“改名规则”可使公式结构更清晰。

**Q：简述将谓词公式化为 Skolem 范式的主要思想。**
**A：**

* 先将公式转化为前束范式（量词全部提前）。
* 通过引入 Skolem 常量（当存在量词不受任何全称量词约束时）或 Skolem 函数（当存在量词依赖于前面的全称量词时）来显式地“固化”存在性。
* 去掉所有存在量词，只保留全称量词，将判定可满足性的问题简化。

## 🔢 计算 / 证明题专项

### 题型说明

本节考试的核心难点在于 **证明谓词公式的蕴涵关系** 或 **证明公式恒真/恒假**。通常需要熟练使用“解释法（语义法）”或“等价推演法（语法法）”。

### 解题方法与步骤

1. **解释法（任取解释 $I$ 证明蕴涵 $A \Rightarrow B$）**：
* 步骤1：假设任取一解释 $I$ 使得前提 $A$ 的真值为 $1$。
* 步骤2：利用量词的语义（$\forall$ 意为对域内每一个元素成立，$\exists$ 意为域内至少存在一个元素成立），将 $A$ 转化为个体的属性。
* 步骤3：推理目标结论 $B$ 在解释 $I$ 下个体的属性，得出 $B$ 的真值也必为 $1$。


2. **反证法（假设结论假）**：
* 步骤1：假设蕴涵式不成立，即前提 $A$ 为真，而结论 $B$ 为假。
* 步骤2：向下推导导致逻辑矛盾，从而证明原蕴涵式成立。


3. **等价演算法**：利用已知定理（如分配律、量词转换律）进行纯符号推导。

### 课件例题精解

**【证明】 $\forall x(A(x) \vee B(x)) \Rightarrow \forall x A(x) \vee \exists x B(x)$**
**解：** （使用解释法与反证法结合）
假设存在某个解释 $I$ 使得该蕴涵式为假。
则前提 $\forall x(A(x) \vee B(x))$ 在 $I$ 下为真，而结论 $\forall x A(x) \vee \exists x B(x)$ 在 $I$ 下为假。
因为结论为假，所以 $\forall x A(x)$ 为假，且 $\exists x B(x)$ 也为假。
由 $\exists x B(x)$ 为假可知：对于 $D$ 中任意元素 $x$，$B(x)$ 均为假。
又因为前提 $\forall x(A(x) \vee B(x))$ 为真，即对任意 $x$，$A(x) \vee B(x)$ 均为真。
既然对于任意 $x$，$B(x)$ 都为假，那就必然要求对任意 $x$，$A(x)$ 必须为真。
这就意味着 $\forall x A(x)$ 应该为真。
这与前面推出的“$\forall x A(x)$ 为假”相矛盾！
故假设不成立，原蕴涵式 $\forall x(A(x) \vee B(x)) \Rightarrow \forall x A(x) \vee \exists x B(x)$ 得证。

**【证明】 $(\exists x A(x) \rightarrow \forall x B(x)) \Rightarrow \forall x(A(x) \rightarrow B(x))$**
**解：** （使用等价推演法）


$$\exists x A(x) \rightarrow \forall x B(x)$$

$$= \neg \exists x A(x) \vee \forall x B(x) \quad \text{(消去蕴涵)}$$

$$= \forall x (\neg A(x)) \vee \forall x B(x) \quad \text{(量词转换律)}$$


根据基本蕴涵式 $\forall x P(x) \vee \forall x Q(x) \Rightarrow \forall x (P(x) \vee Q(x))$，有：


$$\forall x (\neg A(x)) \vee \forall x B(x) \Rightarrow \forall x (\neg A(x) \vee B(x))$$


又因为 $\neg A(x) \vee B(x) = A(x) \rightarrow B(x)$，故：


$$\Rightarrow \forall x (A(x) \rightarrow B(x))$$


得证。

### ⚠️ 易错提醒

1. **量词的分配误区**：切记 $\forall$ 对 $\vee$ 不具有完全的分配律（只能单向蕴涵），$\exists$ 对 $\wedge$ 也不具有分配律。
2. **改名规则的疏漏**：在提取量词前，如果不同的量词约束了同名的变量，必须**先改名**，否则强行提取会导致变量冲突，彻底改变公式原意。
3. **解释法的完整性**：在证明蕴涵 $A \Rightarrow B$ 时，如果举特例只证明了某一个特定域下成立是不够的，必须使用“任取域 $D$ 和解释 $I$”的抽象描述；而证明**不等价**时，只要能举出一个具体的域和解释（反例）即可。
