# 第六章 数理统计的基本概念

## 知识框架

```
总体X（随机变量/分布）
    │
    ├─ 抽样：n次独立重复
    │
    ▼
样本 X₁,X₂,…,Xₙ（i.i.d. ~ 总体X）
    │
    ├─ 经验分布函数 Fₙ(x) ──→ 格里汶科定理：近似F(x)
    │
    ├─ 统计量（不含未知参数的样本函数）
    │       ├─ 样本均值 X̄
    │       ├─ 样本方差 S²
    │       └─ 数字特征：E(X̄)=μ, D(X̄)=σ²/n, E(S²)=σ²
    │
    ├─ 三大抽样分布
    │       ├─ χ²分布：标准正态的平方和
    │       ├─ t分布：正态 / √(χ²/n)
    │       └─ F分布：两个χ²之比
    │
    └─ 正态总体抽样分布（四大结论，必背！）
```

---

## 一、总体与样本

### 引入

做统计之前，先搞清楚我们在研究谁、怎么研究。现实中不可能对每个个体都测量（比如测一批灯泡寿命，测完就烧完了），所以用"抽样"代替"普查"，用**样本**来推断**总体**。

### 知识点

**总体 $X$**：研究对象某指标的全体，本质是一个**随机变量**（或其分布）。

**样本 $X_1, X_2, \cdots, X_n$**：对总体 $X$ 进行 $n$ 次独立重复抽样，满足：
- $X_1, X_2, \cdots, X_n$ **相互独立**
- 每个 $X_i$ **同分布于总体** $X$

简记：**i.i.d.**（独立同分布），$n$ 为**样本容量**。

抽到的具体数值 $x_1, x_2, \cdots, x_n$ 称为**样本观察值**（样本值）。

> **做题翻译口诀**：
> - 总体 $X$ = "随机变量 $X$"
> - 样本 $X_1,\cdots,X_n$ = "i.i.d.于 $X$ 的 $n$ 个随机变量"
> - 样本值 $x_1,\cdots,x_n$ = "抽样得到的 $n$ 个实数"
> - 样本容量 $n$ = "抽样个数 $n$"

---

## 二、经验分布函数

### 引入

有了样本值，怎么用它估计总体分布？用**经验分布函数**——让数据自己"投票"，票数越多，分布函数越大。

### 知识点

**定义**：设样本值为 $x_1,\cdots,x_n$，定义

$$F_n(x) = \frac{\{x_1,x_2,\cdots,x_n \text{ 中小于或等于 } x \text{ 的个数}\}}{n}, \quad -\infty < x < +\infty$$

**格里汶科定理**：当 $n \to \infty$ 时，$F_n(x)$ 以概率1一致收敛于总体分布函数 $F(x)$。即样本量足够大时，$F_n(x)$ 可当 $F(x)$ 使用。

---

## 三、统计量与统计值

### 引入

拿到样本后，我们需要从中提取信息。**统计量**就是对样本数据的一种"加工方式"——关键约束是**不能含未知参数**（否则就不能算了）。

### 知识点

**统计量**：不含任何未知参数的样本函数 $g(X_1, X_2, \cdots, X_n)$。

**统计值**：统计量在样本值处的取值 $g(x_1, x_2, \cdots, x_n)$。

> 判断一个量是否为统计量：看它是否含**未知**参数。已知参数（题目给出的）可以出现。

---

### 【例题·选择】例6.2

设总体 $X \sim N(\mu, \sigma^2)$，其中 $\mu$ 已知，$\sigma^2$ 未知，$X_1, X_2$ 是取自总体 $X$ 的样本，则下列各量为统计量的是（　　）

$$\text{(A) } X_1 + \mu X_2 \qquad \text{(B) } 2X_1 + \sigma\mu \qquad \text{(C) } X_1 + \mu + \sigma^2 \qquad \text{(D) } \dfrac{X_1 - \mu}{\sigma}$$

**解**：逐项判断是否含**未知**参数（$\sigma^2$ 未知，$\mu$ 已知）：

- (A) $X_1 + \mu X_2$：只含已知参数 $\mu$，✅ **是统计量**
- (B) $2X_1 + \sigma\mu$：含未知参数 $\sigma$，❌
- (C) $X_1 + \mu + \sigma^2$：含未知参数 $\sigma^2$，❌
- (D) $\dfrac{X_1 - \mu}{\sigma}$：含未知参数 $\sigma$，❌

**答案：(A)**

## 四、常用统计量

### 知识点

设 $X_1, X_2, \cdots, X_n$ 为总体 $X$ 的样本，$n$ 为样本容量：

| 统计量 | 公式 |
|--------|------|
| 样本均值 | $\overline{X} = \dfrac{1}{n}\displaystyle\sum_{i=1}^n X_i$ |
| 样本方差 | $S^2 = \dfrac{1}{n-1}\displaystyle\sum_{i=1}^n (X_i - \overline{X})^2 = \dfrac{1}{n-1}\left(\displaystyle\sum_{i=1}^n X_i^2 - n\overline{X}^2\right)$ |
| 样本标准差 | $S = \sqrt{S^2}$ |
| 样本 $k$ 阶原点矩 | $A_k = \dfrac{1}{n}\displaystyle\sum_{i=1}^n X_i^k$ |
| 样本 $k$ 阶中心矩 | $B_k = \dfrac{1}{n}\displaystyle\sum_{i=1}^n (X_i - \overline{X})^k$ |

> ⚠️ 样本方差分母是 $n-1$，不是 $n$！这是为了让 $E(S^2) = \sigma^2$（无偏性）。

---

## 五、常用统计量的数字特征

### 引入

统计量本身是随机变量，所以有期望和方差。这组结论是做题的核心工具，**必须背熟**！

### 知识点

已知 $X_1, X_2, \cdots, X_n$ 为来自总体 $X$ 的样本，$E(X) = \mu$，$D(X) = \sigma^2$，则：

$$\boxed{E(X_i) = \mu, \quad D(X_i) = \sigma^2}$$

$$\boxed{E(\overline{X}) = \mu, \quad D(\overline{X}) = \frac{\sigma^2}{n}}$$

$$\boxed{E(S^2) = \sigma^2}$$

> 记忆口诀：**"模仿游戏"——整体识别！** $\overline{X}$ 的期望和总体一样，方差缩小 $n$ 倍；$S^2$ 的期望等于总体方差。

---

### 【例题·选择】例6.4

设总体 $X \sim B(m,\theta)$，$X_1, X_2, \cdots, X_n$ 为来自该总体的简单随机样本，$\overline{X}$ 为样本均值，则

$$E\left[\sum_{i=1}^n (X_i - \overline{X})^2\right] = \text{（　　）}$$

$$\text{(A) }(m-1)n\theta(1-\theta) \qquad \text{(B) }m(n-1)\theta(1-\theta)$$
$$\text{(C) }(m-1)(n-1)\theta(1-\theta) \qquad \text{(D) }mn\theta(1-\theta)$$

**解**：

注意 $\displaystyle\sum_{i=1}^n (X_i - \overline{X})^2 = (n-1)S^2$，所以

$$E\left[\sum_{i=1}^n (X_i - \overline{X})^2\right] = (n-1) \cdot E(S^2) = (n-1)\sigma^2$$

由 $X \sim B(m,\theta)$，知 $\sigma^2 = D(X) = m\theta(1-\theta)$，故

$$= (n-1) \cdot m\theta(1-\theta)$$

**答案：(B)**

---

### 【例题·选择】例6.5

设总体 $X$ 服从参数为 $\lambda$（$\lambda > 0$）的泊松分布，$X_1, X_2, \cdots, X_n$ 为来自该总体的简单随机样本，则对于统计量

$$T_1 = \frac{1}{n}\sum_{i=1}^n X_i, \quad T_2 = \frac{1}{n-1}\sum_{i=1}^{n-1} X_i + \frac{1}{n}X_n$$

有（　　）

$$\text{(A) }E(T_1) > E(T_2),\; D(T_1) > D(T_2) \qquad \text{(B) }E(T_1) > E(T_2),\; D(T_1) < D(T_2)$$
$$\text{(C) }E(T_1) < E(T_2),\; D(T_1) > D(T_2) \qquad \text{(D) }E(T_1) < E(T_2),\; D(T_1) < D(T_2)$$

**解**：泊松分布 $E(X) = \lambda$，$D(X) = \lambda$。

**求期望**：

$$E(T_1) = \frac{1}{n} \cdot n\lambda = \lambda$$

$$E(T_2) = \frac{1}{n-1}(n-1)\lambda + \frac{1}{n}\lambda = \lambda + \frac{\lambda}{n} = \frac{(n+1)\lambda}{n}$$

故 $E(T_1) < E(T_2)$。

**求方差**（各 $X_i$ 独立）：

$$D(T_1) = \frac{1}{n^2} \cdot n\lambda = \frac{\lambda}{n}$$

$$D(T_2) = \frac{1}{(n-1)^2}(n-1)\lambda + \frac{1}{n^2}\lambda = \frac{\lambda}{n-1} + \frac{\lambda}{n^2}$$

因为 $\dfrac{\lambda}{n-1} > \dfrac{\lambda}{n}$，所以 $D(T_1) < D(T_2)$。

**答案：(D)**

## 六、三大抽样分布

### 引入

统计量的分布称为**抽样分布**。从正态总体出发，有三个极其重要的抽样分布：$\chi^2$、$t$、$F$。它们是后续假设检验和区间估计的核心工具。

### 1. $\chi^2$ 分布

**定义**：$X_1, X_2, \cdots, X_n \overset{i.i.d.}{\sim} N(0,1)$，则

$$\boxed{\chi^2 = X_1^2 + X_2^2 + \cdots + X_n^2 \sim \chi^2(n)}$$

$n$ 为**自由度**（唯一参数）。

**性质**：

1. **可加性**：$\chi_1^2 \sim \chi^2(n_1)$，$\chi_2^2 \sim \chi^2(n_2)$，且相互独立，则

$$\chi_1^2 + \chi_2^2 \sim \chi^2(n_1 + n_2)$$

2. **期望与方差**（背）：若 $\chi^2 \sim \chi^2(n)$，则

$$E(\chi^2) = n, \quad D(\chi^2) = 2n$$

3. **上 $\alpha$ 分位点**：满足 $P\{\chi^2 > \chi^2_\alpha(n)\} = \alpha$ 的点 $\chi^2_\alpha(n)$

---

### 2. $t$ 分布

**定义**：$X \sim N(0,1)$，$Y \sim \chi^2(n)$，且 $X, Y$ 独立，则

$$\boxed{t = \frac{X}{\sqrt{Y/n}} \sim t(n)}$$

$n$ 为**自由度**。

**性质**：

1. **对称性**：$t$ 分布的密度 $f(t)$ 为**偶函数**，图像关于 $y$ 轴对称
2. **分位点关系**：$t_{1-\alpha}(n) = -t_\alpha(n)$
3. **与 $F$ 分布关系**：若 $T \sim t(n)$，则 $T^2 \sim F(1,n)$
4. **与正态关系**：当 $n \geq 30$ 时，$t(n)$ 近似 $N(0,1)$

**上 $\alpha$ 分位点**：满足 $P\{t > t_\alpha(n)\} = \alpha$ 的点 $t_\alpha(n)$

---

### 3. $F$ 分布

**定义**：$X \sim \chi^2(n_1)$，$Y \sim \chi^2(n_2)$，且 $X, Y$ 独立，则

$$\boxed{F = \frac{X/n_1}{Y/n_2} \sim F(n_1, n_2)}$$

$(n_1, n_2)$ 为自由度（$n_1$ 第一，$n_2$ 第二）。

**性质**：

$$\text{若 } F \sim F(n_1, n_2)\text{，则 } \frac{1}{F} \sim F(n_2, n_1)$$

**分位点关系**：

$$F_{1-\alpha}(n_1, n_2) = \frac{1}{F_\alpha(n_2, n_1)}$$

---

### 【例题·选择】例6.6（三大分布综合）

设 $X_1, X_2, X_3, X_4$ 为来自总体 $X \sim N(0,1)$ 的简单随机样本，若统计量

$$Y = \frac{X_1^2 + X_2^2}{X_3^2 + X_4^2}$$

且对给定的 $\alpha$（$0 < \alpha < 1$），数 $y_\alpha$ 满足 $P\{Y > y_\alpha\} = \alpha$，则（　　）

$$\text{(A) } y_\alpha y_{1-\alpha} = 1 \qquad \text{(B) } y_\alpha y_{1-\frac{\alpha}{2}} = 1$$
$$\text{(C) } y_{\frac{\alpha}{2}} y_{1-\frac{\alpha}{2}} = 1 \qquad \text{(D) } y_{\frac{\alpha}{2}} y_{1-\frac{\alpha}{2}} = 1$$

**解**：

$X_1^2 + X_2^2 \sim \chi^2(2)$，$X_3^2 + X_4^2 \sim \chi^2(2)$，独立，故

$$Y = \frac{(X_1^2+X_2^2)/2}{(X_3^2+X_4^2)/2} \sim F(2,2)$$

由 $F$ 分布性质 $\frac{1}{F} \sim F(n_2,n_1)$，此处 $n_1=n_2=2$，故 $\frac{1}{Y} \sim F(2,2)$，即 $Y$ 与 $\frac{1}{Y}$ 同分布。

$y_\alpha$ 为上 $\alpha$ 分位点，$y_{1-\alpha}$ 为上 $(1-\alpha)$ 分位点，即下 $\alpha$ 分位点。

由对称性（此处 $\frac{1}{Y}$ 与 $Y$ 同分布），有 $y_\alpha \cdot y_{1-\alpha} = 1$。

**答案：(A)**

---

### 【例题·选择】例6.7（填空）

设 $X_1, X_2, \cdots, X_5$ 为总体 $X \sim N(0,1)$ 的一个样本，设

$$Y = (X_1 + X_2 + X_3)^2 + (X_4 - \sqrt{2}X_5)^2$$

且 $cY \sim \chi^2$ 分布，则 $c =$ \_\_\_\_\_

**解**：要把 $Y$ 凑成标准正态平方和的形式。

**第一项**：$X_1 + X_2 + X_3 \sim N(0,3)$，故 $\dfrac{X_1+X_2+X_3}{\sqrt{3}} \sim N(0,1)$，

$$\Rightarrow (X_1+X_2+X_3)^2 = 3 \cdot \left(\frac{X_1+X_2+X_3}{\sqrt{3}}\right)^2$$

**第二项**：$X_4 - \sqrt{2}X_5 \sim N(0,\, 1+2) = N(0,3)$，故

$$(X_4-\sqrt{2}X_5)^2 = 3 \cdot \left(\frac{X_4-\sqrt{2}X_5}{\sqrt{3}}\right)^2$$

故

$$Y = 3\left[\left(\frac{X_1+X_2+X_3}{\sqrt{3}}\right)^2 + \left(\frac{X_4-\sqrt{2}X_5}{\sqrt{3}}\right)^2\right]$$

方括号内是两个独立 $N(0,1)$ 变量的平方和 $\sim \chi^2(2)$，所以

$$\frac{1}{3}Y \sim \chi^2(2)$$

$$\boxed{c = \frac{1}{3}}$$

---

### 【例题·填空】例6.8

设随机变量 $X, Y$ 相互独立，均服从 $N(0,3^2)$ 分布且 $X_1, X_2, \cdots, X_9$ 与 $Y_1, Y_2, \cdots, Y_9$ 分别是来自总体 $X, Y$ 的简单随机样本，则统计量

$$U = \frac{X_1 + \cdots + X_9}{\sqrt{Y_1^2 + \cdots + Y_9^2}}$$

服从参数为\_\_\_\_的\_\_\_\_分布。

**解**：$t$ 分布"化标准"三步：①分子化 $N(0,1)$；②分母化 $\chi^2(n)$；③验独立，上下摆放。

**分子**：$X_i \sim N(0,9)$，$\sum_{i=1}^9 X_i \sim N(0,81)$，故

$$\frac{\sum_{i=1}^9 X_i}{9} \sim N(0,1)$$

**分母**：$Y_j \sim N(0,9)$，$\dfrac{Y_j}{3} \sim N(0,1)$，故

$$\sum_{j=1}^9 \left(\frac{Y_j}{3}\right)^2 = \frac{Y_1^2+\cdots+Y_9^2}{9} \sim \chi^2(9)$$

**整理**：

$$U = \frac{X_1+\cdots+X_9}{\sqrt{Y_1^2+\cdots+Y_9^2}} = \frac{\dfrac{\sum X_i}{9}}{\sqrt{\dfrac{Y_1^2+\cdots+Y_9^2}{9} \cdot \dfrac{1}{9}}} = \frac{\dfrac{\sum X_i}{9}}{\dfrac{\sqrt{Y_1^2+\cdots+Y_9^2}}{9}}$$

即

$$U = \frac{N(0,1)}{\sqrt{\chi^2(9)/9}} \sim t(9)$$

**答案**：参数为 $9$ 的 $t$ 分布。

---

### 【例题·填空】例6.9

设总体 $X$ 服从正态分布 $X \sim N(0, 2^2)$，而 $X_1, X_2, \cdots, X_{15}$ 是来自总体的简单随机样本，则随机变量

$$Y = \frac{X_1^2 + \cdots + X_{10}^2}{2(X_{11}^2 + \cdots + X_{15}^2)}$$

服从\_\_\_\_分布，参数为\_\_\_\_。

**解**：$X_i \sim N(0,4)$，$\dfrac{X_i}{2} \sim N(0,1)$，故 $\dfrac{X_i^2}{4} \sim \chi^2(1)$（每一项）。

$$\frac{X_1^2+\cdots+X_{10}^2}{4} \sim \chi^2(10), \quad \frac{X_{11}^2+\cdots+X_{15}^2}{4} \sim \chi^2(5)$$

$$Y = \frac{X_1^2+\cdots+X_{10}^2}{2(X_{11}^2+\cdots+X_{15}^2)} = \frac{\dfrac{X_1^2+\cdots+X_{10}^2}{4} \Big/ 10}{\dfrac{X_{11}^2+\cdots+X_{15}^2}{4} \Big/ 5} \sim F(10,5)$$

**答案**：$F$ 分布，参数为 $(10,5)$。

## 七、正态总体抽样分布

### 引入

这是本章的重中之重！知道总体是正态的情况下，样本均值和样本方差服从什么分布？这四个结论是后续推断统计的基础，**必须倒背如流**。

### 单正态总体（必背）

设 $X_1, X_2, \cdots, X_n$ 是来自正态总体 $X \sim N(\mu, \sigma^2)$ 的样本，$\overline{X}$、$S^2$ 分别为样本均值和样本方差，则：

$$\boxed{1)\quad \overline{X} \sim N\!\left(\mu, \frac{\sigma^2}{n}\right), \quad \text{即} \quad \frac{(\overline{X}-\mu)\sqrt{n}}{\sigma} \sim N(0,1)}$$

$$\boxed{2)\quad \overline{X} \text{ 与 } S^2 \text{ 相互独立}}$$

$$\boxed{3)\quad \frac{(n-1)S^2}{\sigma^2} = \frac{\displaystyle\sum_{i=1}^n(X_i-\overline{X})^2}{\sigma^2} \sim \chi^2(n-1)}$$

$$\quad \frac{\displaystyle\sum_{i=1}^n(X_i-\mu)^2}{\sigma^2} \sim \chi^2(n) \quad \text{（注意：用 }\mu\text{ 时自由度是 }n\text{）}$$

$$\boxed{4)\quad T = \frac{\overline{X}-\mu}{S/\sqrt{n}} \sim t(n-1)}$$

> ⚠️ 为什么结论3中用 $\overline{X}$ 自由度是 $n-1$？因为 $\sum_{i=1}^n(X_i - \overline{X}) = 0$，$n$ 个偏差中只有 $n-1$ 个自由变化。

---

### 两个正态总体

设 $X_1,\cdots,X_{n_1}$ 来自 $N(\mu_1,\sigma_1^2)$，$Y_1,\cdots,Y_{n_2}$ 来自 $N(\mu_2,\sigma_2^2)$，两样本独立，则：

$$\boxed{1)\quad \frac{(\overline{X}-\overline{Y})-(\mu_1-\mu_2)}{\sqrt{\dfrac{\sigma_1^2}{n_1}+\dfrac{\sigma_2^2}{n_2}}} \sim N(0,1)}$$

$$\boxed{2)\quad \frac{S_1^2/\sigma_1^2}{S_2^2/\sigma_2^2} \sim F(n_1-1,\, n_2-1)}$$

$$\boxed{3)\quad \text{当 }\sigma_1^2=\sigma_2^2=\sigma^2\text{ 时：}\quad \frac{(\overline{X}-\overline{Y})-(\mu_1-\mu_2)}{S_W\sqrt{\dfrac{1}{n_1}+\dfrac{1}{n_2}}} \sim t(n_1+n_2-2)}$$

其中

$$S_W^2 = \frac{(n_1-1)S_1^2+(n_2-1)S_2^2}{n_1+n_2-2} \quad \text{（合并方差）}$$

---

### 【例题·选择】例6.10

设 $X_1, X_2, X_3, X_4$ 为来自总体 $N(1,\sigma^2)$ 的简单随机样本，则统计量

$$\frac{X_1 - X_2}{|X_3 + X_4 - 2|}$$

的分布为（　　）

$$\text{(A) }N(0,1) \qquad \text{(B) }t(1) \qquad \text{(C) }\chi^2(1) \qquad \text{(D) }F(1,1)$$

**解**：

**分子**：$X_1 - X_2 \sim N(0, 2\sigma^2)$，故 $\dfrac{X_1-X_2}{\sqrt{2}\sigma} \sim N(0,1)$

**分母**：$X_3 + X_4 - 2 \sim N(0, 2\sigma^2)$，故 $\dfrac{X_3+X_4-2}{\sqrt{2}\sigma} \sim N(0,1)$

$$\left(\frac{X_3+X_4-2}{\sqrt{2}\sigma}\right)^2 \sim \chi^2(1)$$

分子分母独立，套 $t$ 分布定义：

$$\frac{\dfrac{X_1-X_2}{\sqrt{2}\sigma}}{\sqrt{\left(\dfrac{X_3+X_4-2}{\sqrt{2}\sigma}\right)^2 / 1}} = \frac{X_1-X_2}{|X_3+X_4-2|} \sim t(1)$$

**答案：(B)**

---

### 【例题·选择】例6.12

设 $X_1, X_2, \cdots, X_n$（$n \geq 2$）为来自总体 $N(0,1)$ 的简单随机样本，$\overline{X}$ 为样本均值，$S^2$ 为样本方差，则（　　）

$$\text{(A) }n\overline{X} \sim N(0,1) \qquad \text{(B) }nS^2 \sim \chi^2(n)$$
$$\text{(C) }\dfrac{(n-1)\overline{X}}{S} \sim t(n-1) \qquad \text{(D) }\dfrac{(n-1)X_1^2}{\displaystyle\sum_{i=2}^n X_i^2} \sim F(1,n-1)$$

**解**：总体 $N(0,1)$，即 $\mu=0$，$\sigma^2=1$。

- (A)：$\overline{X} \sim N(0, \frac{1}{n})$，故 $\sqrt{n}\,\overline{X} \sim N(0,1)$，而非 $n\overline{X}$，❌
- (B)：$(n-1)S^2 \sim \chi^2(n-1)$，不是 $nS^2$，❌
- (C)：正确形式应为 $\dfrac{\sqrt{n}\,\overline{X}}{S} \sim t(n-1)$，❌
- (D)：$X_1^2 \sim \chi^2(1)$，$\displaystyle\sum_{i=2}^n X_i^2 \sim \chi^2(n-1)$，两者独立，故

$$\frac{X_1^2/1}{\displaystyle\sum_{i=2}^n X_i^2/(n-1)} = \frac{(n-1)X_1^2}{\displaystyle\sum_{i=2}^n X_i^2} \sim F(1,n-1) \quad ✅$$

**答案：(D)**

### 【例题·填空】例6.11

在总体 $X \sim N(5,16)$ 中随机地抽取一个容量为36的样本 $X_1, X_2, \cdots, X_{36}$，则关于样本均值 $\overline{X}$ 的概率

$$P\{4 \leq \overline{X} \leq 6\} = \text{\_\_\_\_\_}$$

**解**：$\mu = 5$，$\sigma^2 = 16$，$n = 36$，由结论1：

$$\overline{X} \sim N\!\left(5, \frac{16}{36}\right), \quad \frac{\overline{X}-5}{2/3} \sim N(0,1)$$

$$P\{4 \leq \overline{X} \leq 6\} = P\!\left\{\frac{4-5}{2/3} \leq \frac{\overline{X}-5}{2/3} \leq \frac{6-5}{2/3}\right\} = P\!\left\{-\frac{3}{2} \leq Z \leq \frac{3}{2}\right\}$$

$$= 2\Phi(1.5) - 1 = 2 \times 0.9332 - 1 = \boxed{0.8664}$$

### 【例题·计算】例6.13

设 $X_1, X_2, \cdots, X_n$ 是来自总体 $N(\mu, \sigma^2)$ 的简单随机样本，记

$$\overline{X} = \frac{1}{n}\sum_{i=1}^n X_i, \quad S^2 = \frac{1}{n-1}\sum_{i=1}^n (X_i - \overline{X})^2, \quad T = \overline{X}^2 - \frac{1}{n}S^2$$

（Ⅰ）证明 $E(T) = \mu^2$；　（Ⅱ）当 $\mu=0$，$\sigma=1$ 时，求 $D(T)$。

**解**：

**（Ⅰ）**

$$E(T) = E\!\left(\overline{X}^2\right) - \frac{1}{n}E(S^2)$$

由数字特征：$E(\overline{X}) = \mu$，$D(\overline{X}) = \dfrac{\sigma^2}{n}$，故

$$E(\overline{X}^2) = D(\overline{X}) + [E(\overline{X})]^2 = \frac{\sigma^2}{n} + \mu^2$$

又 $E(S^2) = \sigma^2$，故

$$E(T) = \frac{\sigma^2}{n} + \mu^2 - \frac{1}{n}\sigma^2 = \mu^2 \quad \blacksquare$$

**（Ⅱ）** 当 $\mu=0$，$\sigma=1$ 时：

$$T = \overline{X}^2 - \frac{1}{n}S^2$$

$$D(T) = D(\overline{X}^2) + \frac{1}{n^2}D(S^2) \quad \text{（}\overline{X}\text{ 与 }S^2\text{ 独立）}$$

由正态总体结论：$\overline{X} \sim N(0, \frac{1}{n})$，故 $\sqrt{n}\,\overline{X} \sim N(0,1)$，$n\overline{X}^2 \sim \chi^2(1)$，

$$D(n\overline{X}^2) = 2 \cdot 1 = 2 \Rightarrow D(\overline{X}^2) = \frac{2}{n^2}$$

又 $\dfrac{(n-1)S^2}{\sigma^2} = (n-1)S^2 \sim \chi^2(n-1)$，

$$D((n-1)S^2) = 2(n-1) \Rightarrow D(S^2) = \frac{2}{n-1}$$

故

$$D(T) = \frac{2}{n^2} + \frac{1}{n^2} \cdot \frac{2}{n-1} = \frac{2}{n^2} + \frac{2}{n^2(n-1)} = \frac{2(n-1)+2}{n^2(n-1)} = \boxed{\frac{2}{n(n-1)}}$$
