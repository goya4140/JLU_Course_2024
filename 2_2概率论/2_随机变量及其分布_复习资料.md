## 一、随机变量

### 引入

概率论的研究对象是"随机现象"，但用集合描述（事件）不够方便计算。引入**随机变量**，就是把随机试验的结果数字化——用一个实数值来代表每个结果，这样就可以用微积分等数学工具来分析。

### 知识点

**定义**：设随机试验的样本空间为 $\Omega$，$X = X(\omega)$ 是定义在 $\Omega$ 上的实值单值函数，则称 $X = X(\omega)$ 为**随机变量**。

> 一般用大写字母 $X, Y, Z$ 表示随机变量，小写字母 $x, y, z$ 表示随机变量的取值（实数）。

**分类**：
- **离散型**：所有可能取值是有限个或可列无限多个；
- **连续型**：存在概率密度函数 $f(x)$，使分布函数 $F(x) = \int_{-\infty}^{x} f(t)\,dt$；
- **非离散非连续型**：本课程了解即可。

## 二、分布函数

### 引入

无论是离散还是连续型随机变量，都有分布函数——它是一个统一的描述工具，把随机变量所有的概率信息都"装"进去了。

### 知识点

**定义**：设 $X$ 是一个随机变量，$x$ 是任意实数，函数

$$\boxed{F(x) = P\{X \leq x\},\quad -\infty < x < +\infty}$$

称为 $X$ 的**分布函数**。几何意义：$F(x)$ 表示 $X$ 落在实数轴 $x$ 左侧（含 $x$）的概率。

**利用 $F(x)$ 计算事件概率**（"宇宙唯二概率" + 扣）：

$$P\{X \leq a\} = F(a)$$

$$P\{X < a\} = F(a-0) = \lim_{x \to a^-} F(x)$$

由此推导：

| 事件 | 用 $F(x)$ 表示 |
|------|----------------|
| $P\{a < X \leq b\}$ | $F(b) - F(a)$ |
| $P\{a \leq X \leq b\}$ | $F(b) - F(a-0)$ |
| $P\{a \leq X < b\}$ | $F(b-0) - F(a-0)$ |
| $P\{a < X < b\}$ | $F(b-0) - F(a)$ |
| $P\{X = a\}$ | $F(a) - F(a-0)$ |
| $P\{X > a\}$ | $1 - F(a)$ |

> **记忆口诀**："等号跟大于走，永远右连续！"即 $P\{X \leq a\} = F(a)$（等号贴着 $F$），$P\{X < a\} = F(a-0)$（没有等号就取左极限）。

**分布函数的充要条件（四大性质）**：

1. **有界性**：$0 \leq F(x) \leq 1$；
2. **规范性**：$F(-\infty) = 0,\quad F(+\infty) = 1$；
3. **单调不减**：$\forall x_1 < x_2$，有 $F(x_1) \leq F(x_2)$；
4. **右连续**：$F(x_0) = F(x_0 + 0)$，即 $F$ 在任意点处右连续。

> **关于 F(x) 的常识结论**（速通讲义）：
> - $aF(x) + bG(x)$ 为分布函数 $\Leftrightarrow a + b = 1,\ a \geq 0,\ b \geq 0$；
> - $F(ax+b)$ 为分布函数 $\Leftrightarrow a > 0,\ b$ 为任意常数；
> - $F(x)G(x)$ 必为分布函数。

---

### 【例题·填空】例2.1（讲义）

设随机变量 $X$ 的分布函数

$$F(x) = \begin{cases} 0, & x < 0 \\ \dfrac{1}{2}, & 0 \leq x < 1 \\ 1 - e^{-x}, & x \geq 1 \end{cases}$$

则 $P\{X = 1\} = $\_\_\_\_，$P\left\{\dfrac{1}{3} < X < \dfrac{1}{2}\right\} = $\_\_\_\_。

**解**：

$P\{X = 1\} = F(1) - F(1-0) = (1 - e^{-1}) - \dfrac{1}{2} = \dfrac{1}{2} - e^{-1}$

$P\left\{\dfrac{1}{3} < X < \dfrac{1}{2}\right\} = F\!\left(\dfrac{1}{2}-0\right) - F\!\left(\dfrac{1}{3}\right) = \dfrac{1}{2} - \dfrac{1}{2} = 0$

（在 $F(x)$ 取常数的区间内，$X$ 取值概率为 $0$。）

$$\boxed{P\{X=1\} = \frac{1}{2} - e^{-1},\quad P\left\{\frac{1}{3}<X<\frac{1}{2}\right\} = 0}$$

---

### 【例题·填空】例2-2（速通）

设随机变量 $X$ 的分布函数为

$$F(x) = \begin{cases} A + Be^{-\lambda x}, & x > 0 \\ 0, & x \leq 0 \end{cases} \quad (\lambda > 0)$$

则 $A = $\_\_\_\_，$B = $\_\_\_\_。

**解**：利用充要条件：

① **规范性**：$F(+\infty) = A + B \cdot 0 = A = 1$，故 $A = 1$；

② **右连续**（在 $x = 0$ 处）：$F(0) = F(0+0)$，即 $0 = A + B$，故 $B = -1$。

$$\boxed{A = 1,\quad B = -1}$$

---

### 【例题·选择】例2.2（讲义）/ 例2-3（速通）

设 $F(x), G(x)$ 为分布函数，$a, b$ 为实数，则下列函数中是分布函数的是（　　）

(A) $\dfrac{2}{3}F(x) + \dfrac{2}{3}G(x)$　(B) $2F(x) - G(x)$　(C) $F(-x)$　(D) $F(x)G(x)$

**解**：

- (A)：$a + b = \dfrac{2}{3} + \dfrac{2}{3} = \dfrac{4}{3} \neq 1$，❌；
- (B)：$b = -1 < 0$，❌；
- (C)：$F(-x)$ 单调不增（非单调不减），❌；
- (D)：由常识结论，$F(x)G(x)$ **必为分布函数**，✅。

$$\boxed{\text{答案 D}}$$

## 三、离散型随机变量及其分布律

### 引入

如果随机变量 $X$ 的取值是"一粒一粒"可以数的，就是离散型。描述离散型 $X$ 的概率规律，最直接的方式是列出 $X$ 的每个取值对应的概率——这就是**分布律**。

### 知识点

**定义**：设离散型随机变量 $X$ 所有可能取值为 $x_1, x_2, \ldots, x_k, \ldots$，且

$$P\{X = x_k\} = p_k,\quad (k = 1, 2, \cdots)$$

称上式为 $X$ 的**概率分布**或**分布律**，也可用表格表示：

| $X$ | $x_1$ | $x_2$ | $\cdots$ | $x_n$ | $\cdots$ |
|-----|--------|--------|----------|--------|----------|
| $P$ | $p_1$  | $p_2$  | $\cdots$ | $p_n$  | $\cdots$ |

**充要条件**：
1. $p_k \geq 0,\quad k = 1, 2, \cdots$
2. $\displaystyle\sum_{k=1}^{\infty} p_k = 1$

**离散型 $X$ 的分布函数**：

$$F(x) = P\{X \leq x\} = \sum_{x_k \leq x} p_k$$

> 注：$F(x)$ 在 $x = x_k$ 处有跳跃，跳跃值为 $p_k = P\{X = x_k\}$。

---

### 【例题·计算】例2.1（讲义）/ 例2-4（速通）

一个袋中有 5 只球，编号 1, 2, 3, 4, 5，同时取 3 只，以 $X$ 表示取出的 3 只球中的**最大号码**（速通版为**最小号码**），求：(1) $X$ 的分布律；(2) $F(4.5)$；(3) 写出 $X$ 的分布函数。

（此处以讲义版**最大号码**为例）

**解**：

**（1）求分布律**：$X$ 可能取 3, 4, 5（最大号码至少为 3）。

$$P\{X=3\} = \frac{C_2^2}{C_5^3} = \frac{1}{10},\quad P\{X=4\} = \frac{C_3^2}{C_5^3} = \frac{3}{10},\quad P\{X=5\} = \frac{C_4^2}{C_5^3} = \frac{6}{10}$$

| $X$ | 3 | 4 | 5 |
|-----|---|---|---|
| $P$ | $\dfrac{1}{10}$ | $\dfrac{3}{10}$ | $\dfrac{6}{10}$ |

**（2）** $F(4.5) = P\{X \leq 4.5\} = P\{X=3\} + P\{X=4\} = \dfrac{1}{10} + \dfrac{3}{10} = \dfrac{4}{10} = \dfrac{2}{5}$

**（3）分布函数**：

$$F(x) = \begin{cases} 0, & x < 3 \\ \dfrac{1}{10}, & 3 \leq x < 4 \\ \dfrac{4}{10}, & 4 \leq x < 5 \\ 1, & x \geq 5 \end{cases}$$

## 四、常见离散型分布（四种）

### 1. 0-1 分布（两点分布）

**定义**：$X$ 只取 0 与 1，分布律为

$$\boxed{P\{X=k\} = p^k(1-p)^{1-k},\quad k=0,1\quad (0<p<1)}$$

或用表格：

| $X$ | 0 | 1 |
|-----|---|---|
| $P$ | $1-p$ | $p$ |

> **背景**：单次伯努利试验中，用 $X$ 记事件 $A$ 是否发生（发生记 1，不发生记 0）。

---

### 2. 二项分布 $B(n, p)$

#### 引入

做了 $n$ 次独立重复试验（每次成功概率为 $p$），关注成功次数 $X$——这就是二项分布的来源。

#### 知识点

**定义**：$n$ 重伯努利试验中，事件 $A$ 发生的次数 $X$ 的分布律为

$$\boxed{P\{X=k\} = C_n^k p^k (1-p)^{n-k},\quad k = 0, 1, \cdots, n}$$

记 $X \sim B(n, p)$。0-1 分布是 $B(1, p)$ 的特例。

> **做题翻译**（识别二项分布的三步）：
> 1. $n$ 次独立重复试验；
> 2. 每次 $P(A) = p$ 相同；
> 3. 研究 $A$ 发生的次数 $X$。
> 满足以上 $\Rightarrow X \sim B(n, p)$。

---

### 【例题·填空】例2.2（讲义）

某宿舍有 3 名学生参加概率论期末考试，通过率均为 $\dfrac{2}{3}$，则**至少两人挂科**的概率为\_\_\_\_。

**解**：挂科率 $q = 1 - \dfrac{2}{3} = \dfrac{1}{3}$，设挂科人数 $X \sim B\!\left(3, \dfrac{1}{3}\right)$。

$$P\{X \geq 2\} = P\{X=2\} + P\{X=3\}$$

$$= C_3^2 \left(\frac{1}{3}\right)^2 \left(\frac{2}{3}\right)^1 + C_3^3 \left(\frac{1}{3}\right)^3 = 3 \cdot \frac{1}{9} \cdot \frac{2}{3} + \frac{1}{27} = \frac{6}{27} + \frac{1}{27} = \frac{7}{27}$$

$$\boxed{\frac{7}{27}}$$

---

### 【例题·计算】例2-5（速通）

在区间 $(2,5)$ 上随机取一点，记坐标为 $X$，现对 $X$ 进行 3 次独立重复观测，求至少有两次观测值大于 3 的概率。

**解**：每次观测值大于 3 的概率（均匀分布）：

$$p = P\{X > 3\} = \frac{5-3}{5-2} = \frac{2}{3}$$

设大于 3 的次数 $Y \sim B\!\left(3, \dfrac{2}{3}\right)$，

$$P\{Y \geq 2\} = C_3^2 \left(\frac{2}{3}\right)^2 \left(\frac{1}{3}\right) + C_3^3 \left(\frac{2}{3}\right)^3 = \frac{12}{27} + \frac{8}{27} = \boxed{\frac{20}{27}}$$

---

### 【例题·填空 + 选择】例2-6（速通）——"二项、几何、帕斯卡，一题分清！"

某人向同一目标独立重复射击，每次命中概率为 $p(0<p<1)$，则：

(1) 射击 4 次，命中目标 2 次的概率为\_\_\_\_；

(2) 重复射击直到命中目标停止，总射击次数为 4 的概率是\_\_\_\_；

(3) 第 4 次射击恰好第 2 次命中目标的概率为\_\_\_\_；

**解**：

**(1) 二项分布**：固定次数，研究命中次数。$X \sim B(4, p)$。

$$P\{X=2\} = C_4^2 p^2 (1-p)^2 = \boxed{6p^2(1-p)^2}$$

**(2) 几何分布**：首次命中前的次数。第 4 次才第一次命中（前 3 次均未中）：

$$P = (1-p)^3 \cdot p = \boxed{(1-p)^3 p}$$

**(3) 帕斯卡分布（负二项分布）**：第 4 次恰好是第 2 次命中，即前 3 次中恰有 1 次命中，第 4 次必中：

$$P = C_3^1 p(1-p)^2 \cdot p = \boxed{3p^2(1-p)^2}$$

> **做题口诀**：二项=固定次数数命中；几何=第一次命中等多久；帕斯卡=第 $r$ 次命中等多久。

---

### 3. 几何分布 $Ge(p)$

**定义**：可列重伯努利试验中，$X$ 表示**事件 $A$ 首次发生时的试验次数**：

$$\boxed{P\{X=k\} = (1-p)^{k-1} p,\quad k = 1, 2, \cdots}$$

记 $X \sim Ge(p)$。

---

### 【例题·填空】例2.5（讲义）

对目标进行独立射击，每次命中率均为 $\dfrac{1}{4}$，重复射击直至命中目标为止，设 $X$ 表示射击次数，则 $P\{X=3\} = $\_\_\_\_。

**解**：$X \sim Ge\!\left(\dfrac{1}{4}\right)$，

$$P\{X=3\} = \left(\frac{3}{4}\right)^2 \cdot \frac{1}{4} = \boxed{\frac{9}{64}}$$

---

### 4. 泊松分布 $P(\lambda)$

#### 引入

当二项分布中 $n$ 很大、$p$ 很小（稀有事件）时，$B(n,p)$ 的计算很麻烦，此时用**泊松分布**近似更方便。泊松分布本身也直接描述"单位时间/空间内随机事件发生次数"。

#### 知识点

**定义**：$X$ 的所有可能取值为 $0, 1, 2, \cdots$，分布律为

$$\boxed{P\{X=k\} = \frac{\lambda^k e^{-\lambda}}{k!},\quad k = 0, 1, 2, \cdots \quad (\lambda > 0)}$$

记 $X \sim P(\lambda)$ 或 $X \sim \pi(\lambda)$。

**泊松定理**（二项→泊松近似）：当 $n$ 很大（$n \geq 20$）、$p$ 很小（$p \leq 0.05$）时，设 $\lambda = np$，则

$$\boxed{C_n^k p^k (1-p)^{n-k} \approx \frac{\lambda^k e^{-\lambda}}{k!}}$$

---

### 【例题·填空】例2.7（讲义）

设 1 分钟内进入图书馆的人数 $X$ 服从参数为 $\lambda$ 的泊松分布，且已知 $P\{X=2\} = P\{X=3\}$，则 1 分钟内仅 1 人进入图书馆的概率为\_\_\_\_。

**解**：由 $P\{X=2\} = P\{X=3\}$：

$$\frac{\lambda^2 e^{-\lambda}}{2!} = \frac{\lambda^3 e^{-\lambda}}{3!} \implies \frac{\lambda^2}{2} = \frac{\lambda^3}{6} \implies \lambda = 3$$

$$P\{X=1\} = \frac{3^1 e^{-3}}{1!} = \boxed{3e^{-3}}$$

---

### 【例题·填空】例2.8（讲义）

设随机变量 $X$ 的概率分布为 $P\{X=k\} = \dfrac{C}{k!}\ (k=0,1,2,\cdots)$，则 $P\{X \geq 2\} = $\_\_\_\_。

**解**：由规范性 $\displaystyle\sum_{k=0}^{\infty} \frac{C}{k!} = C \cdot e^1 = 1$，故 $C = e^{-1}$，即 $X \sim P(1)$。

$$P\{X \geq 2\} = 1 - P\{X=0\} - P\{X=1\} = 1 - e^{-1} - e^{-1} = \boxed{1 - 2e^{-1}}$$

## 五、连续型随机变量及其概率密度

### 引入

如果随机变量取值"连续地"充满某个区间，就不能再用分布律了——单个点的概率为 0。改用**概率密度函数** $f(x)$，用积分来算概率。

### 知识点

**定义**：若存在非负可积函数 $f(x)$，使对任意实数 $x$ 有

$$\boxed{F(x) = \int_{-\infty}^{x} f(t)\,dt}$$

则称 $X$ 为**连续型随机变量**，$f(x)$ 称为 $X$ 的**概率密度函数**（概率密度）。

**概率密度的性质（充要条件）**：

1. **非负性**：$f(x) \geq 0$；
2. **规范性**：$\displaystyle\int_{-\infty}^{+\infty} f(x)\,dx = 1$；
3. 对任意 $a \leq b$：$P\{a < X \leq b\} = F(b) - F(a) = \displaystyle\int_a^b f(x)\,dx$；
4. 若 $f(x)$ 在点 $x$ 连续，则 $F'(x) = f(x)$。

**连续型随机变量的重要性质**：
- $F(x)$ 是连续函数；
- 对任意 $a \in \mathbb{R}$，$P\{X = a\} = 0$（单点概率为零，但不是不可能事件！）；
- $P\{a < X \leq b\} = P\{a \leq X \leq b\} = P\{a < X < b\} = P\{a \leq X < b\}$（端点是否含入不影响概率）。

> **小秘杀**（速通讲义）：若 $X$（任意类型）可能取 $a \leq X \leq b$，则当 $x < a$ 时 $F(x) = 0$；当 $x \geq b$ 时 $F(x) = 1$。

---

### 【例题·计算】例2.15（讲义）/ 例2-8（速通）

设连续型随机变量 $X$ 的概率密度为

$$f(x) = \begin{cases} x, & 0 < x < 1, \\ k(2-x), & 1 < x < 2, \\ 0, & \text{其他,} \end{cases}$$

求：(1) $k$ 的值；(2) $P\!\left\{\dfrac{1}{2} < X \leq \dfrac{3}{2}\right\}$；(3) $P\!\left\{X = \dfrac{5}{2}\right\}$；(4) $X$ 的分布函数；(5) $P\!\left\{X > \dfrac{1}{2} \,\middle|\, X \leq 1\right\}$

**解**：

**(1)** 由规范性：

$$\int_0^1 x\,dx + \int_1^2 k(2-x)\,dx = \frac{1}{2} + k \cdot \frac{1}{2} = 1 \implies \boxed{k = 1}$$

**(2)** 

$$P\left\{\frac{1}{2}<X\leq\frac{3}{2}\right\} = \int_{1/2}^{1} x\,dx + \int_1^{3/2} (2-x)\,dx = \frac{3}{8} + \frac{3}{8} = \boxed{\frac{3}{4}}$$

**(3)** $P\!\left\{X = \dfrac{5}{2}\right\} = 0$（连续型随机变量单点概率为零）

**(4)** 分布函数（对 $f(x)$ 分段积分）：

$$F(x) = \begin{cases} 0, & x \leq 0 \\ \dfrac{x^2}{2}, & 0 < x \leq 1 \\ -\dfrac{x^2}{2} + 2x - 1, & 1 < x \leq 2 \\ 1, & x > 2 \end{cases}$$

**(5)** 

$$P\left\{X>\frac{1}{2}\,\middle|\,X\leq 1\right\} = \frac{P\!\left\{\frac{1}{2}<X\leq 1\right\}}{P\{X\leq 1\}} = \frac{F(1)-F(1/2)}{F(1)} = \frac{\frac{1}{2} - \frac{1}{8}}{\frac{1}{2}} = \boxed{\frac{3}{4}}$$

## 六、常见连续型分布（三种）

### 1. 均匀分布 $U(a, b)$

#### 引入

"等可能地"在某区间上取值——就是均匀分布。几何概型的连续版本。

#### 知识点

**概率密度**：

$$\boxed{f(x) = \begin{cases} \dfrac{1}{b-a}, & a < x < b, \\ 0, & \text{其他,} \end{cases}}$$

记 $X \sim U(a, b)$。

**分布函数**：

$$F(x) = \begin{cases} 0, & x < a, \\ \dfrac{x-a}{b-a}, & a \leq x < b, \\ 1, & x \geq b. \end{cases}$$

> **求概率两法**：① 区间长度之比（直接用）；② 对 $f(x)$ 积分。

---

### 【例题·填空】例2.18（讲义）/ 例2-9（速通）

设随机变量 $X$ 在 $[1, 6]$ 上服从均匀分布，则 $X$ 的概率密度为\_\_\_\_，分布函数为\_\_\_\_，方程 $t^2 + Xt + 1 = 0$ 有实根的概率为\_\_\_\_。

**解**：$X \sim U(1, 6)$，

$$f(x) = \begin{cases} \dfrac{1}{5}, & 1 \leq x \leq 6 \\ 0, & \text{其他} \end{cases} \qquad F(x) = \begin{cases} 0, & x < 1 \\ \dfrac{x-1}{5}, & 1 \leq x < 6 \\ 1, & x \geq 6 \end{cases}$$

方程有实根 $\Leftrightarrow \Delta = X^2 - 4 \geq 0 \Leftrightarrow X \geq 2$ 或 $X \leq -2$。

由于 $X \in [1,6]$，实际条件为 $X \geq 2$：

$$P\{X \geq 2\} = \frac{6-2}{6-1} = \boxed{\frac{4}{5}}$$

---

### 2. 指数分布 $E(\lambda)$

#### 引入

描述"等待时间"或"寿命"的分布：等公交车、设备无故障运行时间……指数分布有一个迷人性质——**无记忆性**。

#### 知识点

**概率密度**：

$$\boxed{f(x) = \begin{cases} \lambda e^{-\lambda x}, & x > 0, \\ 0, & x \leq 0 \end{cases} \quad (\lambda > 0)}$$

记 $X \sim E(\lambda)$。

**分布函数**：

$$F(x) = \begin{cases} 1 - e^{-\lambda x}, & x \geq 0, \\ 0, & x < 0. \end{cases}$$

**重要结论**：

$$\boxed{P\{X > a\} = e^{-\lambda a} \quad (a > 0)}$$

**无记忆性**：对任意 $s, t > 0$，

$$\boxed{P\{X > s+t \mid X > s\} = P\{X > t\}}$$

> 直觉：已经等了 $s$ 分钟，再等 $t$ 分钟的概率，和一开始就等 $t$ 分钟一样——指数分布"不记得"已经等了多久。

---

### 【例题·填空】例2.21（讲义）/ 例2-10（速通）

设随机变量 $X$ 服从参数为 2 的指数分布，$a$ 为常数且大于零，则：

$X$ 的概率密度为\_\_\_\_，分布函数为\_\_\_\_，$P\{X \leq a+1 \mid X > a\} = $\_\_\_\_，$F(2) = $\_\_\_\_，$P\{X=1\} = $\_\_\_\_。

**解**：$X \sim E(2)$，

$$f(x) = \begin{cases} 2e^{-2x}, & x > 0 \\ 0, & x \leq 0 \end{cases} \qquad F(x) = \begin{cases} 1 - e^{-2x}, & x \geq 0 \\ 0, & x < 0 \end{cases}$$

由无记忆性：$P\{X \leq a+1 \mid X > a\} = P\{X \leq 1\} = F(1) = 1 - e^{-2}$

$$F(2) = 1 - e^{-4}$$

$$P\{X=1\} = 0 \quad \text{（连续型单点概率为零）}$$

$$\boxed{P\{X \leq a+1 \mid X > a\} = 1 - e^{-2},\quad F(2) = 1 - e^{-4},\quad P\{X=1\} = 0}$$

---

### 3. 正态分布 $N(\mu, \sigma^2)$

#### 引入

正态分布是概率论中最重要的分布——自然界中大量随机现象（如身高、测量误差）都近似服从正态分布，且它是中心极限定理的核心。

#### 知识点

**概率密度**：

$$\boxed{f(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}},\quad -\infty < x < +\infty}$$

记 $X \sim N(\mu, \sigma^2)$，参数 $\mu$ 为均值（对称轴），$\sigma$ 为标准差（控制胖瘦）。

**必考性质**：若 $X \sim N(\mu, \sigma^2)$，则：

1. $P\{X > \mu\} = P\{X < \mu\} = \dfrac{1}{2}$；
2. $Y = aX + b \sim N(a\mu + b,\ a^2\sigma^2),\ a \neq 0$（线性变换仍正态）；
3. 若 $X \sim N(\mu_1, \sigma_1^2)$，$Y \sim N(\mu_2, \sigma_2^2)$ 且独立，则 $aX + bY \sim N(a\mu_1 + b\mu_2,\ a^2\sigma_1^2 + b^2\sigma_2^2)$。

**标准正态分布** $N(0, 1)$：$\mu = 0, \sigma = 1$，概率密度记 $\varphi(x)$，分布函数记 $\Phi(x)$。

**标准化**：若 $X \sim N(\mu, \sigma^2)$，令 $Z = \dfrac{X - \mu}{\sigma} \sim N(0,1)$，则

$$\boxed{P\{a < X \leq b\} = \Phi\!\left(\frac{b-\mu}{\sigma}\right) - \Phi\!\left(\frac{a-\mu}{\sigma}\right)}$$

**$\Phi(x)$ 的性质**：

$$\boxed{\Phi(-x) = 1 - \Phi(x)} \qquad \Phi(0) = \frac{1}{2} \qquad P\{|X| \leq a\} = 2\Phi(a) - 1 \ (a>0)$$

> **记忆口诀**：正态用自己的办法——标准化！遇到正态分布就把它化成 $\Phi$ 函数。

---

### 【例题·填空】例2.22（讲义）

设随机变量 $X \sim N(2, \sigma^2)$ 且 $P\{2 < X < 4\} = 0.3$，则 $P\{0 < X < 2\} = $\_\_\_\_。

**解**：由正态分布关于 $x = \mu = 2$ 对称：

$$P\{2 < X < 4\} = P\{2-2 < X-2 < 4-2\} \longleftrightarrow P\{0 < X < 2\} = P\{-2 < X-2 < 0\}$$

由对称性，$P\{0 < X < 2\} = P\{2 < X < 4\} = \boxed{0.3}$

---

### 【例题·填空】例2.23（讲义）

设 $X \sim N(\mu, \sigma^2),\ \sigma > 0$，且二次方程 $y^2 + 4y + X = 0$ 有实根的概率为 $0.5$，则 $\mu = $\_\_\_\_。

**解**：方程有实根 $\Leftrightarrow \Delta = 16 - 4X \geq 0 \Leftrightarrow X \leq 4$。

$$P\{X \leq 4\} = 0.5 = \Phi\!\left(\frac{4-\mu}{\sigma}\right)$$

由 $\Phi(0) = 0.5$，得 $\dfrac{4-\mu}{\sigma} = 0$，即 $\mu = \boxed{4}$。

---

### 【例题·填空】例2.27（讲义）/ 例2-11（速通）

设随机变量 $X$ 的概率密度为 $f(x) = ae^{-\frac{1}{4}x^2 + x}\ (-\infty < x < +\infty)$，则 $P\{-2\sqrt{2}+2 < X < 2\sqrt{2}+2\} = $\_\_\_\_（用 $\Phi(x)$ 表示）。

**解**：概率密度形如 $f(x) = Ae^{ax^2+bx+c}\ (a<0)$ 必为正态分布。

配方：$-\dfrac{1}{4}x^2 + x = -\dfrac{1}{4}(x^2 - 4x) = -\dfrac{1}{4}(x-2)^2 + 1$

故 $f(x) \propto e^{-\frac{(x-2)^2}{4}}$，对比正态密度得 $\mu = 2,\ 2\sigma^2 = 4 \Rightarrow \sigma^2 = 2,\ \sigma = \sqrt{2}$。

即 $X \sim N(2, 2)$，注意 $-2\sqrt{2}+2 = \mu - 2\sigma$，$2\sqrt{2}+2 = \mu + 2\sigma$：

$$P\{\mu - 2\sigma < X < \mu + 2\sigma\} = \Phi(2) - \Phi(-2) = \boxed{2\Phi(2) - 1}$$

## 七、随机变量函数 $Y = g(X)$ 的分布

### 引入

已知 $X$ 的分布，求 $Y = g(X)$ 的分布——这是本章的压轴题型。核心思路：先搞清楚 $Y$ 的取值范围，再通过分布函数把 $Y$ 的概率转化为 $X$ 的概率。

### 1. 离散型 $X$ ——列表法

**步骤**：
1. 求 $Y$ 所有可能取值（有重复的合并）；
2. 依次计算 $Y$ 取各值的概率。

---

### 【例题·计算】例2.28（讲义）

设 $X$ 的分布律

$$X \sim \begin{pmatrix} -1 & 0 & 1 & 2 \\ \frac{1}{4} & \frac{1}{3} & \frac{1}{6} & \frac{1}{4} \end{pmatrix}$$

求 $Y = X^2$ 的概率分布。

**解**：$Y$ 的可能取值：$0, 1, 4$（注意 $(-1)^2 = 1^2 = 1$，合并）。

$$P\{Y=0\} = P\{X=0\} = \frac{1}{3}$$

$$P\{Y=1\} = P\{X=-1\} + P\{X=1\} = \frac{1}{4} + \frac{1}{6} = \frac{5}{12}$$

$$P\{Y=4\} = P\{X=2\} = \frac{1}{4}$$

| $Y$ | 0 | 1 | 4 |
|-----|---|---|---|
| $P$ | $\dfrac{1}{3}$ | $\dfrac{5}{12}$ | $\dfrac{1}{4}$ |

---

### 2. 连续型 $X$ ——分布函数法（万能套路）

**步骤**：

**第一步**：确定 $Y$ 的可能取值范围（作 $Y = g(X)$ 的图像，定义域限 $X$ 可能取到的范围，值域即 $Y$ 范围）。

**第二步**：写"三件套"：

$$F_Y(y) = P\{Y \leq y\} = P\{g(X) \leq y\}$$

将 $\{g(X) \leq y\}$ 转化为关于 $X$ 的事件（作直线 $Y = y$，直线下方即 $\{Y \leq y\}$，分情况讨论对应的 $X$ 范围）。

**第三步**：对 $F_Y(y)$ 求导，得 $f_Y(y)$。

---

### 【例题·计算】例2.29（讲义）/ 例2-13（速通）

设随机变量 $X$ 具有概率密度

$$f_X(x) = \begin{cases} \dfrac{x}{8}, & 0 < x < 4, \\ 0, & \text{其他.} \end{cases}$$

求 $Y = 2X + 8$ 的概率密度。

**解**：

**第一步**：$X \in (0,4)$，故 $Y = 2X+8 \in (8, 16)$。

**第二步**（三件套）：当 $8 < y < 16$ 时，

$$F_Y(y) = P\{2X+8 \leq y\} = P\left\{X \leq \frac{y-8}{2}\right\} = F_X\!\left(\frac{y-8}{2}\right) = \int_0^{\frac{y-8}{2}} \frac{x}{8}\,dx = \frac{(y-8)^2}{64}$$

**第三步**：

$$f_Y(y) = F_Y'(y) = \begin{cases} \dfrac{y-8}{32}, & 8 < y < 16, \\ 0, & \text{其他.} \end{cases}$$

$$\boxed{f_Y(y) = \begin{cases} \dfrac{y-8}{32}, & 8 < y < 16 \\ 0, & \text{其他} \end{cases}}$$

---

### 【例题·计算】例2.31（讲义）

设 $X \sim U(-1, 2)$，求 $Y = |X|$ 的概率密度函数。

**解**：$f_X(x) = \dfrac{1}{3}$（$-1 < x < 2$），$Y = |X| \in [0, 2)$。

分两段讨论：

**当 $0 < y < 1$ 时**：$F_Y(y) = P\{|X| \leq y\} = P\{-y \leq X \leq y\} = \displaystyle\int_{-y}^{y} \frac{1}{3}\,dx = \dfrac{2y}{3}$

**当 $1 \leq y < 2$ 时**：$F_Y(y) = P\{-y \leq X \leq y\} = P\{-1 \leq X \leq y\} = \displaystyle\int_{-1}^{y} \frac{1}{3}\,dx = \dfrac{y+1}{3}$

求导：

$$\boxed{f_Y(y) = \begin{cases} \dfrac{2}{3}, & 0 < y < 1 \\ \dfrac{1}{3}, & 1 \leq y < 2 \\ 0, & \text{其他} \end{cases}}$$

---

### 【例题·选择】例2.33（讲义）

设随机变量 $X$ 具有概率密度 $f_X(x) = 3e^{-3x}\ (x \geq 0)$，$0\ (x < 0)$，则 $Y = 1 - e^{-3X}$ 服从（　　）

(A) 指数分布　　(B) 正态分布　　(C) 泊松分布　　(D) 均匀分布

**解**：注意 $X \sim E(3)$，其分布函数为 $F_X(x) = 1 - e^{-3x}\ (x \geq 0)$。

因为 $F_X(x)$ 严格单增（当 $0 < F_X(x) < 1$ 时），由定理：$Y = F_X(X) \sim U(0,1)$。

$$\boxed{\text{答案 D}}$$
