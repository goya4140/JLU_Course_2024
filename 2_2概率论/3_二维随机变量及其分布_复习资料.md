## 一、二维随机变量与联合分布函数

### 引入

一维随机变量只能描述单个指标（身高、成绩……），但现实中往往需要同时描述多个指标的联合行为，比如"身高 $X$ 和体重 $Y$ 的联合分布"。这就需要引入**多维随机变量**。

### 知识点

**定义**：设 $X = X(\omega)$，$Y = Y(\omega)$ 是定义在同一样本空间上的两个随机变量，则称向量 $(X, Y)$ 为**二维随机变量**（随机向量）。

**联合分布函数**：对任意实数 $x, y$，定义

$$\boxed{F(x, y) = P\{X \leq x, Y \leq y\}}$$

它表示随机点 $(X, Y)$ 落在"左下方无穷矩形域"中的概率。

**分布函数的性质**（四条）：
1. **非负性**：$0 \leq F(x, y) \leq 1$；
2. **规范性**：$F(-\infty, y) = F(x, -\infty) = 0$，$F(+\infty, +\infty) = 1$；
3. **单调不减性**：关于 $x$ 和 $y$ 各自单调不减；
4. **右连续性**：$F(x+0, y) = F(x, y)$，$F(x, y+0) = F(x, y)$。

**矩形区域概率公式**（重要）：

$$\boxed{P\{x_1 < X \leq x_2,\ y_1 < Y \leq y_2\} = F(x_2,y_2) - F(x_1,y_2) - F(x_2,y_1) + F(x_1,y_1)}$$

> 记忆口诀：**右上角 $-$ 左上角 $-$ 右下角 $+$ 左下角**（容斥原理）。

**边缘分布函数**：

$$\boxed{F_X(x) = F(x, +\infty), \quad F_Y(y) = F(+\infty, y)}$$

**独立性**：$X$ 与 $Y$ 相互独立当且仅当

$$\boxed{F(x, y) = F_X(x) \cdot F_Y(y) \quad \text{对所有 } x, y \text{ 成立}}$$

## 二、二维离散型随机变量

### 引入

若两个随机变量 $X, Y$ 都只取有限或可列无限个值，则 $(X, Y)$ 为**二维离散型随机变量**。描述它只需一张表——**联合分布律表**，掌握这张表就能回答所有问题。

### 知识点

**联合分布律**：

$$P\{X = x_i, Y = y_j\} = p_{ij}, \quad i, j = 1, 2, \cdots$$

充要条件：$p_{ij} \geq 0$，且 $\displaystyle\sum_i \sum_j p_{ij} = 1$。

用表格表示：

|   | $y_1$ | $y_2$ | $\cdots$ | 行和 $p_{i\cdot}$ |
|---|-------|-------|----------|-----------------|
| $x_1$ | $p_{11}$ | $p_{12}$ | $\cdots$ | $p_{1\cdot}$ |
| $x_2$ | $p_{21}$ | $p_{22}$ | $\cdots$ | $p_{2\cdot}$ |
| $\vdots$ | $\vdots$ | $\vdots$ | | $\vdots$ |
| 列和 $p_{\cdot j}$ | $p_{\cdot 1}$ | $p_{\cdot 2}$ | $\cdots$ | $1$ |

**边缘分布律**（行和/列和）：

$$\boxed{P\{X = x_i\} = p_{i\cdot} = \sum_j p_{ij}, \quad P\{Y = y_j\} = p_{\cdot j} = \sum_i p_{ij}}$$

**条件分布律**（固定一个变量，在此条件下另一个变量的分布）：

$$\boxed{P\{X = x_i \mid Y = y_j\} = \frac{p_{ij}}{p_{\cdot j}}, \quad P\{Y = y_j \mid X = x_i\} = \frac{p_{ij}}{p_{i\cdot}}}$$

**独立性充要条件**：

$$\boxed{p_{ij} = p_{i\cdot} \cdot p_{\cdot j} \quad \text{对所有 } i, j \text{ 成立}}$$

> 做题翻译：表中**每一个格子的概率 = 该格子所在行和 × 列和**，全部满足才独立。**只要有一个格子不满足，就不独立。**

### 【例题 3-1】计算题——离散型综合

**题目**（讲义例3.3）：盒子里有2个黑球、2个红球、2个白球，从中随机任取2个球，$X$ 表示取得的黑球个数，$Y$ 表示取得的红球个数。

(1) 试写出 $X$ 和 $Y$ 的联合分布律；(2) 求 $P\{X + Y \leq 1\}$；(3) 求 $X$ 的边缘分布律；(4) 求条件概率 $P\{Y = 1 \mid X = 0\}$；(5) 判断 $X$ 和 $Y$ 的独立性。

**解**：

总共从6球取2球：$C_6^2 = 15$ 种。

$X$ 的可能取值：$0, 1, 2$；$Y$ 的可能取值：$0, 1, 2$（约束条件：$X + Y \leq 2$）。

各 $p_{ij}$ 计算：

$$p_{00} = \frac{C_2^2}{C_6^2} = \frac{1}{15}, \quad p_{10} = \frac{C_2^1 C_2^2}{C_6^2} = \frac{2}{15}, \quad p_{20} = \frac{C_2^2}{C_6^2} = \frac{1}{15}$$

$$p_{01} = \frac{C_2^1 C_2^2}{C_6^2} = \frac{2}{15}, \quad p_{11} = \frac{C_2^1 C_2^1}{C_6^2} = \frac{4}{15}, \quad p_{21} = \frac{C_2^2 C_2^1}{C_6^2} = \frac{2}{15}$$

$$p_{02} = \frac{C_2^2}{C_6^2} = \frac{1}{15}, \quad p_{12} = \frac{C_2^1 C_2^2}{C_6^2} = \frac{2}{15}, \quad p_{22} = 0$$

联合分布律表：

|  $X \backslash Y$  | 0 | 1 | 2 | $p_{i\cdot}$ |
|---|---|---|---|---|
| 0 | 1/15 | 2/15 | 1/15 | **4/15** |
| 1 | 2/15 | 4/15 | 2/15 | **8/15** |
| 2 | 1/15 | 2/15 | 0 | **3/15** |
| $p_{\cdot j}$ | **4/15** | **8/15** | **3/15** | 1 |

**(1)** 联合分布律见上表。

**(2)** $X + Y \leq 1$ 的格子：$(0,0), (1,0), (0,1)$，

$$P\{X + Y \leq 1\} = \frac{1}{15} + \frac{2}{15} + \frac{2}{15} = \frac{5}{15} = \frac{1}{3}$$

**(3)** $X$ 的边缘分布律（行和）：

| $X$ | 0 | 1 | 2 |
|---|---|---|---|
| $P$ | 4/15 | 8/15 | 3/15 |

**(4)** $P\{Y = 1 \mid X = 0\} = \dfrac{p_{01}}{p_{0\cdot}} = \dfrac{2/15}{4/15} = \dfrac{1}{2}$

**(5)** 验证独立性：取 $i=0, j=0$：$p_{00} = \dfrac{1}{15}$，而 $p_{0\cdot} \cdot p_{\cdot 0} = \dfrac{4}{15} \times \dfrac{4}{15} = \dfrac{16}{225} \neq \dfrac{1}{15}$。

∴ $X$ 与 $Y$ **不独立**。

### 【例题 3-2】填空题——由独立性求参数

**题目**（讲义例3.4）：已知随机变量 $(X,Y)$ 的分布律为

| $X \backslash Y$ | 0 | 1 |
|---|---|---|
| 0 | 0.15 | 0.15 |
| 1 | $a$ | $b$ |

且 $X$ 与 $Y$ 相互独立，则 $a = $ \_\_\_\_，$b = $ \_\_\_\_。

**解**：

先求边缘分布：$p_{0\cdot} = 0.30$，$p_{1\cdot} = a + b$，故 $0.30 + a + b = 1$，即 $a + b = 0.70$。

$p_{\cdot 0} = 0.15 + a$，$p_{\cdot 1} = 0.15 + b$。

由独立性，$p_{00} = p_{0\cdot} \cdot p_{\cdot 0}$：

$$0.15 = 0.30 \times (0.15 + a) \Rightarrow 0.15 + a = 0.5 \Rightarrow a = 0.35$$

$$\boxed{a = 0.35, \quad b = 0.35}$$

验证：$p_{01} = 0.15 = 0.30 \times 0.50$ ✓，$p_{10} = 0.35 = 0.70 \times 0.50$ ✓，$p_{11} = 0.35 = 0.70 \times 0.50$ ✓。

### 【例题 3-3】计算题——已知边缘分布律和附加条件求联合分布律

**题目**（讲义例3.6）：已知随机变量 $X$ 和 $Y$ 的概率分布为

$$X \sim \begin{pmatrix} -1 & 0 & 1 \\ \frac{1}{4} & \frac{1}{2} & \frac{1}{4} \end{pmatrix}, \quad Y \sim \begin{pmatrix} 0 & 1 \\ \frac{1}{2} & \frac{1}{2} \end{pmatrix}$$

且 $P\{XY = 0\} = 1$，求 $X$ 和 $Y$ 的联合分布律。

**解**：

联合分布律表有 $3 \times 2 = 6$ 个格子，设为 $p_{ij}$。

条件 $P\{XY = 0\} = 1$ 意味着 $P\{XY \neq 0\} = 0$，即 $XY \neq 0$ 的情况概率为0。

$XY \neq 0$ 当且仅当 $X \neq 0$ 且 $Y \neq 0$，即 $(X, Y) \in \{(-1,1),(1,1)\}$，故

$$p_{-1,1} = 0, \quad p_{1,1} = 0$$

剩余4个格子：$p_{-1,0}, p_{0,0}, p_{1,0}, p_{0,1}$。

由边缘分布律：
- $P\{X = -1\} = p_{-1,0} + p_{-1,1} = p_{-1,0} = \frac{1}{4}$，故 $p_{-1,0} = \frac{1}{4}$；
- $P\{X = 1\} = p_{1,0} + p_{1,1} = p_{1,0} = \frac{1}{4}$，故 $p_{1,0} = \frac{1}{4}$；
- $P\{Y = 1\} = p_{-1,1} + p_{0,1} + p_{1,1} = p_{0,1} = \frac{1}{2}$；
- 故 $p_{0,0} = P\{X=0\} - p_{0,1} = \frac{1}{2} - \frac{1}{2} = 0$。

联合分布律：

| $X \backslash Y$ | 0 | 1 |
|---|---|---|
| $-1$ | 1/4 | 0 |
| $0$ | 0 | 1/2 |
| $1$ | 1/4 | 0 |

## 三、二维连续型随机变量

### 引入

如果随机点 $(X, Y)$ 均匀洒在某个平面区域上，一维随机变量的密度函数"曲线下面积=概率"对应到二维就是"曲面下体积=概率"，这就是**联合概率密度**的直觉。

### 知识点

**联合概率密度**：若存在非负可积函数 $f(x,y)$，使得

$$\boxed{F(x, y) = \int_{-\infty}^{x} \int_{-\infty}^{y} f(u, v)\, du\, dv}$$

则称 $(X, Y)$ 为**二维连续型随机变量**，$f(x, y)$ 为其**联合概率密度**（联合密度函数）。

**充要条件**（两条）：
1. **非负性**：$f(x, y) \geq 0$；
2. **规范性**：$\displaystyle\int_{-\infty}^{+\infty} \int_{-\infty}^{+\infty} f(x, y)\, dx\, dy = 1$。

**求概率**（核心公式）：

$$\boxed{P\{(X, Y) \in G\} = \iint_G f(x, y)\, dx\, dy}$$

> 做题翻译：把概率事件翻译成 $xy$ 平面上的区域 $G$，在 $G$ 上对 $f(x,y)$ 做二重积分。

**由 $F(x,y)$ 反求 $f(x,y)$**：

$$\boxed{f(x, y) = \frac{\partial^2 F(x, y)}{\partial x \partial y}}$$

（在 $f$ 的连续点处成立）

### 【例题 3-4】计算题——求常数与概率

**题目**（讲义例3.8）：设 $(X,Y)$ 的密度函数为

$$f(x,y) = \begin{cases} Cxy, & 0 \leq x \leq 1,\ 0 \leq y \leq 1 \\ 0, & \text{其他} \end{cases}$$

求：(I) 常数 $C$；(II) $P\{X + Y < 1\}$；(III) 联合分布函数 $F(x,y)$。

**解**：

**(I)** 由规范性：

$$\int_0^1 \int_0^1 Cxy\, dx\, dy = C \cdot \frac{1}{2} \cdot \frac{1}{2} = \frac{C}{4} = 1 \Rightarrow \boxed{C = 4}$$

**(II)** 区域 $G: x + y < 1$，$0 \leq x \leq 1$，$0 \leq y \leq 1$，即三角形 $\{x \geq 0, y \geq 0, x+y \leq 1\}$：

$$P\{X + Y < 1\} = \int_0^1 \int_0^{1-x} 4xy\, dy\, dx = \int_0^1 4x \cdot \frac{(1-x)^2}{2}\, dx = \int_0^1 2x(1-x)^2\, dx$$

$$= 2\int_0^1 (x - 2x^2 + x^3)\, dx = 2\left[\frac{1}{2} - \frac{2}{3} + \frac{1}{4}\right] = 2 \cdot \frac{1}{12} = \boxed{\frac{1}{6}}$$

**(III)** 分区域讨论（记 $x_0 = \min(x,1)$，$y_0 = \min(y,1)$）：

当 $0 \leq x \leq 1$，$0 \leq y \leq 1$ 时：

$$F(x,y) = \int_0^x \int_0^y 4uv\, dv\, du = x^2 y^2$$

其他区域按边界值取：
- $x > 1, 0 \leq y \leq 1$：$F = y^2$；
- $0 \leq x \leq 1, y > 1$：$F = x^2$；
- $x > 1, y > 1$：$F = 1$；
- $x < 0$ 或 $y < 0$：$F = 0$。

## 四、边缘概率密度

### 引入

有了联合密度 $f(x,y)$，如何描述单个变量 $X$ 的分布？就是把 $Y$ "积掉"——对 $y$ 从 $-\infty$ 到 $+\infty$ 积分，这就是**边缘概率密度**。

> ⚠️ **积分上下限是本章最大难点**：必须根据 $f(x,y)$ 的非零区域，在固定 $x$（或 $y$）后，确定另一个变量的有效范围。

### 知识点

$$\boxed{f_X(x) = \int_{-\infty}^{+\infty} f(x, y)\, dy, \quad f_Y(y) = \int_{-\infty}^{+\infty} f(x, y)\, dx}$$

**做题步骤**：
1. 画出 $f(x,y)$ 的非零区域（草图必画！）；
2. 固定 $x$，确定 $y$ 的有效范围（受非零区域约束）；
3. 在该范围内对 $y$ 积分；
4. 结果仍要分情况讨论（$x$ 不同范围积分限不同）。

### 【例题 3-5】计算题——非矩形区域的边缘密度

**题目**（讲义例3.9）：设 $(X,Y)$ 的密度函数为

$$f(x,y) = \begin{cases} kxy, & 0 \leq x \leq y \leq 1 \\ 0, & \text{其他} \end{cases}$$

求：(I) 参数 $k$；(II) $X$ 和 $Y$ 的边缘概率密度 $f_X(x)$，$f_Y(y)$。

**解**：

**(I)** 非零区域是三角形 $\{0 \leq x \leq y \leq 1\}$，规范性：

$$\int_0^1 \int_x^1 kxy\, dy\, dx = k\int_0^1 x \cdot \frac{1-x^2}{2}\, dx = \frac{k}{2} \cdot \frac{1}{4} = \frac{k}{8} = 1 \Rightarrow \boxed{k = 8}$$

**(II)** 求 $f_X(x)$：固定 $x \in [0,1]$，$y$ 的范围为 $[x, 1]$（由 $x \leq y \leq 1$）：

$$f_X(x) = \int_x^1 8xy\, dy = 8x \cdot \frac{1-x^2}{2} = 4x(1-x^2), \quad 0 \leq x \leq 1$$

$$\boxed{f_X(x) = \begin{cases} 4x(1-x^2), & 0 \leq x \leq 1 \\ 0, & \text{其他} \end{cases}}$$

求 $f_Y(y)$：固定 $y \in [0,1]$，$x$ 的范围为 $[0, y]$（由 $0 \leq x \leq y$）：

$$f_Y(y) = \int_0^y 8xy\, dx = 8y \cdot \frac{y^2}{2} = 4y^3, \quad 0 \leq y \leq 1$$

$$\boxed{f_Y(y) = \begin{cases} 4y^3, & 0 \leq y \leq 1 \\ 0, & \text{其他} \end{cases}}$$

### 【例题 3-6】计算题——均匀分布区域的边缘密度

**题目**（讲义例3.10）：设 $(X,Y)$ 在由 $x - y = 0$，$x + y = 2$，$y = 0$ 围成的三角形区域 $G$ 上服从均匀分布。求边缘概率密度 $f_X(x)$。

**解**：

三角形顶点为 $(0,0)$，$(2,0)$，$(1,1)$，面积 $S = \frac{1}{2} \times 2 \times 1 = 1$。

故 $f(x,y) = 1$，$(x,y) \in G$。

分两段确定 $y$ 的范围：
- 当 $0 \leq x \leq 1$：从 $y = 0$ 到 $y = x$（左侧受 $y \leq x$ 约束）；
- 当 $1 < x \leq 2$：从 $y = 0$ 到 $y = 2 - x$（右侧受 $y \leq 2 - x$ 约束）。

$$f_X(x) = \begin{cases} \int_0^x 1\, dy = x, & 0 \leq x \leq 1 \\ \int_0^{2-x} 1\, dy = 2-x, & 1 < x \leq 2 \\ 0, & \text{其他} \end{cases}$$

$$\boxed{f_X(x) = \begin{cases} x, & 0 \leq x \leq 1 \\ 2-x, & 1 < x \leq 2 \\ 0, & \text{其他} \end{cases}}$$

## 五、条件概率密度

### 引入

已知 $Y = y$ 这一信息，$X$ 的条件分布是什么？类比离散型"条件分布 = 联合 ÷ 边缘"，连续型也有同样的结构，只是把概率换成密度。

### 知识点

**条件概率密度**：

$$\boxed{f_{X|Y}(x \mid y) = \frac{f(x, y)}{f_Y(y)}, \quad f_Y(y) > 0}$$

$$\boxed{f_{Y|X}(y \mid x) = \frac{f(x, y)}{f_X(x)}, \quad f_X(x) > 0}$$

**条件概率**：

$$P\left\{X \leq x \mid Y = y\right\} = \int_{-\infty}^{x} f_{X|Y}(u \mid y)\, du$$

> ⚠️ 前提：$(X,Y)$ 是**二维连续型**随机变量！

### 【例题 3-7】计算题——条件密度与条件概率

**题目**（讲义例3.11）：已知 $(X,Y)$ 的密度函数为

$$f(x,y) = \begin{cases} 6, & 0 < x^2 < y < x < 1 \\ 0, & \text{其他} \end{cases}$$

求：(1) $f_{Y|X}(y \mid x)$；(2) $P\left\{Y \leq \frac{1}{3} \mid X = \frac{1}{2}\right\}$；(3) $P\left\{Y \leq \frac{1}{3} \mid X > \frac{1}{2}\right\}$。

**解**：

**(1)** 先求 $f_X(x)$：固定 $x \in (0,1)$，$y$ 的范围为 $(x^2, x)$：

$$f_X(x) = \int_{x^2}^{x} 6\, dy = 6(x - x^2), \quad 0 < x < 1$$

条件密度：

$$f_{Y|X}(y \mid x) = \frac{6}{6(x-x^2)} = \frac{1}{x(1-x)}, \quad x^2 < y < x$$

**(2)** $X = \frac{1}{2}$ 时，$y$ 的范围为 $(\frac{1}{4}, \frac{1}{2})$：

$$P\left\{Y \leq \frac{1}{3} \mid X = \frac{1}{2}\right\} = \int_{1/4}^{1/3} \frac{1}{(1/2)(1/2)}\, dy = 4 \times \left(\frac{1}{3} - \frac{1}{4}\right) = 4 \times \frac{1}{12} = \boxed{\frac{1}{3}}$$

**(3)** 注意 (3) 是条件概率（不同于条件密度），需用二重积分：

$$P\left\{Y \leq \frac{1}{3},\ X > \frac{1}{2}\right\} = \int_{1/2}^{1} \int_{x^2}^{\min(x, 1/3)} 6\, dy\, dx$$

当 $\frac{1}{2} < x < 1$ 时，$x^2 > \frac{1}{4}$；$\frac{1}{3} < x$ 即 $x > \frac{1}{3}$，所以当 $x > \frac{1}{2}$ 时，$\min(x, \frac{1}{3}) = \frac{1}{3}$（仅当 $x^2 < \frac{1}{3}$，即 $x < \frac{1}{\sqrt{3}} \approx 0.577$ 时有效）。

分两段：$\frac{1}{2} < x < \frac{1}{\sqrt{3}}$ 时积分限为 $(x^2, \frac{1}{3})$；$x \geq \frac{1}{\sqrt{3}}$ 时 $x^2 \geq \frac{1}{3}$，积分为0。

$$= \int_{1/2}^{1/\sqrt{3}} 6\left(\frac{1}{3} - x^2\right) dx = 6\left[\frac{x}{3} - \frac{x^3}{3}\right]_{1/2}^{1/\sqrt{3}} = \cdots = \frac{2\sqrt{3} - 3}{9}$$

$$P\{X > \frac{1}{2}\} = \int_{1/2}^1 6(x-x^2)\, dx = 6\left[\frac{x^2}{2} - \frac{x^3}{3}\right]_{1/2}^1 = 6\left(\frac{1}{6} - \frac{1}{8} + \frac{1}{24}\right) = \frac{1}{2}$$

$$\boxed{P\left\{Y \leq \frac{1}{3} \mid X > \frac{1}{2}\right\} = \frac{2\sqrt{3}-3}{9} \div \frac{1}{2} = \frac{2(2\sqrt{3}-3)}{9}}$$

## 六、二维连续型随机变量的独立性

### 引入

离散型用"每个格子 = 行和 × 列和"判断独立；连续型的对应条件是联合密度 = 两个边缘密度之积。另外还有一个重要的**图形判断技巧**。

### 知识点

**充要条件**：$X$ 与 $Y$ 相互独立，当且仅当

$$\boxed{f(x, y) = f_X(x) \cdot f_Y(y) \quad \text{在 } R^2 \text{ 上几乎处处成立}}$$

**快速判断方法**（两个条件同时满足 → 独立）：
1. $f(x,y)$ 的**非零区域是矩形**（即 $a < x < b$，$c < y < d$，上下限不互相依赖）；
2. $f(x,y)$ 能**分离变量**写成 $g(x) \cdot h(y)$ 的形式。

> 做题翻译：**区域形状不是矩形（如三角形）→ 直接判断不独立**，无需计算。

---

### 【例题 3-8】选择题——判断独立性

**题目**（讲义速通练一练）：设 $(X,Y)$ 的概率密度为 $f(x,y)$，判断以下情况 $X$ 与 $Y$ 是否独立：

1) $f(x,y) = \begin{cases} 2e^{-(x+y)}, & 0 < x < y \\ 0, & \text{其他} \end{cases}$

2) $f(x,y) = \begin{cases} 6e^{-(2x+3y)}, & x > 0, y > 0 \\ 0, & \text{其他} \end{cases}$

**解**：

**1)** 非零区域为 $\{0 < x < y\}$，是**三角形**（$x, y$ 上下限互相耦合）→ **不独立**。

**2)** 非零区域为 $\{x > 0, y > 0\}$，是**矩形**，且 $f(x,y) = (2e^{-2x}) \cdot (3e^{-3y})$ 可分离变量 → **独立**。

## 七、二维均匀分布

### 引入

把均匀分布推广到二维：在有界区域 $D$ 上"洒点"，每处密度相同，这就是**二维均匀分布**。

### 知识点

**定义**：设 $D$ 是平面有界区域，面积为 $S_D$，若

$$\boxed{f(x,y) = \begin{cases} \dfrac{1}{S_D}, & (x,y) \in D \\ 0, & \text{其他} \end{cases}}$$

则称 $(X,Y)$ 在 $D$ 上服从**二维均匀分布**。

**求概率（几何概型）**：若 $G \subset D$，

$$\boxed{P\{(X,Y) \in G\} = \frac{S_G}{S_D}}$$

**重要结论**：
- $(X,Y)$ 在矩形 $D = \{a \leq x \leq b,\ c \leq y \leq d\}$ 上服从均匀分布 $\Leftrightarrow$ $X \sim U(a,b)$，$Y \sim U(c,d)$，且 $X$ 与 $Y$ 独立；
- $(X,Y)$ 服从二维均匀分布，$X$ 和 $Y$ 的边缘分布**不一定**是一维均匀分布。

> ⚠️ **只有均匀分布才能用"面积比"求概率！** 若 $f(x,y)$ 不均匀，必须算二重积分。

---

### 【例题 3-9】选择题——二维均匀分布

**题目**（讲义例3.17）：设 $X$ 与 $Y$ 相互独立，且都在区间 $(0,1)$ 上随机取值，则

$$P\{X^2 + Y^2 \leq 1\} = \text{?}$$

(A) $\dfrac{1}{4}$ (B) $\dfrac{1}{2}$ (C) $\dfrac{\pi}{8}$ (D) $\dfrac{\pi}{4}$

**解**：

$X, Y$ 独立且 $X, Y \sim U(0,1)$，故 $(X,Y)$ 在正方形 $D = (0,1) \times (0,1)$ 上服从二维均匀分布（$S_D = 1$）。

区域 $G: x^2 + y^2 \leq 1$ 且 $(x,y) \in D$，即单位圆在第一象限部分，面积 $S_G = \dfrac{\pi}{4}$。

$$P\{X^2 + Y^2 \leq 1\} = \frac{S_G}{S_D} = \frac{\pi/4}{1} = \frac{\pi}{4}$$

答案选 **(D)**。

## 八、二维正态分布

### 引入

二维正态分布是最重要的二维连续分布，在工程、经济、自然科学中无处不在。它有5个参数，边缘分布仍是正态，且只要相关系数 $\rho = 0$ 就保证独立。

### 知识点

**定义**：若 $(X,Y)$ 的概率密度为

$$f(x,y) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}} \exp\left\{-\frac{1}{2(1-\rho^2)}\left[\frac{(x-\mu_1)^2}{\sigma_1^2} - 2\rho\frac{x-\mu_1}{\sigma_1}\cdot\frac{y-\mu_2}{\sigma_2} + \frac{(y-\mu_2)^2}{\sigma_2^2}\right]\right\}$$

则称 $(X,Y) \sim N(\mu_1, \mu_2; \sigma_1^2, \sigma_2^2; \rho)$，其中 $\sigma_1 > 0$，$\sigma_2 > 0$，$|\rho| < 1$。

**五条重要性质**（考试高频）：

$$\boxed{\text{性质1：边缘分布是正态}}：X \sim N(\mu_1, \sigma_1^2),\quad Y \sim N(\mu_2, \sigma_2^2)$$

$$\boxed{\text{性质2：独立性}} ：X \text{ 与 } Y \text{ 相互独立} \Longleftrightarrow \rho = 0$$

$$\boxed{\text{性质3：线性组合}}：aX + bY \sim N(a\mu_1 + b\mu_2,\ a^2\sigma_1^2 + b^2\sigma_2^2 + 2ab\rho\sigma_1\sigma_2)$$

$$\text{性质4}：(aX+bY,\ cX+dY) \text{ 也服从二维正态（当} ad-bc \neq 0\text{）}$$

$$\text{性质5：条件分布是正态分布}$$

> 口诀：**二维正态的边缘是正态，线性组合是正态，$\rho=0$ 等价于独立。**

---

### 【例题 3-10】选择题——正态分布独立性

**题目**（讲义例3.19）：设 $(\xi, \eta)$ 服从二维正态分布 $N(a_1, a_2; \sigma_1^2, \sigma_2^2; \rho)$，则 $\rho = 0$ 是 $\xi, \eta$ 独立的（ ）。

(A) 充分但不必要条件 (B) 必要但不充分条件 (C) 充分且必要条件 (D) 既不充分也不必要条件

**解**：

由二维正态分布的性质2，对于服从二维正态分布的随机变量，$\rho = 0 \Leftrightarrow X$ 与 $Y$ 独立（这是二维正态的特殊性质，一般情况不成立）。

答案选 **(C)**。

---

### 【例题 3-11】填空题——正态独立性与线性组合

**题目**（讲义例3.20）：设 $(X,Y) \sim N(0,0;1,1;0)$，则 $P\left\{\dfrac{X}{Y} > 0\right\} = $ \_\_\_\_。

**解**：

$\rho = 0$ 且 $(X,Y)$ 服从二维正态，故 $X$ 与 $Y$ 独立，$X \sim N(0,1)$，$Y \sim N(0,1)$。

$\dfrac{X}{Y} > 0$ 等价于 $X, Y$ 同号，即 $\{X > 0, Y > 0\}$ 或 $\{X < 0, Y < 0\}$：

$$P\left\{\frac{X}{Y} > 0\right\} = P\{X>0\}P\{Y>0\} + P\{X<0\}P\{Y<0\} = \frac{1}{2} \cdot \frac{1}{2} + \frac{1}{2} \cdot \frac{1}{2} = \boxed{\frac{1}{2}}$$

## 九、二维随机变量函数 $Z = g(X, Y)$ 的分布

### 引入

已知 $(X,Y)$ 的分布，如何求 $Z = g(X,Y)$（如 $Z = X + Y$、$Z = \max(X,Y)$）的分布？这是本章最复杂也是考题最多样的部分。根据 $X, Y$ 的类型和 $g$ 的形式，方法各有不同。

### 知识点

**（一）离散型：枚举法**

计算所有可能的 $g(x_i, y_j)$ 值，合并相同取值，概率相加。

**（二）连续型：分布函数法（万能）**

$$F_Z(z) = P\{g(X,Y) \leq z\} = \iint_{g(x,y) \leq z} f(x,y)\, dx\, dy$$

再对 $z$ 求导得 $f_Z(z) = F_Z'(z)$。

**（三）卷积公式（$Z = X + Y$ 的特殊情形）**

$$\boxed{f_Z(z) = \int_{-\infty}^{+\infty} f(z-y, y)\, dy = \int_{-\infty}^{+\infty} f(x, z-x)\, dx}$$

当 $X, Y$ 独立时：

$$\boxed{f_Z(z) = \int_{-\infty}^{+\infty} f_X(z-y) f_Y(y)\, dy = \int_{-\infty}^{+\infty} f_X(x) f_Y(z-x)\, dx}$$

**（四）$X, Y$ 均服从正态且独立，$Z = aX + bY$**：

$$\boxed{aX + bY \sim N\!\left(a\mu_1 + b\mu_2,\ a^2\sigma_1^2 + b^2\sigma_2^2\right)} \quad (ab \neq 0)$$

---

### 【例题 3-12】计算题——离散型函数分布

**题目**（讲义例3.22）：设 $(X,Y)$ 的联合分布律如下：

| $X \backslash Y$ | $-1$ | $0$ | $1$ |
|---|---|---|---|
| 0 | 1/16 | 1/8 | 5/16 |
| 1 | 2/8 | 2/8 | 0 |

(1) 求 $U = X + Y$ 的概率分布；(2) 求 $V = \min(X,Y)$ 的概率分布。

**解**：

**(1)** $U = X + Y$ 的所有可能取值：

| $(X, Y)$ | $U = X+Y$ | $P$ |
|---|---|---|
| $(0,-1)$ | $-1$ | $1/16$ |
| $(1,-1)$ | $0$ | $2/8 = 4/16$ |
| $(0,0)$ | $0$ | $1/8 = 2/16$ |
| $(1,0)$ | $1$ | $2/8 = 4/16$ |
| $(0,1)$ | $1$ | $5/16$ |
| $(1,1)$ | $2$ | $0$ |

合并：$U = 0$：$4/16 + 2/16 = 6/16 = 3/8$；$U = 1$：$4/16 + 5/16 = 9/16$。

| $U$ | $-1$ | $0$ | $1$ |
|---|---|---|---|
| $P$ | 1/16 | 6/16 | 9/16 |

**(2)** $V = \min(X,Y)$ 的所有可能值为 $-1, 0$：

$P\{V = -1\} = P\{X=-1\} = 0 + 1/16 + 2/8 = 0 + 1/16 + 4/16 = 5/16$（注意 $X$ 取值无 $-1$，故只看 $Y = -1$ 时 $X \leq -1$ 的情况，即 $X$ 无法取 $-1$，需重新看：$V = \min(X,Y) = -1$ 当且仅当 $X = -1$ 或 $Y = -1$... 

实际上 $X$ 只取 $0, 1$，$Y$ 取 $-1, 0, 1$，所以 $V = \min(X, Y)$ 的取值为 $-1, 0$（不可能为1，因为 $X$ 的最小值是0）。

$P\{V = -1\} = P\{Y = -1\} = 1/16 + 2/8 = 5/16$（$Y=-1$ 时 $\min = -1$）

$P\{V = 0\} = 1 - 5/16 = 11/16$

| $V$ | $-1$ | $0$ |
|---|---|---|
| $P$ | 5/16 | 11/16 |

### 【例题 3-13】计算题——卷积公式

**题目**（讲义例3.24）：设 $(X,Y)$ 的密度函数为

$$f(x,y) = \begin{cases} e^{-y}, & 0 < x < 1,\ y > 0 \\ 0, & \text{其他} \end{cases}$$

求 $Z = 2X + Y$ 的概率密度 $f_Z(z)$。

**解**：

利用一般卷积公式 $Z = aX + bY$（$a = 2, b = 1$）：

$$f_Z(z) = \frac{1}{|a|}\int_{-\infty}^{+\infty} f\!\left(\frac{z-y}{2}, y\right) dy$$

非零条件：$0 < \dfrac{z-y}{2} < 1$ 且 $y > 0$，即 $z - 2 < y < z$ 且 $y > 0$。

**当 $z \leq 0$**：$y > 0$ 且 $y < z \leq 0$ 矛盾，积分为0。

**当 $0 < z \leq 2$**：$y \in (0, z)$（因为 $z - 2 < 0$，取 $\max(0, z-2) = 0$）：

$$f_Z(z) = \frac{1}{2}\int_0^z e^{-y}\, dy = \frac{1}{2}(1 - e^{-z})$$

**当 $z > 2$**：$y \in (z-2, z)$：

$$f_Z(z) = \frac{1}{2}\int_{z-2}^z e^{-y}\, dy = \frac{1}{2}(e^{-(z-2)} - e^{-z}) = \frac{e^{-z}}{2}(e^2 - 1)$$

$$\boxed{f_Z(z) = \begin{cases} 0, & z \leq 0 \\ \frac{1}{2}(1-e^{-z}), & 0 < z \leq 2 \\ \frac{e^2-1}{2}e^{-z}, & z > 2 \end{cases}}$$

## 十、最大最小值分布

### 引入

设 $n$ 个独立部件组成系统，$X_1, X_2, \cdots, X_n$ 为各部件寿命。**串联系统**寿命 $= \min(X_1,\cdots,X_n)$（一个坏则全坏）；**并联系统**寿命 $= \max(X_1,\cdots,X_n)$（全坏才坏）。这类问题归结为最大最小值的分布。

### 知识点

设 $X_1, X_2, \cdots, X_n$ 相互独立，分布函数分别为 $F_{X_1}(x), \cdots, F_{X_n}(x)$。

**最大值**：$U = \max(X_1, \cdots, X_n)$

$$\boxed{F_U(z) = F_{X_1}(z) F_{X_2}(z) \cdots F_{X_n}(z)}$$

**最小值**：$V = \min(X_1, \cdots, X_n)$

$$\boxed{F_V(z) = 1 - [1 - F_{X_1}(z)][1 - F_{X_2}(z)] \cdots [1 - F_{X_n}(z)]}$$

**特别地**，当 $X_1, \cdots, X_n$ 独立同分布，$X_i \sim F(x)$ 时：

$$\boxed{F_{\max}(z) = [F(z)]^n, \quad F_{\min}(z) = 1 - [1 - F(z)]^n}$$

> 记忆口诀：**最大值 = 全部都不超过（乘积）；最小值 = 1 减去全部都超过（对偶）。**

---

### 【例题 3-14】选择题——最大值分布

**题目**（讲义例3.29）：设 $X, Y$ 独立同分布，$X$ 的分布函数为 $F(x)$，则 $Z = \max\{X, Y\}$ 的分布函数为（ ）

(A) $F^2(z)$ (B) $F(x)F(y)$ (C) $1-[1-F(x)]^2$ (D) $[1-F(x)][1-F(y)]$

**解**：

$Z = \max\{X,Y\}$，$X,Y$ 独立同分布 $F(x)$：

$$F_Z(z) = P\{Z \leq z\} = P\{X \leq z, Y \leq z\} = F(z) \cdot F(z) = F^2(z)$$

答案选 **(A)**。

---

### 【例题 3-15】计算题——最小值的概率密度

**题目**（讲义例3.31）：$X, Y$ 独立，均服从参数为1的指数分布。记 $V = \min\{X, Y\}$，求 $f_V(v)$。

**解**：

$X \sim E(1)$，$F(x) = 1 - e^{-x}$（$x > 0$）。

$$F_V(v) = 1 - [1 - F(v)]^2 = 1 - e^{-2v}, \quad v > 0$$

$$\boxed{f_V(v) = \begin{cases} 2e^{-2v}, & v > 0 \\ 0, & v \leq 0 \end{cases}}$$

即 $V \sim E(2)$，与补充结论"独立指数分布的最小值仍服从指数分布，参数相加"一致。

## 十一、补充：具有可加性的分布

| 分布 | 可加性结论 |
|------|------------|
| 二项分布 | $X_i \sim B(n_i, p)$ 相互独立 $\Rightarrow \sum X_i \sim B(\sum n_i, p)$ |
| 泊松分布 | $X_i \sim P(\lambda_i)$ 相互独立 $\Rightarrow \sum X_i \sim P(\sum \lambda_i)$ |
| 正态分布 | $X_i \sim N(\mu_i, \sigma_i^2)$ 相互独立 $\Rightarrow \sum a_i X_i \sim N(\sum a_i \mu_i, \sum a_i^2 \sigma_i^2)$ |
| 卡方分布 | $X_i \sim \chi^2(n_i)$ 相互独立 $\Rightarrow \sum X_i \sim \chi^2(\sum n_i)$ |
| 指数分布（最小值）| $X_i \sim E(\lambda_i)$ 相互独立 $\Rightarrow \min\{X_i\} \sim E(\sum \lambda_i)$ |
