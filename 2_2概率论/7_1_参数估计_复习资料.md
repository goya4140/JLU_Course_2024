# 第七章第1节 点估计与估计量的评价

## 知识框架

```
参数估计（点估计）
    │
    ├─ 点估计
    │       ├─ 矩估计法
    │       │       ├─ 离散型总体：令 X̄ = E(X)，解出参数
    │       │       └─ 连续型总体：令 X̄ = E(X)，解出参数
    │       │               └─ 特殊：一阶矩无效时，用二阶矩
    │       │
    │       └─ 最大似然估计法
    │               ├─ 似然函数 L(θ)：抽到当前样本的"概率"
    │               ├─ 有极值：解似然方程 d[lnL]/dθ = 0
    │               └─ L(θ) 单调：直接分析单调性求最值
    │
    └─ 估计量的评价标准
            ├─ 无偏性：E(θ̂) = θ
            ├─ 有效性：D(θ̂₁) < D(θ̂₂) 则 θ̂₁ 更有效
            └─ 一致性（相合性）：θ̂ →ᵖ θ
```

---

## 一、点估计的基本概念

### 引入

我们知道总体分布的**类型**（比如正态分布），但不知道**参数**（比如 $\mu$、$\sigma^2$）。点估计的目的就是：从样本出发，构造一个统计量来"猜"这个参数。

### 知识点

**定义**：设总体 $X$ 的分布函数 $F(x;\theta)$ 形式已知，$\theta$ 是待估参数，$X_1,X_2,\cdots,X_n$ 为总体 $X$ 的一个样本。

- **估计量**：统计量 $\hat{\theta} = \theta(X_1,X_2,\cdots,X_n)$（含样本的随机变量）
- **估计值**：$\hat{\theta} = \theta(x_1,x_2,\cdots,x_n)$（代入样本值后的具体数字）

> **做题翻译**：估计量是公式，估计值是数字。题目问"估计量"给含 $X_i$ 的表达式，问"估计值"则代入样本值算出数字。

---

## 二、矩估计法

### 引入

最直观的估计思路：**用样本的矩代替总体的矩**。第一阶原点矩就是均值，于是令 $\bar{X} = E(X)$，解出参数即可。本质是"用频率代概率"的推广。

### 知识点

**核心做法（单参数）**：令样本一阶原点矩等于总体一阶原点矩：

$$\boxed{\bar{X} = E(X)}$$

解方程得矩估计量 $\hat{\theta} = \theta(X_1,\cdots,X_n)$。

**双参数情形**：同时用一、二阶矩联立方程组：

$$\frac{1}{n}\sum_{i=1}^n X_i = E(X), \qquad \frac{1}{n}\sum_{i=1}^n X_i^2 = E(X^2)$$

**特殊情形（一阶矩无效）**：若 $E(X)$ 不含待估参数（恒为常数），则一阶矩方程退化，需改用**二阶矩**：

$$\frac{1}{n}\sum_{i=1}^n X_i^2 = E(X^2)$$

> **注意**：矩估计用样本二阶中心矩 $\dfrac{1}{n}\sum(X_i-\bar{X})^2$ 估计方差 $D(X)$，分母是 $n$ 而非 $n-1$。

---

### 【例题·填空】作业填空1（泊松分布，最简单的矩估计）

设总体 $X$ 服从参数为 $\lambda$ 的泊松分布，$\lambda>0$ 未知，$X_1,\cdots,X_n$ 为样本，则 $\lambda$ 的矩估计量为 $\hat{\lambda} =$ \_\_\_\_。

**解**：泊松分布 $E(X) = \lambda$，令 $\bar{X} = E(X)$：

$$\boxed{\hat{\lambda} = \bar{X}}$$

---

### 【例题·填空】作业填空2（均匀分布，连续型矩估计）

设总体 $X \sim U(\theta, 2)$，$\theta < 2$ 未知，$X_1,\cdots,X_n$ 为样本，则 $\theta$ 的矩估计量为 $\hat{\theta} =$ \_\_\_\_。

**解**：$E(X) = \dfrac{\theta+2}{2}$，令 $\bar{X} = E(X)$：

$$\frac{\theta+2}{2} = \bar{X} \implies \theta = 2\bar{X} - 2$$

$$\boxed{\hat{\theta} = 2\bar{X} - 2}$$

---

### 【例题·计算】作业计算1（离散型，矩估计+MLE，入门级）

设总体 $X$ 的概率分布为

| $X$ | $1$ | $2$ | $3$ |
|:---:|:---:|:---:|:---:|
| $P$ | $\theta^2$ | $2\theta(1-\theta)$ | $(1-\theta)^2$ |

其中 $0 < \theta < 1$ 未知，样本值为 $1, 2, 1$，求 $\theta$ 的矩估计值和最大似然估计值。

**解（矩估计）**：

$$E(X) = 1\cdot\theta^2 + 2\cdot2\theta(1-\theta) + 3\cdot(1-\theta)^2 = -2\theta + 3$$

样本均值 $\bar{x} = \dfrac{1+2+1}{3} = \dfrac{4}{3}$，令 $E(X) = \bar{x}$：

$$-2\theta + 3 = \frac{4}{3} \implies \boxed{\hat{\theta}_M = \frac{5}{6}}$$

**解（MLE）**：

$$L(\theta) = P\{X=1\}\cdot P\{X=2\}\cdot P\{X=1\} = \theta^2\cdot2\theta(1-\theta)\cdot\theta^2 = 2\theta^5(1-\theta)$$

$$\ln L = \ln 2 + 5\ln\theta + \ln(1-\theta)$$

$$\frac{d\ln L}{d\theta} = \frac{5}{\theta} - \frac{1}{1-\theta} = 0 \implies 5(1-\theta) = \theta \implies \boxed{\hat{\theta}_{MLE} = \frac{5}{6}}$$

---

### 【例题·选择】作业选择1（一阶矩无效，改用二阶矩）

设 $X_1,\cdots,X_n$ 为来自总体 $X$ 的样本，$X$ 的分布律为

| $X$ | $-1$ | $0$ | $1$ |
|:---:|:---:|:---:|:---:|
| $P$ | $\theta$ | $1-2\theta$ | $\theta$ |

其中 $\theta > 0$ 未知，则 $\theta$ 的矩估计量为（　）

(A) $\hat{\theta} = \bar{X}$　　(B) $\hat{\theta} = \bar{X}^2$　　(C) $\hat{\theta} = \dfrac{1}{2}\bar{X}^2$　　(D) $\hat{\theta} = \dfrac{1}{2n}\displaystyle\sum_{i=1}^n X_i^2$

**解**：

计算一阶矩：$E(X) = (-1)\cdot\theta + 0\cdot(1-2\theta) + 1\cdot\theta = 0$，与 $\theta$ 无关，无法解出 $\theta$。

改用二阶矩：$E(X^2) = 1\cdot\theta + 0 + 1\cdot\theta = 2\theta$

令 $E(X^2) = \dfrac{1}{n}\displaystyle\sum_{i=1}^n X_i^2$：

$$2\theta = \frac{1}{n}\sum X_i^2 \implies \hat{\theta} = \frac{1}{2n}\sum_{i=1}^n X_i^2$$

答案：$\boxed{(D)}$

---

### 【例题·计算】例7.3（正态总体，双参数矩估计）

设总体 $X \sim N(\mu,\sigma^2)$，$\mu,\sigma^2$ 均未知，$X_1,\cdots,X_n$ 为样本，求 $\mu,\sigma^2$ 的矩估计量。

**解**：

一阶矩：令 $\bar{X} = E(X) = \mu$，得 $\hat{\mu} = \bar{X}$。

二阶中心矩：令样本二阶中心矩等于总体方差：

$$\hat{\sigma}^2 = \frac{1}{n}\sum_{i=1}^n(X_i-\bar{X})^2$$

$$\boxed{\hat{\mu} = \bar{X}, \qquad \hat{\sigma}^2 = \frac{1}{n}\sum_{i=1}^n(X_i-\bar{X})^2}$$

> **注意**：矩估计 $\hat{\sigma}^2$ 分母为 $n$，而样本方差 $S^2$ 分母为 $n-1$，两者不同！

## 三、最大似然估计法

### 引入

矩估计"简单粗暴"，最大似然估计更有逻辑：既然已经抽到了这组样本，那最合理的参数，应该是**使得抽到这组数据概率最大**的那个参数值。

### 知识点

**似然函数**：

- 离散型：$L(\theta) = \displaystyle\prod_{i=1}^n p(x_i;\theta)$
- 连续型：$L(\theta) = \displaystyle\prod_{i=1}^n f(x_i;\theta)$

> **直觉口诀**：$L(\theta)$ = "恰好抽到样本值 $x_1,\cdots,x_n$" 的概率（密度之积）。

**求解步骤**：

1. 构造似然函数 $L(\theta)$
2. 取对数 $\ln L(\theta)$（乘积变求和，计算更方便）
3. 解似然方程 $\dfrac{d[\ln L(\theta)]}{d\theta} = 0$，若有解即为 MLE

**特殊情形（$L(\theta)$ 单调）**：若似然方程无解，分析 $L(\theta)$ 单调性，在约束范围内直接取极值（通常结果为 $\min\{X_i\}$ 或 $\max\{X_i\}$）。

**不变性原理**：

$$\boxed{\text{若 } \hat{\theta} \text{ 是 } \theta \text{ 的 MLE，则 } g(\hat{\theta}) \text{ 是 } g(\theta) \text{ 的 MLE}}$$

---

### 【例题·计算】作业计算3（从分布函数出发，求密度再做MLE）

设总体 $X$ 的分布函数为

$$F(x;\beta) = \begin{cases} 1-\left(\dfrac{1}{x}\right)^\beta, & x>1 \\ 0, & x \le 1 \end{cases}$$

$\beta > 1$ 未知，$X_1,\cdots,X_n$ 为样本。（1）求密度函数；（2）求 $\beta$ 的矩估计量；（3）求 $\beta$ 的 MLE。

**解（1）**：

$$f(x;\beta) = F'(x;\beta) = \begin{cases} \dfrac{\beta}{x^{\beta+1}}, & x>1 \\ 0, & x \le 1 \end{cases}$$

**解（2）矩估计**：

$$E(X) = \int_1^{+\infty} x\cdot\frac{\beta}{x^{\beta+1}}\,dx = \beta\int_1^{+\infty}x^{-\beta}\,dx = \frac{\beta}{\beta-1}$$

令 $E(X) = \bar{X}$：$\dfrac{\beta}{\beta-1} = \bar{X} \implies \beta(\bar{X}-1) = \bar{X}$

$$\boxed{\hat{\beta}_M = \frac{\bar{X}}{\bar{X}-1}}$$

**解（3）MLE**：

$$L(\beta) = \prod_{i=1}^n \frac{\beta}{x_i^{\beta+1}} = \frac{\beta^n}{\left(\prod x_i\right)^{\beta+1}}$$

$$\ln L = n\ln\beta - (\beta+1)\sum_{i=1}^n\ln x_i$$

$$\frac{d\ln L}{d\beta} = \frac{n}{\beta} - \sum_{i=1}^n\ln x_i = 0$$

$$\boxed{\hat{\beta}_{MLE} = \frac{n}{\displaystyle\sum_{i=1}^n\ln X_i}}$$

---

### 【例题·计算】例7.4（双参数 MLE，含归一化约束）

设总体 $X$ 的概率密度为

$$f(x;\alpha,\beta) = \begin{cases} \alpha, & -1 < x < 0 \\ \beta, & 0 < x < 1 \\ 0, & \text{其他} \end{cases}$$

样本值为 $-0.5,\,0.3,\,-0.2,\,-0.6,\,-0.1,\,0.4,\,0.5,\,-0.8$，求 $\alpha$ 的最大似然估计值。

**解**：由 $\displaystyle\int f\,dx = 1$ 得 $\alpha + \beta = 1$，即 $\beta = 1-\alpha$。

样本中落在 $(-1,0)$ 的有 5 个，落在 $(0,1)$ 的有 3 个：

$$L(\alpha) = \alpha^5(1-\alpha)^3$$

$$\frac{d\ln L}{d\alpha} = \frac{5}{\alpha} - \frac{3}{1-\alpha} = 0 \implies \boxed{\hat{\alpha} = \frac{5}{8}}$$

---

### 【例题·计算】例7.5（$L(\theta)$ 单调，MLE 取边界）

设某元件使用寿命 $X$ 的概率密度为

$$f(x;\theta) = \begin{cases} 2e^{-2(x-\theta)}, & x > \theta \\ 0, & x \le \theta \end{cases}$$

$\theta > 0$ 未知，$x_1,\cdots,x_n$ 为样本观测值，求 $\theta$ 的 MLE。

**解**：

要使每个 $f(x_i;\theta) > 0$，需要所有 $x_i > \theta$，即 $\theta \le \min\{x_i\}$。

$$L(\theta) = 2^n e^{-2\sum(x_i-\theta)} = 2^n e^{-2\sum x_i + 2n\theta}$$

$L(\theta)$ 关于 $\theta$ **单调递增**，在约束 $\theta \le \min\{x_i\}$ 下，取：

$$\boxed{\hat{\theta} = \min\{X_1,X_2,\cdots,X_n\}}$$

---

### 【例题·计算】例7.1（综合：连续型矩估计 + MLE + 一致性）

设总体 $X$ 的概率密度为

$$f(x;\theta) = \begin{cases} \dfrac{\theta^2}{x^3}e^{-\theta/x}, & x > 0 \\ 0, & \text{其他} \end{cases}$$

$\theta > 0$ 未知，$X_1,\cdots,X_n$ 为样本。（I）求 $\theta$ 的矩估计量 $\hat{\theta}_M$，并证明 $\hat{\theta}_M \xrightarrow{P} \theta$；（II）求 $\theta$ 的 MLE。

**解（I）**：令 $t = \theta/x$，计算 $E(X) = \theta$（积分过程略）。

令 $\bar{X} = E(X) = \theta$：

$$\boxed{\hat{\theta}_M = \bar{X}}$$

由辛钦大数定律，$\bar{X} \xrightarrow{P} E(X) = \theta$，故 $\hat{\theta}_M \xrightarrow{P} \theta$。✓

**解（II）**：

$$\ln L(\theta) = 2n\ln\theta - 3\sum\ln x_i - \theta\sum\frac{1}{x_i}$$

$$\frac{d\ln L}{d\theta} = \frac{2n}{\theta} - \sum\frac{1}{x_i} = 0 \implies \boxed{\hat{\theta}_{MLE} = \frac{2}{\frac{1}{n}\sum_{i=1}^n\frac{1}{X_i}}}$$

---

## 四、估计量的评价标准

### 引入

同一个参数可以有很多种估计量——哪个更好？我们用三个标准评判：准不准（无偏性）、方差小不小（有效性）、样本大了以后收不收敛（一致性）。

### 知识点

**1. 无偏性**

$$\boxed{E(\hat{\theta}) = \theta}$$

> **必背五个结论**（设 $E(X) = \mu$，$D(X) = \sigma^2$）：
> 1. $E(X_i) = \mu$，$D(X_i) = \sigma^2$
> 2. $E(\bar{X}) = \mu$，$D(\bar{X}) = \dfrac{\sigma^2}{n}$
> 3. $E(S^2) = \sigma^2$（$S^2$ 是 $\sigma^2$ 的无偏估计，分母 $n-1$ 的来源！）

**2. 有效性**

设 $\hat{\theta}_1$、$\hat{\theta}_2$ 都是 $\theta$ 的无偏估计量：

$$\boxed{D(\hat{\theta}_1) < D(\hat{\theta}_2) \implies \hat{\theta}_1 \text{ 比 } \hat{\theta}_2 \text{ 更有效}}$$

**3. 一致性（相合性）**

$$\hat{\theta} \xrightarrow{P} \theta \quad (n \to \infty)$$

> **做题翻译**：通常直接用大数定律得到，$\bar{X} \xrightarrow{P} \mu$ 是最常用的结论。

---

### 【例题·证明】作业证明1（$S^2$ 的无偏性——最重要的证明）

设总体 $X$ 的均值 $\mu$ 及方差 $\sigma^2$ 都存在，$X_1,\cdots,X_n$ 为样本，证明样本方差 $S^2 = \dfrac{1}{n-1}\displaystyle\sum_{i=1}^n(X_i-\bar{X})^2$ 是 $\sigma^2$ 的无偏估计。

**证明**：

先化简：

$$\sum_{i=1}^n(X_i-\bar{X})^2 = \sum_{i=1}^n X_i^2 - n\bar{X}^2$$

取期望：

$$E\!\left[\sum(X_i-\bar{X})^2\right] = \sum E(X_i^2) - nE(\bar{X}^2)$$

利用 $E(X_i^2) = D(X_i) + [E(X_i)]^2 = \sigma^2 + \mu^2$，以及 $E(\bar{X}^2) = D(\bar{X}) + [E(\bar{X})]^2 = \dfrac{\sigma^2}{n} + \mu^2$：

$$= n(\sigma^2+\mu^2) - n\!\left(\frac{\sigma^2}{n}+\mu^2\right) = n\sigma^2 - \sigma^2 = (n-1)\sigma^2$$

故：

$$E(S^2) = \frac{1}{n-1}\cdot(n-1)\sigma^2 = \sigma^2 \qquad \blacksquare$$

> **这就是为什么样本方差分母是 $n-1$ 而非 $n$**：用 $n-1$ 正好能补偿用 $\bar{X}$ 代替 $\mu$ 带来的系统性偏小，使结果无偏。

---

### 【例题·计算】例7.7（矩估计 + 无偏性判断）

设总体 $X$ 的概率密度为

$$f(x;\theta) = \begin{cases} \dfrac{6x}{\theta^3}(\theta-x), & 0 < x < \theta \\ 0, & \text{其他} \end{cases}$$

$X_1,\cdots,X_n$ 为样本，求 $\theta$ 的矩估计量 $\hat{\theta}$，并判断是否无偏。

**解**：

$$E(X) = \frac{6}{\theta^3}\int_0^\theta(\theta x^2-x^3)\,dx = \frac{6}{\theta^3}\cdot\frac{\theta^4}{12} = \frac{\theta}{2}$$

令 $\bar{X} = \dfrac{\theta}{2}$，得 $\hat{\theta} = 2\bar{X}$。

**判断无偏性**：$E(\hat{\theta}) = 2E(\bar{X}) = 2\mu = 2\cdot\dfrac{\theta}{2} = \theta$，故 $\hat{\theta} = 2\bar{X}$ **是无偏估计**。✓

---

### 【例题·证明】作业证明4（无偏性的综合应用）

设 $X_1,\cdots,X_n$ 为来自 $N(\mu,\sigma^2)$ 的样本，$\bar{X}$ 为样本均值，$S^2$ 为样本方差，统计量 $T = \bar{X}^2 - \dfrac{1}{n}S^2$，证明 $T$ 是 $\mu^2$ 的无偏估计量。

**证明**：

$$E(T) = E(\bar{X}^2) - \frac{1}{n}E(S^2)$$

由 $E(\bar{X}^2) = D(\bar{X}) + [E(\bar{X})]^2 = \dfrac{\sigma^2}{n} + \mu^2$，以及 $E(S^2) = \sigma^2$：

$$E(T) = \left(\frac{\sigma^2}{n}+\mu^2\right) - \frac{1}{n}\cdot\sigma^2 = \mu^2 \qquad \blacksquare$$

---

### 【例题·计算】例7.8（有效性比较）

取容量 $n=3$ 的样本 $X_1,X_2,X_3$，三个无偏估计量

$$\hat{\mu}_1 = \bar{X} = \frac{1}{3}\sum_{i=1}^3 X_i, \quad \hat{\mu}_2 = \frac{1}{2}X_1+\frac{1}{3}X_2+\frac{1}{6}X_3, \quad \hat{\mu}_3 = X_1$$

证明 $\hat{\mu}_1$ 比 $\hat{\mu}_2$、$\hat{\mu}_3$ 都有效。

**解**：设 $D(X) = \sigma^2$，比较方差：

$$D(\hat{\mu}_1) = \frac{\sigma^2}{3}$$

$$D(\hat{\mu}_2) = \left(\frac{1}{4}+\frac{1}{9}+\frac{1}{36}\right)\sigma^2 = \frac{14}{36}\sigma^2 = \frac{7}{18}\sigma^2$$

$$D(\hat{\mu}_3) = \sigma^2$$

由于 $\dfrac{1}{3} < \dfrac{7}{18} < 1$，故 $\hat{\mu}_1$ 最有效。✓

## 附：常见分布的矩估计量和最大似然估计量速查

| $X$ 服从分布 | 矩估计量 | 最大似然估计量 |
|---|---|---|
| 0-1 分布（参数 $p$）| $\hat{p} = \bar{X}$ | $\hat{p} = \bar{X}$ |
| $B(n,p)$ | $\hat{p} = \bar{X}/n$ | $\hat{p} = \bar{X}/n$ |
| $P(\lambda)$ | $\hat{\lambda} = \bar{X}$ | $\hat{\lambda} = \bar{X}$ |
| $U(a,b)$ | $\hat{a} = \bar{X}-\sqrt{\frac{3}{n}\sum(X_i-\bar{X})^2}$，$\hat{b} = \bar{X}+\sqrt{\frac{3}{n}\sum(X_i-\bar{X})^2}$ | $\hat{a} = \min\{X_i\}$，$\hat{b} = \max\{X_i\}$ |
| $E(\lambda)$（指数分布）| $\hat{\lambda} = 1/\bar{X}$ | $\hat{\lambda} = 1/\bar{X}$ |

> **规律口诀**：大多数常见分布两种方法结果一致；均匀分布是例外，矩估计和 MLE 结果不同。
