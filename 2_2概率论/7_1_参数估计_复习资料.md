# 第七章第1节 点估计与估计量的评价

## 知识框架

```
参数估计（点估计）
    │
    ├─ 点估计
    │       ├─ 矩估计法
    │       │       ├─ 离散型总体：令 X̄ = E(X)，解出参数
    │       │       └─ 连续型总体：令 X̄ = E(X)，解出参数
    │       │
    │       └─ 最大似然估计法
    │               ├─ 似然函数 L(θ)：抽到当前样本的"概率"
    │               ├─ 有极值：解似然方程 d[lnL]/dθ = 0
    │               └─ L(θ) 单调：直接分析单调性求最值
    │
    └─ 估计量的评价标准
            ├─ 无偏性：E(θ̂) = θ
            ├─ 有效性：D(θ̂₁) < D(θ̂₂) 则 θ̂₁ 更有效
            └─ 一致性（相合性）：θ̂ →ᵖ θ
```

---

## 一、点估计的基本概念

### 引入

我们知道总体分布的**类型**（比如正态分布），但不知道**参数**（比如 $\mu$、$\sigma^2$）。点估计的目的就是：从样本出发，构造一个统计量来"猜"这个参数。

### 知识点

**定义**：设总体 $X$ 的分布函数 $F(x;\theta)$ 形式已知，$\theta$ 是待估参数，$X_1,X_2,\cdots,X_n$ 为总体 $X$ 的一个样本。

- **估计量**：统计量 $\hat{\theta} = \theta(X_1,X_2,\cdots,X_n)$（含样本的随机变量）
- **估计值**：$\hat{\theta} = \theta(x_1,x_2,\cdots,x_n)$（代入样本值后的具体数）

> **做题翻译**：估计量是公式，估计值是数字。

---

## 二、矩估计法

### 引入

最直观的估计思路：**用样本的矩代替总体的矩**。第一阶原点矩就是均值，于是令 $\bar{X} = E(X)$，解出参数即可。本质是"用频率代概率"的推广。

### 知识点

**核心做法（单参数）**：令样本一阶原点矩等于总体一阶原点矩，即

$$\boxed{\bar{X} = E(X)}$$

解方程得矩估计量 $\hat{\theta} = \theta(X_1,\cdots,X_n)$。

**双参数情形**：同时用一、二阶矩联立方程组：

$$\frac{1}{n}\sum_{i=1}^n X_i = E(X), \qquad \frac{1}{n}\sum_{i=1}^n X_i^2 = E(X^2)$$

> **注意**：矩估计使用前提是相关总体矩存在。用样本二阶中心矩 $\dfrac{1}{n}\sum_{i=1}^n(X_i-\bar{X})^2$ 估计方差 $D(X)$ 也是常见用法。

---

### 【例题·计算】例7.1（连续型矩估计 + 最大似然 + 一致性）

设总体 $X$ 的概率密度为

$$f(x;\theta) = \begin{cases} \dfrac{\theta^2}{x^3}e^{-\theta/x}, & x > 0 \\ 0, & \text{其他} \end{cases}$$

其中 $\theta > 0$ 为未知参数，$X_1,X_2,\cdots,X_n$ 为来自总体的简单随机样本。

（I）求 $\theta$ 的矩估计量 $\hat{\theta}_M$，并证明 $\hat{\theta}_M \xrightarrow{P} \theta$；

（II）求 $\theta$ 的最大似然估计量。

**解（I）**：计算 $E(X)$：

$$E(X) = \int_0^{+\infty} x \cdot \frac{\theta^2}{x^3}e^{-\theta/x}\,dx = \int_0^{+\infty} \frac{\theta^2}{x^2}e^{-\theta/x}\,dx$$

令 $t = \theta/x$（$x = \theta/t$，$dx = -\theta/t^2\,dt$）：

$$E(X) = \int_0^{+\infty} \frac{\theta^2}{\theta^2/t^2}\,e^{-t}\cdot\frac{\theta}{t^2}\,dt = \theta\int_0^{+\infty}e^{-t}\,dt = \theta \cdot 1 = \theta$$

故令 $\bar{X} = E(X) = \theta$，得矩估计量：

$$\boxed{\hat{\theta}_M = \bar{X}}$$

**证明一致性**：由辛钦大数定律，$\bar{X} \xrightarrow{P} E(X) = \theta$，故 $\hat{\theta}_M \xrightarrow{P} \theta$。✓

**解（II）**：构造似然函数：

$$L(\theta) = \prod_{i=1}^n f(x_i;\theta) = \prod_{i=1}^n \frac{\theta^2}{x_i^3}e^{-\theta/x_i} = \frac{\theta^{2n}}{\prod x_i^3} \cdot e^{-\theta\sum(1/x_i)}$$

取对数：

$$\ln L(\theta) = 2n\ln\theta - 3\sum\ln x_i - \theta\sum\frac{1}{x_i}$$

解似然方程：

$$\frac{d\ln L}{d\theta} = \frac{2n}{\theta} - \sum\frac{1}{x_i} = 0 \implies \hat{\theta}_{MLE} = \frac{2n}{\sum_{i=1}^n \frac{1}{X_i}} = \boxed{\frac{2}{\frac{1}{n}\sum_{i=1}^n \frac{1}{X_i}}}$$

（即 $2$ 除以样本调和平均数的倒数，即 $2$ 乘以 $X_i$ 倒数的样本均值再取倒数。）

---

### 【例题·计算】例7.2（离散型矩估计 + 最大似然）

设总体 $X$ 的概率分布为

| $X$ | 0 | 1 | 2 | 3 |
|-----|---|---|---|---|
| $P$ | $\theta^2$ | $2\theta(1-\theta)$ | $\theta^2$ | $1-2\theta$ |

其中 $0 < \theta < \dfrac{1}{2}$，利用样本值 $3,1,3,0,3,1,2,3$，求 $\theta$ 的矩估计值和最大似然估计值。

**解（矩估计）**：

$$E(X) = 0\cdot\theta^2 + 1\cdot2\theta(1-\theta) + 2\cdot\theta^2 + 3\cdot(1-2\theta) = 2\theta(1-\theta) + 2\theta^2 + 3 - 6\theta = 3 - 4\theta$$

样本均值：$\bar{x} = \dfrac{3+1+3+0+3+1+2+3}{8} = \dfrac{16}{8} = 2$

令 $\bar{x} = E(X)$：$2 = 3 - 4\theta \implies \theta = \dfrac{1}{4}$

矩估计值：$\boxed{\hat{\theta}_M = \dfrac{1}{4}}$

**解（最大似然）**：

样本中 $0$ 出现 1 次，$1$ 出现 2 次，$2$ 出现 1 次，$3$ 出现 4 次。

$$L(\theta) = (\theta^2)^1 \cdot [2\theta(1-\theta)]^2 \cdot (\theta^2)^1 \cdot (1-2\theta)^4$$

$$= \theta^2 \cdot 4\theta^2(1-\theta)^2 \cdot \theta^2 \cdot (1-2\theta)^4 = 4\theta^6(1-\theta)^2(1-2\theta)^4$$

$$\ln L = \ln 4 + 6\ln\theta + 2\ln(1-\theta) + 4\ln(1-2\theta)$$

$$\frac{d\ln L}{d\theta} = \frac{6}{\theta} - \frac{2}{1-\theta} - \frac{8}{1-2\theta} = 0$$

通分化简（令 $t = \theta$）：

$$6(1-\theta)(1-2\theta) - 2\theta(1-2\theta) - 8\theta(1-\theta) = 0$$

展开：$6(1-3\theta+2\theta^2) - 2\theta(1-2\theta) - 8\theta(1-\theta) = 0$

$6 - 18\theta + 12\theta^2 - 2\theta + 4\theta^2 - 8\theta + 8\theta^2 = 0$

$24\theta^2 - 28\theta + 6 = 0 \implies 12\theta^2 - 14\theta + 3 = 0$

$$\theta = \frac{14 \pm \sqrt{196 - 144}}{24} = \frac{14 \pm \sqrt{52}}{24}$$

取 $0 < \theta < \dfrac{1}{2}$ 范围内的解：$\hat{\theta}_{MLE} \approx \dfrac{14 - 7.21}{24} \approx \boxed{0.283}$

### 【例题·计算】例7.3（正态总体矩估计）

设总体 $X \sim N(\mu,\sigma^2)$，其中 $\mu,\sigma^2$ 是未知参数，$X_1,\cdots,X_n$ 是样本，求 $\mu,\sigma^2$ 的矩估计量。

**解**：

一阶矩：$E(X) = \mu$，令 $\bar{X} = \mu$，故 $\hat{\mu} = \bar{X}$。

二阶中心矩：$E(X-\mu)^2 = D(X) = \sigma^2$，用样本二阶中心矩估计：

$$\hat{\sigma}^2 = \frac{1}{n}\sum_{i=1}^n(X_i - \bar{X})^2$$

$$\boxed{\hat{\mu} = \bar{X}, \qquad \hat{\sigma}^2 = \frac{1}{n}\sum_{i=1}^n(X_i-\bar{X})^2}$$

> **注意**：矩估计 $\hat{\sigma}^2$ 用 $\dfrac{1}{n}$，而样本方差 $S^2$ 用 $\dfrac{1}{n-1}$，两者不同！

## 三、最大似然估计法

### 引入

矩估计"简单粗暴"，最大似然估计更有逻辑：既然我已经抽到了这组样本值，那最合理的参数估计，应该是**使得抽到这组数据概率最大**的那个参数值。

### 知识点

**似然函数**：

- 离散型：$L(\theta) = \displaystyle\prod_{i=1}^n p(x_i;\theta)$
- 连续型：$L(\theta) = \displaystyle\prod_{i=1}^n f(x_i;\theta)$

> **直觉口诀**：$L(\theta)$ = "恰好抽到样本值 $x_1,\cdots,x_n$" 的概率（密度之积）。

**求解步骤**（以单参数为例）：

1. 构造似然函数 $L(\theta) = \displaystyle\prod_{i=1}^n f(x_i;\theta)$
2. 取对数 $\ln L(\theta) = \displaystyle\sum_{i=1}^n \ln f(x_i;\theta)$（乘积变求和，计算更方便）
3. 解似然方程 $\dfrac{d[\ln L(\theta)]}{d\theta} = 0$，若有解即为 MLE

> **特殊情形（$L(\theta)$ 单调）**：若似然方程无解，分析 $L(\theta)$ 单调性，在约束范围内直接取极值点（如 $\hat{\theta} = \min\{X_i\}$）。

**不变性原理**：设 $\hat{\theta}$ 是 $\theta$ 的 MLE，若 $g(\theta)$ 有单值反函数，则 $g(\hat{\theta})$ 是 $g(\theta)$ 的 MLE。

$$\boxed{\text{不变性：} \hat{g(\theta)} = g(\hat{\theta})}$$

> 常见应用：已知 $\hat{\mu}$ 和 $\hat{\sigma}^2$，则 $\sigma$ 的 MLE 为 $\hat{\sigma} = \sqrt{\hat{\sigma}^2}$。

---

### 【例题·计算】例7.4（连续型 MLE，参数在约束边界）

设总体 $X$ 的概率密度为

$$f(x;\alpha,\beta) = \begin{cases} \alpha, & -1 < x < 0 \\ \beta, & 0 < x < 1 \\ 0, & \text{其他} \end{cases}$$

其中 $\alpha,\beta$ 为未知参数，总体 $X$ 的一组样本值为 $-0.5,\,0.3,\,-0.2,\,-0.6,\,-0.1,\,0.4,\,0.5,\,-0.8$，求 $\alpha$ 的最大似然估计值。

**解**：

由 $\displaystyle\int_{-\infty}^{+\infty}f(x)dx = 1$，得 $\alpha + \beta = 1$，即 $\beta = 1 - \alpha$。

样本中：$(-1,0)$ 内有 5 个观测值（$-0.5,-0.2,-0.6,-0.1,-0.8$），$(0,1)$ 内有 3 个（$0.3,0.4,0.5$）。

$$L(\alpha) = \alpha^5 \cdot \beta^3 = \alpha^5(1-\alpha)^3$$

$$\ln L = 5\ln\alpha + 3\ln(1-\alpha)$$

$$\frac{d\ln L}{d\alpha} = \frac{5}{\alpha} - \frac{3}{1-\alpha} = 0 \implies \hat{\alpha} = \boxed{\frac{5}{8}}$$

---

### 【例题·计算】例7.5（$L(\theta)$ 单调，MLE 取边界）

设某元件使用寿命 $X$ 的概率密度为

$$f(x;\theta) = \begin{cases} 2e^{-2(x-\theta)}, & x > \theta \\ 0, & x \le \theta \end{cases}$$

其中 $\theta > 0$ 为未知参数，$x_1,\cdots,x_n$ 是 $X$ 的一组样本观测值，求 $\theta$ 的最大似然估计量。

**解**：

要使 $L(\theta) = \displaystyle\prod_{i=1}^n 2e^{-2(x_i-\theta)} = 2^n e^{-2\sum(x_i-\theta)}$ 有意义（每个 $f(x_i;\theta) > 0$），需要所有 $x_i > \theta$，即 $\theta < \min\{x_i\}$。

$$L(\theta) = 2^n e^{-2\sum x_i + 2n\theta}$$

$L(\theta)$ 关于 $\theta$ **单调递增**（$\theta$ 越大 $L$ 越大），在约束 $\theta \le \min\{x_i\}$ 下，取

$$\hat{\theta} = \boxed{\min\{X_1,X_2,\cdots,X_n\}}$$

---

### 【例题·计算】例7.6（正态总体 MLE）

设 $x_1,\cdots,x_n$ 是来自 $X \sim N(\mu,\sigma^2)$（$\mu,\sigma^2$ 均未知）的样本观测值，求 $\mu$ 和 $\sigma^2$ 的最大似然估计值。

**解**：

$$L(\mu,\sigma^2) = \prod_{i=1}^n \frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x_i-\mu)^2}{2\sigma^2}} = (2\pi\sigma^2)^{-n/2}\exp\!\left(-\frac{\sum(x_i-\mu)^2}{2\sigma^2}\right)$$

$$\ln L = -\frac{n}{2}\ln(2\pi) - \frac{n}{2}\ln\sigma^2 - \frac{\sum(x_i-\mu)^2}{2\sigma^2}$$

对 $\mu$ 和 $\sigma^2$ 分别求偏导并令为 0：

$$\frac{\partial \ln L}{\partial \mu} = \frac{\sum(x_i-\mu)}{\sigma^2} = 0 \implies \hat{\mu} = \bar{x}$$

$$\frac{\partial \ln L}{\partial(\sigma^2)} = -\frac{n}{2\sigma^2} + \frac{\sum(x_i-\mu)^2}{2\sigma^4} = 0 \implies \hat{\sigma}^2 = \frac{1}{n}\sum(x_i-\bar{x})^2$$

$$\boxed{\hat{\mu} = \bar{X}, \qquad \hat{\sigma}^2 = \frac{1}{n}\sum_{i=1}^n(X_i-\bar{X})^2}$$

> **结论**：正态总体中，$\mu$ 和 $\sigma^2$ 的矩估计量与最大似然估计量**完全一致**！

## 四、估计量的评价标准

### 引入

同一个参数可以有很多种估计量——哪个更好？我们用三个标准评判：准不准（无偏性）、方差小不小（有效性）、样本大了以后收不收敛（一致性）。

### 知识点

**1. 无偏性**

若 $E(\hat{\theta}) = \theta$ 对所有 $\theta \in \Theta$ 成立，则称 $\hat{\theta}$ 是 $\theta$ 的**无偏估计量**。

$$\boxed{E(\hat{\theta}) = \theta}$$

> **必背结论**（设总体 $E(X) = \mu$，$D(X) = \sigma^2$）：
> 1. $E(X_i) = \mu$，$D(X_i) = \sigma^2$
> 2. $E(\bar{X}) = \mu$，$D(\bar{X}) = \dfrac{\sigma^2}{n}$
> 3. $E(S^2) = \sigma^2$（样本方差 $S^2$ 是 $\sigma^2$ 的无偏估计）
>
> **注意**：$\dfrac{1}{n}\sum(X_i-\bar{X})^2$（矩估计量）不是 $\sigma^2$ 的无偏估计！

**2. 有效性**

设 $\hat{\theta}_1$、$\hat{\theta}_2$ 都是 $\theta$ 的无偏估计量，若

$$D(\hat{\theta}_1) < D(\hat{\theta}_2)$$

则称 $\hat{\theta}_1$ 比 $\hat{\theta}_2$ **更有效**。方差越小，估计越集中，越好。

$$\boxed{\text{方差小者更有效}}$$

**3. 一致性（相合性）**

若对任意 $\varepsilon > 0$ 有

$$\lim_{n\to\infty} P\{|\hat{\theta} - \theta| < \varepsilon\} = 1$$

即 $\hat{\theta} \xrightarrow{P} \theta$，则称 $\hat{\theta}$ 是 $\theta$ 的**一致估计量**。

> **做题翻译**：一致性就是"样本量 $n \to \infty$ 时，估计量依概率收敛到真值"。通常用大数定律直接得到。

---

### 【例题·计算】例7.7（矩估计量 + 无偏性判断）

设总体 $X$ 的概率密度为

$$f(x;\theta) = \begin{cases} \dfrac{6x}{\theta^3}(\theta - x), & 0 < x < \theta \\ 0, & \text{其他} \end{cases}$$

$X_1,\cdots,X_n$ 是来自 $X$ 的简单随机样本，求未知参数 $\theta$ 的矩估计量 $\hat{\theta}$，并判断 $\hat{\theta}$ 是否为 $\theta$ 的无偏估计。

**解**：

计算 $E(X)$：

$$E(X) = \int_0^\theta x \cdot \frac{6x}{\theta^3}(\theta-x)\,dx = \frac{6}{\theta^3}\int_0^\theta(\theta x^2 - x^3)\,dx$$

$$= \frac{6}{\theta^3}\left[\frac{\theta x^3}{3} - \frac{x^4}{4}\right]_0^\theta = \frac{6}{\theta^3}\left(\frac{\theta^4}{3} - \frac{\theta^4}{4}\right) = \frac{6}{\theta^3}\cdot\frac{\theta^4}{12} = \frac{\theta}{2}$$

令 $\bar{X} = E(X) = \dfrac{\theta}{2}$，解得矩估计量：

$$\boxed{\hat{\theta} = 2\bar{X}}$$

**判断无偏性**：

$$E(\hat{\theta}) = E(2\bar{X}) = 2E(\bar{X}) = 2\mu = 2\cdot\frac{\theta}{2} = \theta$$

$E(\hat{\theta}) = \theta$，故 $\hat{\theta} = 2\bar{X}$ **是** $\theta$ 的无偏估计量。✓

---

### 【例题·计算】例7.8（有效性比较）

取容量 $n=3$ 的样本 $X_1,X_2,X_3$，证明总体均值 $\mu$ 的三个无偏估计量

$$\hat{\mu}_1 = \bar{X} = \frac{1}{3}\sum_{i=1}^3 X_i, \quad \hat{\mu}_2 = \frac{1}{2}X_1 + \frac{1}{3}X_2 + \frac{1}{6}X_3, \quad \hat{\mu}_3 = X_1$$

中，$\hat{\mu}_1 = \bar{X}$ 比 $\hat{\mu}_2$、$\hat{\mu}_3$ 都有效。

**解**：设 $D(X) = \sigma^2$，三者均为无偏估计（可验证），比较方差：

$$D(\hat{\mu}_1) = D(\bar{X}) = \frac{\sigma^2}{3}$$

$$D(\hat{\mu}_2) = \left(\frac{1}{2}\right)^2\sigma^2 + \left(\frac{1}{3}\right)^2\sigma^2 + \left(\frac{1}{6}\right)^2\sigma^2 = \left(\frac{1}{4}+\frac{1}{9}+\frac{1}{36}\right)\sigma^2 = \frac{14}{36}\sigma^2 = \frac{7}{18}\sigma^2$$

$$D(\hat{\mu}_3) = D(X_1) = \sigma^2$$

由于 $\dfrac{1}{3} < \dfrac{7}{18} < 1$，故 $D(\hat{\mu}_1) < D(\hat{\mu}_2) < D(\hat{\mu}_3)$，

即 $\hat{\mu}_1 = \bar{X}$ 比 $\hat{\mu}_2$、$\hat{\mu}_3$ 都更有效。✓

## 附：常见分布的矩估计量和最大似然估计量速查

| $X$ 服从分布 | 矩估计量 | 最大似然估计量 |
|---|---|---|
| 0-1 分布（参数 $p$）| $\hat{p} = \bar{X}$ | $\hat{p} = \bar{X}$ |
| $B(n,p)$ | $\hat{p} = \bar{X}/n$ | $\hat{p} = \bar{X}/n$ |
| $P(\lambda)$ | $\hat{\lambda} = \bar{X}$ | $\hat{\lambda} = \bar{X}$ |
| $U(a,b)$ | $\hat{a} = \bar{X} - \sqrt{\frac{3}{n}\sum(X_i-\bar{X})^2}$，$\hat{b} = \bar{X} + \sqrt{\frac{3}{n}\sum(X_i-\bar{X})^2}$ | $\hat{a} = \min\{X_i\}$，$\hat{b} = \max\{X_i\}$ |
| $E(\lambda)$（指数分布）| $\hat{\lambda} = 1/\bar{X}$ | $\hat{\lambda} = 1/\bar{X}$ |

> **规律口诀**：大多数常见分布的矩估计和最大似然估计结果一致；均匀分布是个例外，两种方法结果不同。
