## 二、正文：知识点重排

### 模块 A｜概念三层次：微处理器 → 微型计算机 → 微型计算机系统

理解本章的起点是区分这三个层层包含的概念，它们在选择题和填空题中频繁以"请选出正确描述"的形式出现。

---

> **【练习题 Q1】** 微处理器、微型计算机、微型计算机系统的区别是什么？

**知识铺垫**

三者是范围逐级扩大的"包含关系"，可以用同心圆来理解：

```
┌────────────────────────────────────────┐
│          微型计算机系统                │
│  ┌──────────────────────────────────┐  │
│  │        微型计算机（裸机）        │  │
│  │  ┌────────────────────────────┐  │  │
│  │  │     微处理器（CPU核心）    │  │  │
│  │  └────────────────────────────┘  │  │
│  └──────────────────────────────────┘  │
└────────────────────────────────────────┘
```

**✅ 参考答案要点**

- **微处理器**：由一片或少数几片大规模集成电路组成的 CPU，是微机系统的**核心部件**，本身不能独立工作。
- **微型计算机**：由微处理器 + 主存储器 + I/O 接口电路 + 系统总线构成的**裸机整体**，有完整的运算和控制能力，但没有外设和软件。
- **微型计算机系统**：以微型计算机为中心，配以 I/O 外围设备、系统软件（操作系统、编译器等），以及电源、面板和机架等，组成的**完整可用系统**。

---

> **【练习题 Q2】** 微型计算机有哪些基本部分构成？

**✅ 参考答案要点**

四个组成部分，缺一不可：

1. **微处理器（CPU）**
2. **主存储器**
3. **I/O 接口电路**
4. **系统总线**（数据总线 DB + 地址总线 AB + 控制总线 CB）

> 📝 **画图提示**：课件图1.1.1是高频考点。画图时，4个模块之间用三组总线（DB/AB/CB）相连，注意 CPU 与存储器/接口之间数据线是双向箭头，地址线是从CPU出去的单向箭头。

### 模块 B｜CISC vs RISC——本章唯一官方划定的简答考点

有了微型计算机的概念基础，进一步了解 CPU 指令体系的两大流派。CISC/RISC 对比是本章最有可能独立出题的知识点（简答题或选择题），务必能从多个维度展开。

---

> **【练习题 Q3 ⚠️考点 🔥高频】** CISC、RISC 的技术特点。

**知识铺垫**

CISC（复杂指令系统计算机）的设计思路是"用硬件实现复杂功能"，代表就是我们课程里研究的 Intel x86 体系；RISC（精简指令系统计算机）的思路是"简化指令，靠软件组合完成复杂操作"，代表是 ARM、MIPS。

两者的核心差异可以从三个维度记忆：**指令系统 / 内存访问 / 电路结构**。

| 维度 | RISC | CISC |
|---|---|---|
| 指令系统 | 简单，指令短且**等长** | 复杂，指令长且**不等长** |
| 内存访问 | **专有指令**访问内存（Load/Store），其余指令只操作寄存器 | 很多指令可直接访问内存，寻址方式多且复杂 |
| 电路结构 | 简单，设计周期短，面积小，功耗低；易于采用并行/流水线技术 | 复杂，设计周期长，技术更新难，但功能强 |
| 中断响应时机 | 可在指令执行**适当地方**响应中断（流水线友好） | 在**一条指令执行结束后**响应中断 |

**✅ 参考答案要点**

CISC 特点：
- 指令系统丰富且复杂，指令长度**不等长**
- 访问存储器的指令较多，逻辑控制复杂
- 电路设计周期长，功耗和面积较大
- 在**一条指令执行结束后**响应中断

RISC 特点：
- 指令系统简单，指令短且**等长**
- 仅包含必要指令，专有指令（Load/Store）访问内存，电路结构简单
- 设计周期短，功耗低，**易于采用并行计算技术**
- 可以在**指令执行的适当地方**响应中断

典型代表（选答）：
- CISC：Intel x86、AMD、Cyrix、VIA
- RISC：IBM PowerPC、Sun SPARC、ARM、MIPS R系列

### 模块 C｜课件补充·无对应原题：微处理器发展史脉络

> 📝 课件补充·无对应原题（此内容课件§1.3 有详细讲，但练习题中无对应题目）

发展史通常以选择题形式考查"某款芯片的位数/特性"，记住几个关键节点即可：

| 代际 | 典型芯片 | 关键特点 |
|---|---|---|
| 4位（1970s初） | Intel 4004 | 第一款商用微处理器 |
| 8位（1974-1977） | Intel 8080/8085、Z80 | 有中断和DMA，可用汇编/BASIC |
| **16位（1978-）** | **Intel 8086**（DB16/AB20）| 本课程核心研究对象 |
| 16位准16位 | Intel 8088 | IBM PC原型机，成就了微软 |
| 高档16位 | Intel 80286 | AB24位，引入保护模式 |
| 32位（1980s-） | 80386（首次多任务）、80486、**Pentium** | Pentium=AB32位/DB64位，首次超标量结构 |
| 64位（2001-） | Itanium、Core i系列 | 现代多核处理器 |

**Intel 8086 的关键参数**（后续章节大量使用）：数据总线 DB = 16 位，地址总线 AB = 20 位（因此寻址空间为 $2^{20}$ = 1MB）。
