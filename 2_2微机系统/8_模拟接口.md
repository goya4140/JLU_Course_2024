## 1. 知识体系

本章解决的核心问题是：**计算机（数字系统）如何与现实世界（模拟信号）进行双向交互**。现实世界的信号（温度、压力、电压）是连续的模拟量，而 CPU 只能处理离散的数字量，因此必须在两者之间建立桥梁。

本章分为两个方向：**D/A 转换**（数字 → 模拟，CPU 控制外部设备）和 **A/D 转换**（模拟 → 数字，外部信号输入 CPU）。首先介绍 D/A 转换的原理（T型电阻网络）和典型芯片 DAC0832，再介绍 A/D 转换的三种方法（计数式/逐次逼近/双积分）和典型芯片 ADC0809，最后讨论采样保持器的必要性。

## 2. 核心知识点精讲

### 2.1 D/A 转换原理——T型电阻网络

**核心定义**：T型（R-2R）电阻网络 D/A 转换，利用分流原理，将二进制各位代码控制的支路电流叠加后，经运算放大器转换为输出电压。

**原理解析**：

```
V_REF ──R──┬──R──┬──R──...──R──┐
           2R   2R              2R   → I_OUT → 运放 → V_OUT
           │    │
          K7   K6  (开关由D7、D6...控制，接地=0，接I_OUT=1)
```

- 电阻只有 R 和 2R 两种，阻值 100Ω～1000Ω
- 每节电路 = 两个电阻 + 一个开关，对应二进制的一位
- 输出电流：$I_{OUT} = \frac{V_{REF}}{2^n R} \times N$（N为输入数字量）

**核心公式**（必背）：

$$V_{OUT} = -\frac{V_{REF}}{2^n} \times N$$

其中：
- $V_{REF}$：参考电压（可为正或负）
- $n$：DAC 位数（DAC0832 为 8 位，即 $2^n = 256$）
- $N$：输入数字量（0～255）
- 负号来自运算放大器的**反相**特性

**考试写法**：

一般形式：$\dfrac{V_{REF}}{2^n} = \dfrac{V_{OUT}}{N}$，即 $V_{OUT} = \dfrac{V_{REF} \times N}{2^n}$

> ⚠️ 注意：课件公式含负号（$V_{OUT} = -E_0 \times N$），但很多实际接口题用非反相放大电路，结果为正值。**题目若问"输出电压"，按题目给出的电压范围判断正负**。

⚠️ 考点：计算题 | ⭐⭐⭐  
❌ 易错：忘记 $2^n$ 是分母（8位DAC为256，12位DAC为4096），不要写成 n。

---

### 2.2 D/A 转换器主要技术指标

**分辨率**：输入数字量 LSB 变化1时，输出模拟量的最小变化量。

$$\text{分辨率} = \frac{FS}{2^n}$$

其中 FS 为满量程值。**分辨率通常直接用位数 n 表示**（如"8位分辨率"）。

**建立时间**（转换时间）：输入数字量变化后，输出模拟量稳定到误差范围内（稳定值±ε）所需的时间。DAC0832 建立时间为 **1μs**。

⚠️ 考点：简答/选择 | ⭐⭐

### 2.3 DAC0832 芯片（重点）

**主要特性**（考试常考选择/简答）：

| 特性 | 参数 |
|------|------|
| 分辨率 | **8位** |
| 输出类型 | **电流型**（需外接运放转换为电压） |
| 电源 | 单电源 +5V～+15V |
| 建立时间 | **1μs** |
| 网络结构 | R-2R T型解码网络 |
| 参考电压 | -10V～+10V |
| 工作方式 | **双缓冲、单缓冲、直接输入（直通）三种** |

**三种工作方式**（简答必背）：

```
双缓冲方式：
  数据 → [8位输入寄存器] → [8位DAC寄存器] → D/A转换
  两级锁存，适合多片DAC同步输出的场合
  
单缓冲方式（典型连接）：
  数据 → [寄存器（旁路其中一级）] → D/A转换
  适合只有单路D/A输出，或不要求同步的场合
  
直通方式：
  数据直接进入D/A转换，不经寄存器锁存
  适合外部已有锁存器的场合
```

**典型单缓冲接口（必背端口地址判断）**：

DAC0832 通常通过 74LS138 译码，A7～A2 接译码器，A1、A0 悬空或接低位。例：A7=1、A6=0、A5=0、A4=0、A3=0、A2=0 → 端口地址 = `80H～83H`，输出指令：`OUT 80H, AL`。

⚠️ 考点：综合题地址计算 | ⭐⭐⭐

### 2.4 DAC0832 典型波形程序

**方波发生器**（🔥 高频，可能出编程题）：

```asm
A1: MOV  AL, 00H       ; 输出低电平（0V）
    OUT  78H, AL       ; 送 DAC0832
    CALL DELAY         ; 延时（控制低电平持续时间→调频）
    MOV  AL, 0FFH      ; 输出高电平（满量程电压）
    OUT  78H, AL
    CALL DELAY
    JMP  A1            ; 循环，产生连续方波
; 改变 00H/0FFH 的值可调幅；改变延时可调频
```

**锯齿波发生器**：

```asm
    MOV  AL, 00H
    OUT  78H, AL
A1: INC  AL            ; AL 递增（0→255→溢出回0，实现锯齿）
    OUT  78H, AL
    MOV  CX, n         ; 延时循环
A2: LOOP A2
    JMP  A1
```

**三角波发生器**：

```asm
    MOV  AL, 00H
A1: OUT  78H, AL       ; 上升段
    CALL DELAY
    INC  AL
    JNZ  A1            ; AL 从 0 递增到 FFH 后溢出为 0，ZF=1 退出
    DEC  AL            ; 使 AL=FFH（从FFH-1开始下降，消除平顶）
    DEC  AL            ; 使 AL=FEH
A2: OUT  78H, AL       ; 下降段
    CALL DELAY
    DEC  AL
    JNZ  A2
    JMP  A1
```

⚠️ 考点：编程题 | ⭐⭐  
❌ 易错：三角波中两个 DEC AL 的目的是"消除平顶"，不要漏写。

### 2.5 A/D 转换原理与方法

**三种方法比较**（简答常考）：

| 方式 | 速度 | 精度 | 成本 | 适用场合 |
|------|------|------|------|----------|
| 计数式 | 最慢 | 低 | 低 | 简单低速 |
| **逐次逼近式** | **较高** | 适中 | 适中 | **最常用**，微机外围接口主流 |
| 双积分式 | 慢 | **高** | 较高 | 高精度仪器仪表 |

**逐次逼近原理**：将被测电压与 D/A 转换生成的电压比较，用**对分搜索**（二分法）逐次逼近被测电压，8位ADC需8次比较。

**A/D 转换的基本过程**（简答必背）：

```
模拟信号 → [采样] → [保持] → [量化] → [编码] → 数字量
           └──────采样──────┘ └──────量化──────┘
```

- **采样**：将连续模拟量变为离散量（包含采样+保持两步）
- **量化**：将离散量变为数字量（包含量化+编码两步）

⚠️ 考点：简答 | ⭐⭐

### 2.6 采样定理（必背）

**规范表述**：

对一个有限频谱（最高频率为 $f_{max}$）的连续信号进行采样，当**采样频率 $f_s \geq 2f_{max}$** 时，采样输出信号能无失真地恢复到原来的连续信号。

$$f_s \geq 2 f_{max}$$

- $f_s$：采样频率
- $f_{max}$：输入模拟信号的最高频率

⚠️ 考点：简答 | ⭐⭐  
❌ 易错：条件是"大于等于"2倍，不是"大于"。

### 2.7 A/D 转换器主要技术指标

**分辨率**：输出数字量变化一个相邻数码（即 LSB 变化1）所对应的最小模拟输入电压变化量。

$$\text{分辨率} = \frac{V_{FS}}{2^n}$$

其中 $V_{FS}$ 为满量程电压，n 为 ADC 位数。

**转换时间**：从启动信号开始到转换结束并得到稳定数字输出量所需的时间。ADC0809 在 640KHz 时钟下转换时间为 **100μs**。

**量程**：ADC 所能转换的输入电压范围。ADC0809 为 **0～+5V**。

⚠️ 考点：简答/选择 | ⭐⭐

### 2.8 ADC0809 芯片（重点）

**主要特性**：

| 特性 | 参数 |
|------|------|
| 分辨率 | **8位**，无漏码 |
| 输入通道 | **8路**单端模拟量（IN0～IN7） |
| 输入电压 | 0～+5V |
| 参考电压 | +5V |
| 转换时间 | **100μs**（640KHz 标准时钟） |
| 时钟频率 | 100KHz～1280KHz，标准 **640KHz** |
| 电源 | 单一 +5V |
| 转换原理 | **逐次逼近式** |
| 误差 | ±1LSB |

**关键引脚**（简答/综合必背）：

| 引脚 | 功能 |
|------|------|
| IN0～IN7 | 8路模拟量输入 |
| ADDA/ADDB/ADDC | 3位地址，选择模拟通道（000=IN0，...，111=IN7） |
| ALE | 地址锁存允许，**上升沿**锁存 ADDC/ADDB/ADDA |
| START | 启动转换，**正脉冲**有效 |
| EOC | 转换结束信号（输出）。**低电平** = 正在转换；**高电平** = 转换结束 |
| OE | 输出允许（输入线），高电平允许输出数据 |
| CLOCK | 时钟输入，100KHz～1280KHz |

**通道选择表**：

| ADDC | ADDB | ADDA | 选中通道 |
|------|------|------|---------|
| 0 | 0 | 0 | IN0 |
| 0 | 0 | 1 | IN1 |
| … | … | … | … |
| 1 | 1 | 1 | IN7 |

⚠️ 考点：综合题/简答 | ⭐⭐⭐  
❌ 易错：EOC 低电平表示**正在转换**（不是完成！），高电平才是转换完成可读数据。

### 2.9 ADC0809 与 CPU 的三种接口方式

**等待方式**：
- 启动 ADC → 软件延时（等待转换时间，如100μs）→ 读入数据
- 特点：对硬件要求低，但速度比查询方式慢

```asm
MOV  AL, 00H      ; 通道号0（选IN0）
OUT  84H, AL      ; 启动 ADC0809 通道0
CALL DELAY100     ; 延时 100μs，等待转换结束
IN   AL, 84H      ; 读入转换结果
```

**查询方式**：
- 启动 ADC → 循环读 EOC 状态 → EOC=1 时读数据
- 特点：不浪费等待时间，比中断方式硬件简单

```asm
; 查询 EOC 由低变高的过程（见课件例8.3.2）
A2: IN   AL, ADCS      ; 读状态口（EOC接D7）
    TEST AL, 80H       ; 检测D7位（EOC）
    JNZ  A2            ; EOC≠0，说明还未开始转换，等待
A3: IN   AL, ADCS
    TEST AL, 80H
    JZ   A3            ; EOC=0，转换未完成，等待
    IN   AL, ADCC      ; EOC=1，读转换结果
```

**中断方式**：
- EOC 信号连接到 8259A 的中断请求线，转换完成时自动触发中断，CPU 在中断服务程序中读数据
- 特点：效率最高，CPU 无需等待

⚠️ 考点：综合题 | ⭐⭐⭐

### 2.10 采样保持器

**作用**（简答必背）：

在 A/D 转换期间，要求被测模拟信号保持不变。采样保持器在**采样阶段**跟踪输入信号，在**保持阶段**将信号锁定，使 ADC 转换过程中信号稳定。当转换快速变化的模拟信号时，采样保持器能有效减小**孔径误差**。

**可省略采样保持器的条件**：当外接模拟信号的变化速度相对于 A/D 转换速度来说足够慢，在转换期间可视为直流信号时，可以省略采样保持电路。

⚠️ 考点：简答 | ⭐⭐

## 3. 高频计算题型与模板

### 3.1 D/A 转换——已知数字量求输出电压

**公式**：

$$V_{OUT} = \frac{V_{REF} \times N}{2^n}$$

（若运放反相，则加负号；若题目给出电压范围为正，直接用正值）

**解题步骤**：
1. 确定 $n$（DAC位数）和 $2^n$
2. 将十六进制数字量 N 转换为十进制
3. 代入公式计算

**示例**（练习题 113）：

> DAC0832 参考电压=3V，输出电压范围 0～3V，送入数字量为 128，输出电压是多少？

Step 1：n=8，$2^n = 256$  
Step 2：N = 128（十进制）  
Step 3：$V_{OUT} = \dfrac{3 \times 128}{256} = \dfrac{384}{256} = \mathbf{1.5V}$

---

### 3.2 A/D 转换——已知输出数字量求输入电压

**公式**（与 D/A 互逆）：

$$\frac{V_{FS}}{2^n} = \frac{V_{IN}}{N} \implies V_{IN} = \frac{V_{FS} \times N}{2^n}$$

**解题步骤**：
1. 确定满量程电压 $V_{FS}$ 和位数 n
2. 代入数字量 N（注意十六进制转十进制）
3. 计算结果

**示例**（练习题 108）：

> ADC0809 输入电压=5V、参考电压=5V，CPU 接收到的数字量为 32，外部输入电压是多少？

Step 1：$V_{FS} = 5V$，n=8，$2^n = 256$  
Step 2：N = 32  
Step 3：$V_{IN} = \dfrac{5 \times 32}{256} = \dfrac{160}{256} = \mathbf{0.625V}$（注：课件答案写0.125V，但按公式应为0.625V）

❌ 易错：注意 ADC0809 参考电压即为满量程电压（此题为5V）。

---

### 3.3 D/A 转换——已知输出电压求数字量

**公式**：

$$N = \frac{2^n \times V_{OUT}}{V_{REF}}$$

**示例**（练习题 115）：

> DAC0832 单极输出，参考电压=5V，最大+5V，若输出 2V 需写入多少数字量？

Step 1：$V_{REF} = 5V$，n=8，$2^n = 256$  
Step 2：$N = \dfrac{256 \times 2}{5} = \dfrac{512}{5} = 102.4 \approx \mathbf{102}$（取整，即 66H）

---

### 3.4 DAC0832 输出波形分析（峰值电压+周期）

**解题步骤**：
1. 找程序中输出的最大和最小数字量
2. 用 D/A 公式换算电压（确定峰值）
3. 统计循环输出的数值个数 × 每步延时 = 周期

**示例**（练习题 114）：

> 参考电压=5V，程序循环输出 0 到 80H，每步延时 1ms，画出波形并标出峰值和周期。

Step 1：最大数字量 = 80H = 128  
Step 2：峰值电压 = $\dfrac{5 \times 128}{256} = \mathbf{2.5V}$  
Step 3：从 0 到 80H 共 129 个值（0,1,...,128），每步 1ms  
Step 4：周期 = 129 × 1ms = **129ms**

## 4. 练习题精析

### 题目 103（练习题第103题）

**题目**：A/D 和 D/A 转换器在微机应用中起什么作用？

**解答**：
- D/A 转换器（DAC）：将 CPU 输出的数字量转换为模拟量，驱动现场执行机构（如电机、加热器）进行控制输出。
- A/D 转换器（ADC）：将传感器采集的模拟量（如温度、压力等物理量对应的电压信号）转换为数字量，供 CPU 进行处理和分析。

两者共同构成微机控制系统与外部物理世界之间的接口桥梁。

**考察点**：A/D、D/A 的功能定义

---

### 题目 104（练习题第104题）

**题目**：为什么 DAC0832 特别适用于多个模拟量同时输出的场合？

**解答**：
DAC0832 支持**双缓冲工作方式**。在双缓冲方式下，可先将不同数据分别写入各 DAC0832 的输入寄存器（第一级锁存），然后同时发出控制信号，使所有 DAC 的 DAC 寄存器（第二级）同时更新，从而实现多路模拟信号的**同步输出**。若只有一级缓冲，各路更新有时间差，无法同步。

**考察点**：双缓冲方式的应用场合

---

### 题目 105（练习题第105题）

**题目**：简述采样定理。

**解答**：
对一个有限频谱（最高频率为 $f_{max}$）的连续信号进行采样，当采样频率 $f_s \geq 2f_{max}$ 时，采样输出信号能无失真地恢复到原来的连续信号。其中 $f_{max}$ 是输入模拟信号的最高频率，$2f_{max}$ 也称为奈奎斯特频率。

**考察点**：采样定理定义（必背）

### 题目 106（练习题第106题）

**题目**：采样/保持器有什么作用？

**解答**：
在 A/D 转换期间，需要被测信号在整个转换时间内保持稳定，采样/保持器的作用是：
1. **采样阶段**：跟踪输入模拟信号的瞬时值；
2. **保持阶段**：将该瞬时值锁定不变，供 ADC 在转换期间使用，从而减小因信号变化引起的**孔径误差**。

**考察点**：采样保持器作用

## 5. 简答题备考模板

### 简答题1：DAC0832 的三种工作方式

**规范答法**：

DAC0832 有三种工作方式：
1. **双缓冲方式**：数据依次经过8位输入寄存器和8位DAC寄存器两级锁存后才进行D/A转换。适用于多片DAC需要同步更新模拟输出的场合。
2. **单缓冲方式**：其中一级寄存器直通（旁路），数据只经一级锁存即进行转换。适用于单路或不需要同步的场合，是最常见的典型连接。
3. **直通方式（直接输入）**：两级寄存器均直通，数据直接进入D/A转换器。适用于外部已有数据锁存器的场合。

---

### 简答题2：ADC0809 与 CPU 的三种数据传送方式

**规范答法**：

ADC0809 与 CPU 的数据传送控制方式有三种：
1. **等待方式**：发出启动信号后，进行软件延时等待转换完成（ADC0809 在 640KHz 时约 100μs），然后读入数据。对硬件要求低，但效率较低。
2. **查询方式**：不断查询 EOC 引脚状态，EOC=1 时表示转换完成，才读取数据。效率高于等待方式。
3. **中断方式**：EOC 信号连接中断控制器，转换完成后主动发中断请求，CPU 在中断服务程序中读数据。效率最高，CPU 无需等待。

---

### 简答题3：采样定理

**规范答法**：

采样定理（奈奎斯特定理）：对一个有限频谱的连续模拟信号进行采样，当采样频率 $f_s \geq 2f_{max}$（$f_{max}$ 为信号最高频率）时，采样得到的离散信号能够无失真地恢复到原来的连续信号。实际应用中，采样频率通常取信号最高频率的 3～5 倍以留有裕量。

---

### 简答题4：A/D 转换三种方法比较

**规范答法**：

常用 A/D 转换方法有三种：
1. **计数式**：最简单，但转换速度很低（最慢）；
2. **逐次逼近式**：速度较高，结构简单，价格适中，是微型机应用系统中**最常用**的 A/D 转换方式（如 ADC0809、AD574）；
3. **双积分式**：精度高，抗干扰能力强，但速度低，适用于精度要求高而速度要求不高的场合（如仪器仪表）。

---

### 简答题5：采样保持器的作用

**规范答法**：

采样保持器的作用：在 A/D 转换期间，将采样所得的瞬时模拟值保持稳定不变，确保 ADC 在整个转换时间内输入信号不发生变化，从而减小**孔径误差**，提高转换精度。特别是对于快速变化的模拟信号，不使用采样保持器会因转换时间内信号变化而引起较大误差。

## 6. 易错点汇总

- ❌ 易错：D/A 公式中 $2^n$ 写成 n（如 8位DAC写成除以8）  
  ✅ 正确：8位DAC，$2^8 = 256$；12位DAC，$2^{12} = 4096$

- ❌ 易错：EOC=0 认为是转换完成  
  ✅ 正确：EOC **低电平** = 正在转换或即将转换；EOC **高电平** = 转换结束可读数据

- ❌ 易错：采样定理写成 $f_s > 2f_{max}$（漏掉等号）  
  ✅ 正确：$f_s \geq 2f_{max}$（大于等于）

- ❌ 易错：DAC0832 是电压型输出  
  ✅ 正确：DAC0832 是**电流型**输出（$I_{OUT}$），需外接运算放大器才能得到电压输出

- ❌ 易错：三角波程序中忘写两个 DEC AL  
  ✅ 正确：从递增转递减时，需两次 DEC AL（先使AL=FFH，再DEC使AL=FEH），消除波顶的平顶失真

- ❌ 易错：ADC0809 标准时钟写错  
  ✅ 正确：标准时钟 **640KHz**，转换时间 **100μs**（不是 1MHz 或其他值）

- ❌ 易错：分辨率的分子写为满量程数字量  
  ✅ 正确：分辨率 = $V_{FS} / 2^n$，分子是**满量程电压**（模拟量），分母是 $2^n$

- ❌ 易错：双缓冲方式说不清楚"两级锁存"的意义  
  ✅ 正确：两级锁存使得可以先加载数据，后统一触发，实现**多路同步更新**

## 7. 一页纸速记

**核心公式**：

$$D/A：V_{OUT} = \frac{V_{REF} \times N}{2^n} \quad A/D：V_{IN} = \frac{V_{FS} \times N}{2^n}$$

$$\text{分辨率（DA）}= \frac{FS}{2^n} \quad \text{采样定理：}f_s \geq 2f_{max}$$

**必背结论**：

- DAC0832：8位 / 电流型输出 / 三种方式（双缓冲/单缓冲/直通）/ 建立时间 1μs
- ADC0809：8位 / 8通道 / 逐次逼近 / 100μs / 三种接口（等待/查询/中断）
- EOC：低=转换中，高=转换完成
- 最常用ADC方式：**逐次逼近式**
- 高精度场合：**双积分式**
- 采样保持器作用：减小**孔径误差**

**最易混淆的区别**：

| 对比项 | DAC0832 | ADC0809 |
|--------|---------|---------|
| 方向 | 数字→模拟 | 模拟→数字 |
| 位数 | 8位 | 8位 |
| 通道数 | 1路输出 | 8路输入 |
| 输出类型 | 电流型（需运放） | 三态数字输出 |
| 建立/转换时间 | 1μs | 100μs（@640KHz）|

| 对比项 | 等待方式 | 查询方式 | 中断方式 |
|--------|---------|---------|---------|
| 判断转换完成 | 固定延时 | 查询EOC | EOC触发中断 |
| CPU效率 | 最低 | 中等 | 最高 |
| 硬件要求 | 最低 | 中等 | 最高（需8259A）|
