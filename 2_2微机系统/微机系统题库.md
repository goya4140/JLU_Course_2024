【练习题 Q1】 微处理器、微型计算机、微型计算机系统的区别是什么？
```
┌────────────────────────────────────────┐
│          微型计算机系统                │
│  ┌──────────────────────────────────┐  │
│  │        微型计算机（裸机）        │  │
│  │  ┌────────────────────────────┐  │  │
│  │  │     微处理器（CPU核心）    │  │  │
│  │  └────────────────────────────┘  │  │
│  └──────────────────────────────────┘  │
└────────────────────────────────────────┘
```
- **微处理器**：由一片或少数几片大规模集成电路组成的 CPU，是微机系统的**核心部件**，本身不能独立工作。
- **微型计算机**：由微处理器 + 主存储器 + I/O 接口电路 + 系统总线构成的**裸机整体**，有完整的运算和控制能力，但没有外设和软件。
- **微型计算机系统**：以微型计算机为中心，配以 I/O 外围设备、系统软件（操作系统、编译器等），以及电源、面板和机架等，组成的**完整可用系统**。

1. **微处理器（CPU）**
2. **主存储器**
3. **I/O 接口电路**
4. **系统总线**（数据总线 DB + 地址总线 AB + 控制总线 CB）

【练习题 Q3 ⚠️考点 🔥高频】** CISC、RISC 的技术特点。
CISC 特点：
- 指令系统丰富且复杂，指令长度**不等长**
- 访问存储器的指令较多，逻辑控制复杂
- 电路设计周期长，功耗和面积较大
- 在**一条指令执行结束后**响应中断

RISC 特点：
- 指令系统简单，指令短且**等长**
- 仅包含必要指令，专有指令（Load/Store）访问内存，电路结构简单
- 设计周期短，功耗低，**易于采用并行计算技术**
- 可以在**指令执行的适当地方**响应中断
