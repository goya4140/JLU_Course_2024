# 操作系统期末复习资料

这套资料按课内 9 章组织，原则是：课内课件定范围，习题课、作业和期末真题定考法，王道只用于把抽象概念讲清楚。

## 推荐复习顺序

1. [00_复习总览.md](00_复习总览.md)：先看全局地图和题型分布。
2. [01_操作系统概述.md](01_操作系统概述.md)：建立 OS 的角色、特征和运行环境。
3. [02_进程线程与作业.md](02_进程线程与作业.md)：理解多道程序、进程、线程、作业。
4. [03_中断与处理机调度.md](03_中断与处理机调度.md)：掌握中断、调度算法、实时调度。
5. [04_互斥同步与通讯.md](04_互斥同步与通讯.md)：重点练 PV、管程、互斥正确性。
6. [05_死锁与饥饿.md](05_死锁与饥饿.md)：重点练银行家算法、死锁检测、资源图。
7. [06_存储管理.md](06_存储管理.md)：理解地址映射、分区、页式/段式/段页式。
8. [07_虚拟存储系统.md](07_虚拟存储系统.md)：重点练缺页、置换、页表、工作集。
9. [08_文件系统.md](08_文件系统.md)：掌握文件组织、目录、打开文件表、外存分配。
10. [09_设备与IO管理.md](09_设备与IO管理.md)：重点练 I/O 流程、缓冲缓存、磁盘调度、SPOOLing。

## 来源优先级

1. `操作系统_JLU` 中课内小版 PDF。
2. `操作系统习题课.pdf`、`23级计院操作系统期末.pdf`、`操作系统_JLU/作业.md`。
3. `操作系统_JLU/计算机操作系统教程 第4版.pdf`。
4. `操作系统_王道`，只作为解释材料。

## 使用方式

- 第一轮：只看每章“知识框架”和“从 0 理解”，把名词串起来。
- 第二轮：精看“核心机制/算法/公式”和“易错点”。
- 第三轮：只刷“题型与解析”和“期末复习清单”。
