# JLU_Course_2024

吉林大学计算机相关课程学习资料整理项目，主要面向 2024 级计算机相关专业同学的课程学习、期末复习和资料检索。仓库按学期、课程和章节归档，尽量把零散的 Markdown、图片、PDF 作业和复习材料整理成适合在线阅读的形式。

## 在线浏览

- 在线阅读站：[https://goya4140.github.io/JLU_Course_2024/](https://goya4140.github.io/JLU_Course_2024/)
- GitHub 仓库：[https://github.com/goya4140/JLU_Course_2024](https://github.com/goya4140/JLU_Course_2024)

在线阅读站基于 Docsify 构建，支持侧边栏导航、全文搜索、代码复制、图片预览和数学公式渲染。

## 如何使用

- 第一次访问建议先看 [课程总览](课程总览.md)，了解各学期课程资料的分布。
- 如果正在准备大二下课程，可直接阅读 [大二下课程总览](大二下课程总览.md)，再进入具体课程。
- 复习时可以优先使用左侧目录和搜索框定位知识点；含 PDF 的作业或题目会作为独立入口保留。
- 资料主要用于学习交流和复习辅助，具体考试范围、作业要求和课堂安排仍以任课教师发布内容为准。
